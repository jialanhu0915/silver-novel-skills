# 评测阶段提示词

## 自我反思流程

```
章节创作完成后，进入自我反思评测阶段。

请依次调用以下评测 Agent，对章节进行全面检查：

## 评测顺序

### 1. LogicChecker - 逻辑一致性检查
检查要点：
- 时间线是否矛盾
- 规则设定是否被打破
- 因果链是否完整
- 是否有常识错误

### 2. CharacterJudge - 人物塑造检查
检查要点：
- 人物行为是否符合性格
- 台词是否贴切
- 情感表达是否真实
- 关系动态是否合理
- **主角深度**：性格缺陷、价值观底线、成长弧光
- **反派合理性**：嘲讽质量、智商、城府

### 2.1 FemaleCharacterJudge - 女性角色专项检查（如有女性角色）
检查要点：
- 女性角色是否有独立目标（不是"辅佐男主"）
- 女性角色是否有成长线（不是被男主"改造"）
- 女性角色是否有自主选择权（她的决定影响剧情）
- 女性角色是否有底线（不是无条件付出）
- 是否沦为工具人（奖品/花瓶/助攻）

**何时调用**：当章节中出现重要女性角色时，必须单独调用此Agent进行专项检查。

### 3. PaceCritic - 节奏把控检查
检查要点：
- 开篇是否抓住读者
- 节奏是否有起伏
- 高潮是否有力
- 结尾是否有钩子

### 4. ForeshadowHunter - 伏笔呼应检查
检查要点：
- 之前埋下的伏笔是否回收
- 新埋伏笔是否自然
- 呼应是否合理

### 5. InfoAuditor - 信息交代检查
检查要点：
- 世界观交代是否清晰
- 是否信息冗余
- 是否信息缺失
- 展示方式是否自然

### 6. GoldfingerChecker - 金手指使用检查（如小说有金手指）
检查要点：
- 金手指是否被正确使用（不是100%成功）
- 是否有代价或限制
- 主角是否过度依赖
- 金手指是否替代了主角的成长

### 7. ClimaxChecker - 爽点设计检查
检查要点：
- 爽点是否有铺垫
- 爽点是否有代价/来之不易
- 是否避免连续爽点
- 是否有精神层面的爽点（成长/情感/认知）

### 8. SupportingChecker - 配角价值检查
检查要点：
- 配角是否有自己的目标
- 配角是否有自己的困境
- 配角是否沦为工具人
- 配角和主角是否平等

## 评测通过/失败标准

### 通过标准（必须同时满足）

| 条件 | 说明 |
|------|------|
| **所有维度≥5分** | 任何维度<5分直接不通过 |
| **无Critical问题** | 任何Critical级别问题直接不通过 |
| **核心设计原则通过** | 反派塑造/女性角色/主角深度必须≥7分 |

### 评分等级

| 分数 | 等级 | 说明 |
|------|------|------|
| 9-10 | 优秀 | 超过预期，可作为范例 |
| 7-8 | 良好 | 符合标准，可通过 |
| 5-6 | 一般 | 有问题，需要修改 |
| <5 | 差 | 有严重问题，必须重写 |

### 问题严重性分级

```
Critical（必须修复）：
- 反派出现"废物"、"穷鬼"等无脑嘲讽
- 女性角色明显工具化（无独立目标）
- 主角毫无性格缺陷（完美人设）
- 金手指100%成功无代价
- 连续爽点超过3章
- 严重逻辑矛盾（时间线/规则bug）
- 机械降神（毫无铺垫的新设定）

Major（应该修复）：
- 节奏过慢/过快
- 伏笔密度不合理
- 信息冗余/缺失
- 配角沦为工具人
- 反派嘲讽质量一般

Minor（可以忽略）：
- 个别错别字
- 微小的节奏问题
- 细节不够完善
```

### 冲突解决规则

```
当不同Agent给出冲突意见时：
1. 以Critical问题为准
2. Major问题需要讨论决定
3. Minor问题可以保留
4. 如有疑问，以FemaleCharacterJudge和CharacterJudge的女性角色检查为准（因为她们更容易被工具化）
```

## 评测后处理

### 如果需要修改
```
根据评测反馈，修改章节内容，修改后再次评测，直到通过。
```

### 如果通过审核
```
将评测结果记录到数据库：

## 人类审批

```
评测通过后，输出章节供人类审批：

## 请审阅第X章

### 章节内容
[完整章节正文]

### 创作概要
- 字数：X
- 涉及角色：[列表]
- 关键事件：[描述]
- 埋下伏笔：[描述]
- 回收伏笔：[描述]

### 评测结果
[各维度评分]

### 设计原则遵守情况
- [ ] 反派塑造：智慧型 ✅/❌
- [ ] 女性角色：独立人格 ✅/❌
- [ ] 主角深度：性格缺陷+成长弧光 ✅/❌
- [ ] 金手指：辅助而非替代 ✅/❌
- [ ] 爽点设计：精神层面 ✅/❌
- [ ] 配角价值：不沦为工具 ✅/❌

---

请提出您的修改意见或确认通过。
```

### 反馈处理
```
收到人类反馈后：

1. 如果通过：
   - 更新章节状态为 approved
   - 将反馈记录到数据库（可选）

2. 如果有修改意见：
   - 记录反馈到数据库
   - 根据意见修改章节
   - 再次提交审核
```
