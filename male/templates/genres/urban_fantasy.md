# 都市异能题材模板

## 类型特征

都市异能小说是以现代都市为背景，融合异能觉醒、修真传承、地下势力与网文爽感的作品。

**核心要素：**
- 现代都市：日常生活+异能世界双线
- 异能/修真：超能力觉醒、修炼升级
- 扮猪吃虎：隐藏身份、都市装逼
- 都市地下：异能者组织、灵异世界

**与都市文的区别：**
- 有异能/修真元素
- 有升级体系
- 有超凡势力与冲突

**与玄幻的区别：**
- 都市背景为主
- 现代生活元素
- 都市规则与超凡规则碰撞

## 力量体系

### 异能觉醒体系

```
异能等级：
- F级：初觉醒、不稳定
- E级：入门、可控制
- D级：稳定输出
- C级：成为正式成员
- B级：精英级别
- A级：强者/首领级别
- S级：传说级别

异能类型：
- 元素系：火/水/风/雷/土/冰
- 精神系：读心/心灵控制/幻象
- 强化系：力量/速度/防御强化
- 空间系：瞬移/空间切割/储物
- 感知系：预知/千里眼/顺风耳
- 特殊系：规则类/因果类/时间类
```

### 修真体系

```
修为等级（任选其一）：

体系一（简化版）：
- 炼气期 → 筑基期 → 金丹期 → 元婴期 → 出窍期
- 凡人 → 修士 → 高手 → 大能 → 仙人

体系二（都市版）：
- 觉醒者 → 修炼者 → 大师 → 宗师 → 大宗师 → 陆地神仙

能力来源：
- 天地灵气：自然修炼
- 天材地宝：丹药/灵草
- 传承功法：师徒/遗迹
- 系统加点：氪金/任务
```

### 都市能力设定

```
信息能力：
- 黑客技能：网络信息
- 情报收集：人物背景
- 占卜算命：预知未来

商业能力：
- 神豪系统：财富异能
- 投资眼光：商业洞察
- 人脉经营：社交异能

武力能力：
- 武术修为：传统武术
- 热武器：枪械使用
- 异能战斗：超能力对抗
```

## 世界构成

### 都市表面世界

```
日常生活：
- 学校：校园生活
- 公司：职场环境
- 家庭：亲人关系
- 社交圈：朋友聚会

公共场合：
- 商业中心：高端消费
- 地下黑市：灰色交易
- 娱乐场所：KTV/酒吧/会所
- 医院：急救与秘密
```

### 都市暗面世界

```
异能者组织：
- 官方组织：国家安全部门
- 家族势力：传承古老的异能家族
- 门派：修真宗门/古武门派
- 地下联盟：自由异能者联盟
- 商业组织：异能者企业/研究所

灵异场所：
- 凶宅：死过人的地方
- 古墓/遗迹：传承所在
- 深山老林：灵气充沛
- 阴阳交界：两界缝隙

都市传说：
- 深夜公交：灵异事件
- 废弃医院：恐怖传闻
- 诡异校园：学生失踪
```

### 都市规则

```
明面规则：
- 法律：违法犯罪会被抓
- 社会秩序：正常人生活方式
- 道德约束：社会公德

暗面规则：
- 异能者协议：不能暴露给普通人
- 地下交易：黑市/灵物交易
- 势力范围：不同组织控制区域
- 冲突解决：擂台/谈判/暗杀
```

## 经典套路

### 兵王回归流

```
经典开局：
- 特种兵王/杀手之王退役
- 隐姓埋名/保护任务
- 回归都市/与亲人团聚
- 扮猪吃虎、暗中守护

关键转折：
- 美人总裁/清纯校花
- 敌人追杀/幕后黑手
- 地下势力/古武家族
- 都市显圣/众人震惊
```

### 风水相师流

```
经典开局：
- 获得传承/天师血脉
- 算命看相/风水布局
- 治病救人/驱邪捉鬼
- 名声大噪/各方追捧

关键转折：
- 富豪求卦/大单生意
- 得罪权贵/暗中报复
- 古墓探险/获得传承
- 灵气复苏/乱世将起
```

### 医术修真流

```
经典开局：
- 获得上古医仙传承
- 医术通神/活死人肉白骨
- 低调行医/暗中积累
- 重返都市/医术震惊

关键转折：
- 治疗绝症/大佬感恩
- 疑难杂症/医学难题
- 中医崛起/文化自信
- 医患纠纷/权贵打压
```

### 灵气复苏流

```
经典开局：
- 灵气开始复苏
- 动植物开始进化
- 主角先行一步
- 末世求生/成为强者

关键转折：
- 怪物入侵/异界生物
- 人类幸存者基地
- 修炼功法流传
- 收拢人才/建立势力
```

## 势力冲突

### 异能者势力冲突

```
类型：
- 家族vs门派：传承之争
- 官方vs民间：秩序与自由
- 新兴vs古老：势力更替
- 商业vs武力：利益分配

核心矛盾：
- 资源（灵气/灵物/功法）
- 地盘（城市控制权）
- 话语权（规则制定）
- 生存空间
```

### 都市势力冲突

```
类型：
- 商战：市场竞争
- 黑帮：地下秩序
- 权贵：官商勾结
- 民间：舆论与正义

核心矛盾：
- 经济利益
- 权力寻租
- 势力范围
- 面子与尊严
```

## 女性角色设计

### 都市异能女性类型

```
都市丽人：
- 美女总裁：霸道女总
- 清纯校花：校园女神
- 知性御姐：职场精英
- 神秘网红：身份成谜

超凡女性：
- 女修士：修为高深
- 女异能者：能力出众
- 女武者：古武传人
- 鬼魂/妖修：非人类女性

设计原则：
- 要有独立事业/目标
- 要有超凡能力
- 要避免工具化
- 感情线要自然发展
```
