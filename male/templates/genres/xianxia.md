# 仙侠题材模板

## 类型特征

仙侠小说是以修真问道为核心，融合中国传统道教文化、神魔传说、神话体系的作品。

**核心要素：**
- 修炼目标：长生不老、飞升成仙
- 世界观：仙凡差距、洞天福地
- 冲突：道统之争、因果劫难
- 主线：问道求真、渡劫飞升

**与玄幻的区别：**
- 玄幻偏重力量和争斗
- 仙侠偏重道心和悟性
- 仙侠更注重心境修炼

## 力量体系

### 修真境界（仙侠标准）

```
凡境：
1. 炼气期：吸纳灵气、筑基之基
2. 筑基期：丹田圆满、可寿百年
3. 金丹期：丹成而寿五百
4. 元婴期：元婴出窍、可夺舍延寿
5. 化神期：神魂化虚、触摸法则
6. 炼虚期：炼化虚空、与道合真
7. 合道期：合道成功、即为大乘

仙境：
8. 地仙：地仙之位、开辟福地
9. 天仙：天仙之位、掌控法则
10. 金仙：金仙不朽、法则圆满
11. 大罗金仙：大罗果位、万劫不灭
12. 混元大罗金仙：混元如一、天道圣人

（可简化为5-7个境界）
```

### 仙侠特色能力

```
神通：
- 御剑术：飞剑杀人于千里之外
- 瞬移：缩地成寸
- 炼丹：丹药妙用无穷
- 炼器：法宝炼制
- 符箓：天地之力凝聚
- 阵法：借天地之势

道法：
- 五行法术：金木水火土
- 雷法：至阳至刚
- 剑意：万法归剑
- 佛门神通：金刚伏魔
```

### 仙凡差距

```
寿命差距：
- 凡人：百年以内
- 筑基：百年左右
- 金丹：五百年
- 元婴：千年
- 化神：数千年

态度差异：
- 仙人视凡人为蝼蚁
- 但天道运转需要凡人香火
- 仙凡不可通婚（天道禁忌）
```

## 世界构成

### 六道轮回设定

```
仙界：
- 仙人居所
- 灵气充沛
- 法则完善

人界：
- 凡人生存
- 宗门王朝
- 洞天福地

妖魔界：
- 妖族、魔族
- 弱肉强食
- 血腥法则

幽冥界：
- 鬼魂归宿
- 轮回运转
- 忘川河、奈何桥

佛界（如有）：
- 佛教势力
- 度化苍生
- 因果报应

神界（如有）：
- 天庭管辖
- 封神榜
- 因果功德
```

### 仙门设定

```
仙门类型：
- 剑修门派：以剑入道、一剑破万法
- 道统宗门：传承道法、清心寡欲
- 魔道圣地：修炼魔功、不择手段
- 佛门寺院：慈悲为怀、度化苍生
- 世家仙族：血脉传承、底蕴深厚

宗门规矩：
- 师徒传承：徒弟是师父的延续
- 道统之争：理念不同的冲突
- 资源争夺：灵脉、秘境、弟子
- 劫难降临：天劫、人劫、因果劫
```

### 因果与劫难

```
因果法则：
- 种因得果、善恶有报
- 杀人夺宝会沾染因果
- 功德和业力影响修炼

劫难类型：
- 天劫：修炼到一定境界必须渡劫
- 人劫：与人争斗带来的劫难
- 心魔劫：心境不纯会走火入魔
- 杀劫：杀伐过多必遭反噬

渡劫设计：
- 九死一生的紧张感
- 渡劫失败的后果
- 渡劫成功的收获
```

## 势力体系

### 势力类型

```
正道仙门：
- 传承有序、规矩森严
- 以斩妖除魔为己任
- 内部有派系之争

魔道势力：
- 修炼魔功
- 不受天道庇护
- 渡劫更加困难

散修：
- 无门无派
- 独自摸索
- 生存艰难

妖族：
- 开启灵智可修仙
- 被人族视为异类
- 有妖王统御

鬼修：
- 鬼魂修炼
- 需要轮回或夺舍
- 幽冥界势力
```

## 经典套路

### 问道求真型

```
经典开局：
- 主角偶入仙门/得到传承
- 发现修炼之路漫漫
- 追求长生/问道成仙
- 经历各种劫难

关键转折：
- 拜入名师
- 获得独特功法
- 道心考验
- 渡劫飞升
```

### 妖魔乱世间

```
经典开局：
- 天下大乱、妖魔横行
- 主角立志斩妖除魔
- 仙门式微、正道衰落
- 主角力挽狂澜

关键转折：
- 发现幕后黑手
- 妖魔势大、正道内斗
- 主角悟道、突破境界
- 最终决战
```

### 轮回转世型

```
经典开局：
- 前世是超级强者
- 转世重生、重新修炼
- 前世记忆逐步恢复
- 复仇/探索真相

关键转折：
- 前世仇敌仍在
- 前世的爱恨纠葛
- 恢复全部记忆
- 重回巅峰/超越前世
```

## 伏笔类型

### 仙缘伏笔

```
类型：
- 奇遇：偶得宝物/传承
- 师徒：师父的特殊身份
- 因果：前世的羁绊
- 天道：冥冥中的安排

设计要点：
- 仙缘要合理铺垫
- 不是凭空出现
- 揭示时要震撼
```

### 身世伏笔

```
类型：
- 上古大能转世
- 远古神兽后裔
- 天道眷顾之人
- 被封印的存在

设计要点：
- 初期只有零星线索
- 中期逐渐清晰
- 后期完全揭示
```

### 道心伏笔

```
类型：
- 心魔：隐藏的心结
- 道心：坚守的理念
- 执念：成道的阻碍
- 顿悟：突破的契机

设计要点：
- 影响修炼进度
- 关键时刻考验
- 克服后才能突破
```

## 设计检查清单

- [ ] 仙凡差距是否明确？
