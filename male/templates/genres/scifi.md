# 科幻题材模板

## 类型特征

科幻小说是以未来宇宙、科技文明、星际战争为背景，融合硬核科幻设定与网文爽感幻想作品。

**核心要素：**
- 星际宇宙：星际战争、殖民开拓、文明建设
- 科技设定：基因改造、机甲星舰、AI意识
- 双重升级：科技装备+个人武力并行
- 文明冲突：人类联邦、星际帝国、外星文明

**与玄幻的区别：**
- 科技为底层逻辑，非修炼体系
- 升级基于科技/基因/精神力，非灵气
- 敌人包含外星种族/AI/星际势力

## 力量体系

### 科幻升级体系

```
个人等级（基因进化）：
- 普通人 → 觉醒者 → 进化者 → 变异者 → 超越者
- 精神力等级：F-E-D-C-B-A-S-SS-SSS
- 基因强度：1-9阶基因锁

科技装备：
- 机械战甲：单兵→小队→军团级
- 星舰等级：小型巡逻舰→战列舰→母舰→行星级
- 武器系统：实弹→能量→湮灭

文明等级（宇宙文明）：
- 行星级 → 恒星级 → 星系级 → 星云级 → 宇宙级
```

### 核心能力类型

```
基因类能力：
- 体能强化：力量/速度/耐力提升
- 精神力：感知/控制/攻击
- 特殊体质：元素亲和/空间/时间

科技类能力：
- 机械外骨骼：战斗增强
- 智能武器：自适应/学习
- 能量护盾：攻防一体

精神类能力：
- 心灵感应：读取/植入思想
- 精神攻击：摧毁/控制
- 意识上传：数字永生
```

## 世界构成

### 宇宙势力格局

```
人类阵营：
- 星际联邦：统一的人类政权
- 贵族世家：掌握核心技术的家族
- 星际海盗：自由佣兵/掠夺者
- 星际企业：财团控制的商业势力

外星种族：
- 虫族：集群意识、无限繁殖
- 机械族：AI觉醒、硅基生命
- 能量族：纯能量体、神秘存在
- 古老文明：失落的星际遗迹

宇宙规则：
- 黑暗森林：文明之间的猜疑链
- 资源争夺：能源/领土/人口
- 技术封锁：高等文明限制低等
```

### 世界观层次

```
微观（个人）：
- 个人装备、武器、载具
- 基因进化、精神力觉醒

中观（势力）：
- 家族/企业/舰队
- 星球领地、资源采集

宏观（宇宙）：
- 星际国家、文明碰撞
- 宇宙法则、古老秘密
```

### 科幻规则

```
物理限制：
- 光速不可超越（除非特殊设定）
- 能量守恒（武器有冷却）
- 资源有限（稀缺性）

技术约束：
- 曲速/跃迁需要能量
- 高级装备需要匹配基因
- AI可能叛变/失控
```

## 经典套路

### 星际废柴逆袭流

```
经典开局：
- 偏远星球/底层矿工/废物学员
- 意外获得高等文明遗产/超级系统
- 基因觉醒、科技传承
- 从废柴到星际强者

关键转折：
- 学院比赛/舰队演习
- 发现古老遗迹
- 虫族/外星入侵
- 联邦内斗、贵族打压
```

### 星际领主流

```
经典开局：
- 继承一艘破船/废弃星球
- 招募船员、修复装备
- 发展领地、组建舰队
- 成为星际霸主

关键转折：
- 领地遭遇袭击
- 发现稀有资源
- 星际政治博弈
- 文明存亡危机
```

### 星际战争流

```
经典开局：
- 普通士兵/低级军官
- 战争爆发、绝境考验
- 战术天才/英雄崛起
- 决定战争走向

关键转折：
- 遭遇敌军主力
- 发现敌方弱点
- 以弱胜强战役
- 战争与阴谋交织
```

### 星际探险流

```
经典开局：
- 星际探险家/赏金猎人
- 接到神秘任务
- 发现失落文明
- 揭开宇宙秘密

关键转折：
- 遗迹机关/守护者
- 远古科技/知识
- 文明真相
- 宇宙终极谜题
```

## 势力冲突

### 文明冲突

```
类型：
- 人类vs外星：生存之战
- 联邦vs帝国：政权争夺
- 贵族vs平民：阶级矛盾
- 科技vs神秘：路线之争

核心矛盾：
- 生存空间
- 资源分配
- 意识形态
- 文明存亡
```

### 内部势力

```
类型：
- 继承人之争：星际家族内斗
- 派系斗争：鸽派vs鹰派
- 企业竞争：市场份额/技术垄断
- 星际海盗：自由vs秩序

核心矛盾：
- 权力分配
- 利益再分配
- 路线选择
- 势力消长
```

## 女性角色设计

### 科幻女性类型

```
星际女性：
- 女舰长：冷静果断、领导才能
- 女科学家：智慧过人、科研突破
- 女战士：战力强悍、英姿飒爽
- 贵族千金：出身高贵、内心复杂

外星女性（可选）：
- 虫族女皇：蜂巢意识首领
- 能量族少女：神秘飘渺
- 机械少女：AI觉醒

设计原则：
- 要有独立目标（不是只为男主）
- 要有成长线
- 要避免工具化
```

## 设计检查清单

### 世界观
- [ ] 宇宙势力是否清晰？
- [ ] 科技上限是否合理？
- [ ] 升级体系是否自洽？

### 设定
- [ ] 基因进化逻辑是否清晰？
- [ ] 科技装备是否平衡？
- [ ] 文明冲突是否有层次？

### 人物
- [ ] 主角是否有独特金手指？
- [ ] 外星种族是否有特色？
- [ ] 女性角色是否独立？

### 爽点
- [ ] 星际战斗是否热血？
- [ ] 以弱胜强是否有铺垫？
- [ ] 科技碾压是否有代价？

### 创新
- [ ] 是否有独特科幻设定？
- [ ] 是否避免套路化？
- [ ] 是否有记忆点？
