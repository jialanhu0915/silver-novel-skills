# 都市题材模板

## 类型特征

都市小说是以现代都市为背景，融合身份反差、商战、暧昧、地下势力等元素的作品。

**核心要素：**
- 现代都市：职场、商战、豪门
- 身份反差：隐藏身份、马甲、扮猪吃虎
- 冲突：家族恩怨、商业竞争、情感纠葛
- 主线：逆袭崛起、复仇、探索真相

**与玄幻的区别：**
- 没有修炼体系
- 现实规则为主
- 能力来源：财富、权力、人脉、特殊能力

## 力量体系

### 都市权力结构

```
社会阶层：
顶层：顶级豪门、世家、隐形富豪
中层：精英阶层、企业高管、金领
底层：普通人、打工者、小市民

权力来源：
- 财富：金钱可以解决大部分问题
- 权力：政商关系、灰色地带
- 人脉：背景、关系网
- 特殊能力：信息、技术、武力
```

### 都市能力体系

```
信息能力：
- 情报收集：掌握他人秘密
- 数据分析：商业洞察
- 网络技术：黑客技能

商业能力：
- 经营管理：企业运营
- 投资眼光：发现商机
- 谈判技巧：利益最大化

武力体系（可选）：
- 地下势力：打手、保镖
- 特种兵：退役军人
- 武术高手：传统武术

特殊能力（可选）：
- 医术：绝世医术
- 玄学：风水、算命
- 异能：超能力设定
```

## 世界构成

### 社会环境

```
上流社会：
- 顶级会所、私人俱乐部
- 豪门宴会、社交场合
- 家族聚会、联姻

商业场所：
- 摩天大楼、企业总部
- CBD商务区
- 交易所、金融中心

底层社会：
- 老城区、城中村
- 地下市场、黑市
- 夜总会、灰色地带
```

### 势力类型

```
豪门世家：
- 传承几代人的大家族
- 家族企业内部竞争
- 继承人之争

商业帝国：
- 上市公司、跨国企业
- 创始人家族控股
- 股权争夺

地下势力：
- 灰色产业
- 地下拳场、黑市
- 与官场有勾结

政界力量：
- 官员、公务员
- 政商旋转门
- 保护伞
```

### 都市规则

```
明规则：
- 法律、合同、商业法则
- 社会道德、舆论
- 公平竞争

潜规则：
- 关系大于能力
- 金钱万能
- 权力寻租

灰色地带：
- 不违法但不道德
- 游走在法律边缘
- 潜规则运作
```

## 经典套路

### 豪门少爷/千金流

```
经典开局：
- 主角是豪门失散/被抛弃的孩子
- 在底层成长为有能力的人
- 被家族找到/主动回归
- 揭露真相、复仇/证明自己

关键转折：
- 家族内斗、继承人之争
- 商业对手的打压
- 真爱与利益的冲突
- 身份暴露
```

### 身份反差型

```
经典开局：
- 表面身份：普通人/废物/受气包
- 隐藏身份：超级富豪/绝世高手/神秘背景
- 被羞辱后身份逐渐暴露
- 震惊所有人

变体：
- 女神眼中的废物→实际是大人物
- 被离婚的废物→实际是隐藏大佬
- 穷小子→实际是隐形富豪
```

### 商战崛起型

```
经典开局：
- 主角有独特商业眼光/技术
- 被大公司打压/陷害
- 从小生意开始逆袭
- 最终成为商业巨头

关键要素：
- 商业策略、计谋
- 融资、并购、上市
- 对手打压、绝地反击
- 社会责任、企业道德
```

### 职场逆袭型

```
经典开局：
- 主角进入顶级公司
- 被同事排挤、被上司打压
- 默默积累、展现实力
- 最终成为核心人物

关键转折：
- 项目危机、需要力挽狂澜
- 竞争对手的阴谋
- 获得高层赏识
- 揭露对手阴谋
```

## 势力冲突

### 家族冲突

```
类型：
- 继承人之争：兄弟阋墙、叔侄夺位
- 嫡庶之争：嫡系vs旁支
- 联姻博弈：利益交换、情感纠葛
- 分家之争：财产分割、反目成仇

核心矛盾：
- 权力（谁说了算）
- 利益（财产分配）
- 面子（家族荣誉）
```

### 商业冲突

```
类型：
- 价格战：恶性竞争
-收购与反收购：资本博弈
- 技术封锁：核心技术的争夺
- 市场垄断：排挤竞争对手

核心矛盾：
- 市场利益
- 市场份额
- 核心资源
```

### 暗线冲突

```
类型：
- 官商勾结：保护伞与黑手套
- 地下势力：灰色地带
- 复仇者：与主角有旧怨
- 内鬼：身边人的背叛

核心矛盾：
- 秘密被揭露
- 势力平衡被打破
- 新旧势力的更替
```

## 情感线设计

### 女性角色类型

```
豪门千金：
- 出身高贵、能力出众
- 与主角从对立到合作
- 要避免"花瓶化"

职场女性：
- 独立自主、有事业心
- 可能与主角是同事/竞争对手
- 要有自己的人生目标

底层女性：
- 善良坚韧、默默付出
- 可能是主角的发小/青梅竹马
- 要避免"圣母化"

复杂女性：
- 亦正亦邪、有秘密
- 身份复杂、命运波折
- 要有独立的故事线
```

### 情感线设计原则

```
✅ 平等关系：不是一方依附另一方
✅ 共同成长：相互促进、不是单方面付出
✅ 真实冲突：理念不合、利益冲突
✅ 各自独立：有自己的人生目标

❌ 工具化：女性只是奖励或帮助主角
❌ 无脑倒贴：毫无理由的爱
❌ 雌竞：女性之间只有争夺
```

## 身份反差类如何避免套路化

### 核心原则

```
写好"扮猪吃老虎"：
1. 铺垫足够憋屈感（家族排挤、爱人背叛、资源掠夺）
2. 让读者共情"受气包"阶段
3. 金手指激活后精准打脸情节
4. 关键：不是让主角永远是受气包，而是每段潜伏都埋下未来的爆发点
```

### 避免套路化的技巧

| 技巧 | 示例 |
|------|------|
| 赋予深度性格矛盾 | 温柔医者带毒药救赎，暴躁黑帮老大偷资助学生 |
| 差异化爽点 | 被嘲笑的废柴实为上古神兽转世但被迫隐藏实力 |
| 日常生活仪式化 | 主角刻意选择最平凡身份，租房、打扫、处理日常琐事 |
| 配角也赋予微故事线 | 管家每周三神秘消失，原来在秘密抚养仇敌的孩子 |
| 回归真实生活细节 | 打斗时主角踩到香蕉皮的狼狈瞬间 |

## 写作注意事项

| 问题 | 后果 | 解决方案 |
|------|------|----------|
| 日常感消失 | 代入感随之崩塌 | 用真实世界的规则承载超凡想象 |
| 身份反差太悬浮 | 读者失去共鸣 | 回归真实生活细节 |
| 感情线过于急躁 | 破坏节奏 | "似有若无、似无若有"的克制感 |
| 爽点脱离都市背景 | 失去特色 | 基于都市真实感的爽点（拆迁纠纷、商业并购等） |

## 设计检查清单

### 世界观
- [ ] 社会阶层是否清晰？
- [ ] 都市规则是否自洽？
- [ ] 势力分布是否合理？

### 人物
- [ ] 身份反差是否有逻辑？
- [ ] 女性角色是否有独立人格？
- [ ] 配角是否有自己故事？

### 冲突
- [ ] 冲突是否有层次？
- [ ] 商战/家族争斗是否真实？
- [ ] 主角能力是否有展现？

### 情感
- [ ] 情感线是否平等？
- [ ] 女性角色是否工具化？
- [ ] 情感冲突是否有意义？

### 创新
- [ ] 是否有独特设定？
- [ ] 是否避免套路化？
- [ ] 是否有记忆点？
