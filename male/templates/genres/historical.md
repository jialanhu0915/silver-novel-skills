# 历史题材模板

## 类型特征

历史小说是以古代历史为背景，融合朝堂权谋、战争军事、寒门逆袭与网文爽感的作品。

**核心要素：**
- 古代背景：朝堂江湖、历史节点
- 权谋博弈：政治斗争、党派之争
- 战争军事：冷兵器作战、战术谋略
- 逆袭崛起：寒门崛起、知识碾压

**与玄幻的区别：**
- 没有修炼体系
- 武力来源：武术、兵法、个人勇武
- 成功路径：科举、军功、智谋、帝王术

## 力量体系

### 古代权力结构

```
社会阶层：
- 皇族：天子、宗室
- 贵族：世家、门阀、勋贵
- 士族：官员、地主、乡绅
- 平民：农工商、庶民
- 贱籍：奴婢、戏子、囚犯

权力来源：
- 皇权：天子授权
- 科举：文官晋升
- 军功：武职晋升
- 荫封：血缘世袭
- 关系：师生、同乡、联姻
```

### 古代武力体系

```
个人武力：
- 力大无穷：天生神力
- 武术流派：拳法、剑法、刀法
- 内功心法：（非玄幻系）养生气功
- 骑射功夫：冷兵器时代核心

军事体系：
- 步兵：基础兵种
- 骑兵：机动力量
- 水军：江河湖海作战
- 特种：死士、刺客、精骑

情报系统：
- 密探：暗中收集情报
- 细作：打入敌方内部
- 镖局：物流与信息
```

### 官场晋升体系

```
文官序列：
- 京官：翰林→六部→内阁
- 地方官：县令→知府→布政使→巡抚

武官序列：
- 底层：士兵→什长→百夫长
- 中层：千夫长→校尉→将军
- 高层：将军→元帅→封疆大吏

考核制度：
- 科举：童生→秀才→举人→进士
- 考核：政绩、军功、考核评语
- 荫庇：子孙直接受封
```

## 世界构成

### 朝堂势力

```
中央势力：
- 皇帝：最高权力
- 内阁/宰相：文官之首
- 太监：内廷势力
- 皇子：储君之争
- 权臣：霍光类型

派系斗争：
- 文官vs武官：重文轻武/重武轻文
- 皇子派系：夺嫡之争
- 清流vs浊流：路线之争
- 地方vs中央：藩镇割据

地方势力：
- 节度使/总督：地方大员
- 世家大族：地方豪强
- 江湖门派：民间力量
```

### 社会环境

```
上流社会：
- 朝堂：政治斗争中心
- 贵族圈子：婚丧嫁娶、社交
- 书院：士人聚集

市井阶层：
- 街市：商业贸易
- 酒楼茶肆：信息集散
- 青楼赌坊：灰色地带
- 帮派：底层势力

边疆环境：
- 军营：战争前线
- 城池：攻防焦点
- 关隘：战略要地
```

### 历史规则

```
时代特征：
- 政治制度：科举、监察、赋税
- 社会风气：礼教、风俗
- 经济水平：农业、商业
- 科技水平：冷兵器、农业文明

限制约束：
- 皇权至上
- 等级森严
- 信息传递慢
- 交通不便利
```

## 经典套路

### 寒门崛起流

```
经典开局：
- 穿越到寒门/庶子/废物
- 地狱开局、被轻视
- 获得现代知识/系统
- 科举崛起、朝堂新星

关键转折：
- 院试/乡试/会试/殿试
- 进入官场、被卷入派系
- 政绩卓著、获得赏识
- 权臣打压、绝地反击
```

### 战神归来流

```
经典开局：
- 穿越为边关小兵/庶子
- 展现军事天赋
- 一次次战役积累军功
- 成为战神、重塑格局

关键转折：
- 防守战：以少胜多
- 攻城战：智取为上
- 决战：决战决胜
- 朝堂：军功震主
```

### 铁血帝王流

```
经典开局：
- 重生为皇子/藩王
- 宫廷阴谋、兄弟残杀
- 韬光养晦、积蓄实力
- 夺嫡/起兵、一统天下

关键转折：
- 获得秘密支持
- 关键战役胜利
- 登基大统
- 开创新的盛世
```

### 穿越改革流

```
经典开局：
- 穿越到历史节点
- 利用现代知识改革
- 遭遇旧势力阻挠
- 推动历史进程

关键转折：
- 军事改革：武器、训练
- 经济改革：商业、税务
- 教育变革：开学堂、办报纸
- 与保守派决战
```

## 势力冲突

### 朝堂博弈

```
类型：
- 夺嫡之争：皇子夺位
- 党争：不同派系争斗
- 君臣博弈：皇权vs相权
- 改革vs守旧：变法运动

核心矛盾：
- 权力分配
- 路线之争
- 利益再分配
- 生存与前程
```

### 战场厮杀

```
类型：
- 攻防战：城池争夺
- 野战：决战胜负
- 偷袭战：出其不意
- 围点打援：战术经典

核心矛盾：
- 胜负存亡
- 军功归属
- 敌我力量对比
- 天时地利人和
```

## 女性角色设计

### 历史女性类型

```
大家闺秀：
- 世家小姐：知书达理、谋略深远
- 将门虎女：武艺高强、巾帼英雄
- 宫廷女子：皇后妃嫔、各有算盘

江湖女子：
- 女侠：剑术高超、侠义心肠
- 青楼女子：才情出众、身份复杂
- 医女：医术精湛、救人无数

设计原则：
- 要有独立人格（不是只为男主）
- 要有谋略/能力
- 要避免工具化
- 感情线要符合时代背景
```

## 设计检查清单

### 历史背景
- [ ] 时代特征是否准确？
- [ ] 制度描写是否合理？
- [ ] 社会风貌是否真实？

### 权谋描写
- [ ] 政治斗争是否有层次？
- [ ] 智谋是否合理？
- [ ] 派系设定是否清晰？

### 人物
- [ ] 主角是否有成长线？
- [ ] 女性角色是否独立？
- [ ] 配角是否有自己故事？

### 创新
- [ ] 是否有独特切入点？
- [ ] 是否避免套路化？
- [ ] 是否有记忆点？

### 历史感
- [ ] 语言风格是否契合？
- [ ] 称呼/礼节是否准确？
- [ ] 是否有历史质感？
