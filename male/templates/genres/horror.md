# 恐怖灵异题材模板

## 类型特征

恐怖灵异小说是以灵异事件、恐怖副本、茅山道术为核心，融合中式恐怖、逃生挑战与网文爽感的作品。

**核心要素：**
- 中式恐怖：鬼魂、僵尸、妖怪、民间禁忌
- 恐怖副本：死亡游戏、限时逃生
- 道术破局：茅山道术、符咒、八卦
- 无限流：多副本、多世界挑战

**核心卖点：**
- 极限压迫：寂静、黑暗、未知带来的心理压迫
- 道术爽点：四宝齐出、神鬼辟易
- 逃生紧张：规则约束、绝境求生
- 反杀快感：主角反杀恶灵、找到破解方法

## 力量体系

### 道术体系

```
茅山四宝：
- 宝剑：斩妖除魔
- 符篆：驱邪镇鬼
- 咒语：沟通天地
- 八卦盘：推演天机

道术等级：
- 入门：画符→驱邪
- 初成：捉鬼→灭僵
- 小成：布阵→请神
- 大成：雷法→天师
- 圆满：天师→飞升

道术类型：
- 雷法：掌心雷、五雷轰顶
- 火法：三味真火、烈火符
- 水法：御水术、避水符
- 金法：金刚咒、护体符
- 镇魂术：定身、封印

辅助能力：
- 天眼：通灵、见鬼
- 阴阳眼：人鬼分辨
- 算命：推演吉凶
- 风水：地理堪舆
```

### 异能体系（可选）

```
感知类：
- 灵觉：感知灵体存在
- 共情：感受鬼魂情绪
- 预警：预知危险

战斗类：
- 武艺：近战能力
- 法器：特殊武器
- 血统：特殊血脉加成

辅助类：
- 符箓：制作和使用符咒
- 阵法：布置困阵杀阵
- 丹药：临时增强/治疗
```

### 恐怖副本设定

```
副本类型：
- 凶宅类：老宅、鬼屋
- 校园类：灵异传说
- 医院类：停尸房、废弃病房
- 古墓类：陵墓、地下墓穴
- 都市传说：特定地点

副本规则：
- 生存规则：必须遵守的规则
- 逃生条件：完成条件才能离开
- 时间限制：限时逃生
- 奖励惩罚：通关奖励/失败惩罚

副本难度：
- 简单：低风险、低奖励
- 普通：中等风险、中等奖励
- 困难：高风险、高奖励
- 噩梦：极高风险、极品奖励
```

## 世界构成

### 都市灵异世界

```
现实世界：
- 普通人：不知道灵异存在
- 特殊人群：能看到鬼的人
- 灵异事件：都市传说

灵异场所：
- 凶宅：死过人的地方
- 废弃建筑：烂尾楼、废弃医院
- 阴气之地：乱葬岗、古战场
- 阴阳交界：两界缝隙

灵异生物：
- 鬼魂：枉死、怨气、执念
- 僵尸：白僵、黑僵、毛僵、飞僵
- 妖怪：动物修炼、邪祟
- 邪物：古曼童、养小鬼
```

### 势力格局

```
正道势力：
- 茅山派：正统道术
- 龙虎山：天师府
- 阁皂山：符箓正宗
- 民间散修：各有传承

邪道势力：
- 邪修：修炼邪术
- 养鬼派：豢养恶鬼
- 邪教：邪教组织
- 妖修：妖怪势力

官方组织：
- 超自然事件处理局
- 特殊事件调查组
- 国家安全部门
```

### 茅山道术规则

```
使用限制：
- 修为限制：道行不够无法使用高级道术
- 媒介限制：需要符咒/法器配合
- 时机限制：某些道术需要特定时辰
- 代价限制：高级道术有代价

克制关系：
- 雷法克鬼：至阳至刚
- 火法克僵：烈火焚烧
- 符咒克邪：镇压封印
- 阵法困敌：以弱胜强

茅山禁忌：
- 画符禁忌：心不静则符不灵
- 施法禁忌：口诀手印不能错
- 捉鬼禁忌：不能滥杀无辜
- 因果禁忌：不能随意介入
```

## 经典套路

### 茅山传人流

```
经典开局：
- 获得茅山传承/血脉觉醒
- 成为茅山弟子/天师传人
- 回归都市/除魔卫道
- 都市隐藏的灵异世界

关键转折：
- 遇到厉鬼/僵尸
- 宗门任务/同道邀请
- 敌人追杀/报复
- 发现更大的阴谋
```

### 恐怖副本流

```
经典开局：
- 意外进入恐怖副本
- 必须完成规则才能存活
- 获得奖励/变强
- 逐渐发现副本真相

关键转折：
- 副本难度升级
- 队友死亡/背叛
- 隐藏boss出现
- 副本背后的秘密
```

### 双主角互补流

```
经典开局：
- 现代人+茅山道士组合
- 现代人负责调查/推理
- 道士负责捉鬼/道术
- 共同解决灵异事件

关键转折：
- 能力互补应对危机
- 道术+现代科技的结合
- 生死与共的羁绊
- 揭开灵异世界真相
```

### 灵异直播流

```
经典开局：
- 灵异主播/探险博主
- 直播灵异事件
- 意外卷入真实灵异
- 粉丝见证/声名大噪

关键转折：
- 直播中的恐怖事件
- 粉丝变成受害者
- 直播救人/涨粉
- 成为真正的驱魔人
```

## 势力冲突

### 正邪对立

```
类型：
- 道vs魔：茅山vs邪修
- 正vs邪：正道vs邪教
- 人vs鬼：人与恶灵
- 秩序vs混乱：灵异秩序维护者vs破坏者

核心矛盾：
- 道统之争
- 信仰冲突
- 利益争夺
- 生死对立
```

### 恐怖副本冲突

```
类型：
- 生存者vs鬼魂：生死搏杀
- 生存者vs生存者：竞争/合作
- 隐藏boss：最终威胁
- 副本意志：副本本身的规则

核心矛盾：
- 生存压力
- 资源争夺
- 信任与背叛
- 规则与自由
```

## 女性角色设计

### 灵异女性类型

```
正道女性：
- 女修士：道术高强
- 女天师：茅山传人
- 女警察：灵异案件调查
- 女记者：灵异事件调查

邪道女性：
- 女鬼：厉鬼/怨魂
- 女妖：狐狸精/蛇妖
- 女邪修：修炼邪术

普通人女性：
- 女主角：能看到鬼的普通人
- 女配角：被卷入灵异事件的普通人

设计原则：
- 要有独立人格
- 要有自己的故事线
- 要避免工具化
- 恐怖氛围中的情感线
```

## 设计检查清单

### 恐怖氛围
- [ ] 压迫感是否足够？
- [ ] 悬念是否到位？
- [ ] 恐怖≠血腥猎奇？

### 道术设定
- [ ] 道术体系是否自洽？
- [ ] 克制关系是否合理？
- [ ] 使用代价是否存在？

### 人物
- [ ] 主角是否有成长？
- [ ] 女性角色是否独立？
- [ ] 配角是否有故事？

### 爽点设计
- [ ] 道术破局是否痛快？
- [ ] 反杀恶灵是否爽快？
- [ ] 逃生成功是否有成就感？

### 创新
- [ ] 是否有独特恐怖设定？
- [ ] 是否避免套路化？
- [ ] 是否有记忆点？
