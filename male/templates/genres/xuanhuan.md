# 玄幻题材模板

## 类型特征

玄幻小说是以中国古代神话传说和修真文化为基底，融合异世大陆、修炼升级、宗门争斗等元素的作品。

**核心要素：**
- 修炼体系：境界递进、功法技能
- 世界观：宗门/世家/王朝/散修并立
- 冲突：资源争夺、宗门恩怨、血脉传承
- 主线：废材逆袭、不断变强、最终登顶

## 力量体系

### 修炼境界（标准九阶）

```
基础境界：
1. 淬体境：锻炼肉身、气血初成
2. 练气境：引气入体、丹田初开
3. 筑基境：丹田稳固、可纳外物
4. 金丹境：丹成无悔、实力飞跃
5. 元婴境：元婴出窍、神游太虚
6. 化神境：神魂化虚、操控天地
7. 大乘境：接近真仙、呼风唤雨
8. 渡劫境：渡天劫、浴火重生
9. 真仙境：超凡入圣、长生不老

每境界分初期/中期/后期/巅峰
```

### 力量差距表现

```
同境界：
- 小境界差距：可战、需付出代价
- 大境界差距：几乎不可战胜

越级挑战：
- 小境界越级：可能、需有手段
- 大境界越级：极难、需金手指/机缘

战力影响因素：
- 功法等级
- 法宝强弱
- 战斗经验
- 体质血脉
- 秘术底牌
```

## 势力体系

### 主要势力类型

```
正道宗门：
- 特征：传承有序、规矩森严
- 内部：派系竞争、师徒传承
- 冲突：正邪之争、资源争夺

魔道势力：
- 特征：行事狠辣、不择手段
- 内部：强者为尊、弱肉强食
- 冲突：魔道内斗、与正道对立

世家大族：
- 特征：血脉传承、资源丰厚
- 内部：嫡庶之争、继承人之争
- 冲突：家族利益、联姻博弈

王朝帝国：
- 特征：国家机器、皇权至上
- 内部：皇权与宗门博弈、文臣武将争斗
- 冲突：统一战争、改朝换代

散修势力：
- 特征：无依无靠、独自摸索
- 生存：野外历练、秘境探险
- 冲突：被势力压迫、争夺残羹
```

### 势力冲突设计

```
经典冲突模式：
- 正道 vs 魔道：理念冲突、千年恩怨
- 宗门 vs 皇室：主导权争夺
- 新秀 vs 老牌：话语权更替
- 嫡系 vs 旁支：继承权争夺
- 散户 vs 势力：生存空间之争
```

## 世界构成

### 地理环境

```
大陆设定：
- 中州：修炼中心、资源最丰
- 东域：世家林立、保守封闭
- 西域：魔道聚集、危险重重
- 南疆：蛮荒之地、妖兽横行
- 北荒：苦寒之所、强者流放

特殊区域：
- 秘境：独立空间、机缘所在
- 遗迹：上古传承、危险与机遇并存
- 禁地：无人能入、必有古怪
- 绝境：九死一生、历练圣地
```

### 社会规则

```
修炼资源：
- 灵石：通用货币、分品级
- 灵药：炼丹材料、天生地养
- 妖兽：内丹/皮毛/血肉皆有用
- 功法：传承根本、分天地玄黄

交易法则：
- 以物换物
- 灵石结算
- 势力间：利益交换

弱肉强食：
- 实力为尊
- 规则内解决问题
- 暗杀/争夺屡见不鲜
```

## 经典套路

### 废材逆袭型

```
经典开局：
- 主角天生体质特殊（无法修炼/被封印）
- 遭遇重大变故（退婚/羞辱/家族抛弃）
- 获得金手指（传承/系统/老爷爷）
- 逐步崛起、证明自己

关键转折：
- 首次公开比赛崭露头角
- 秘境探险获得机缘
- 宗门大比震惊众人
- 最终大战一鸣惊人
```

### 退婚流

```
结构：
1. 未婚妻/未婚夫家上门退婚
2. 主角受辱、发誓雪耻
3. 多年后主角崛起
4. 对方后悔/求复合
5. 主角态度转变（打脸/释然）

变体：
- 不是真的退婚，而是冷嘲热讽
- 对方态度傲慢、羞辱主角
- 后期反转打脸
```

### 宗门崛起型

```
结构：
1. 主角加入小势力/创建势力
2. 势力遭遇危机、濒临灭亡
3. 主角带领势力崛起
4. 与大势力争锋
5. 最终成为顶尖势力

关键点：
- 势力要有独特特色
- 成员要有成长
- 冲突要有层次
```

## 伏笔类型

### 体质血脉伏笔

```
常见体质：
- 特殊血脉：神兽血脉/上古血脉
- 废物体质：实为特殊体质
- 双重体质：表里不一
- 成长体质：逐步觉醒

伏笔设计：
- 初期：体质被压制/无法显现
- 中期：偶有异象、暗示强大
- 后期：体质觉醒、大放异彩
```

### 身世伏笔

```
常见身世：
- 遗孤：实为大人物之后
- 弃子：被家族低估/迫害
- 转世：前世强者归来
- 棋子：被人操控、不知真相

伏笔设计：
- 初期：身世成谜、不被重视
- 中期：陆续有人试探/暗示
- 后期：真相大白、身世震惊
```

### 传承伏笔

```
常见传承：
- 远古大能：留下功法/法宝
- 失传秘术：突然出现
- 上古宗门：遗迹传承
- 天道眷顾：特殊机缘

伏笔设计：
- 初期：主角偶得、不知来历
- 中期：传承有灵、逐步显现
- 后期：传承完整、揭示因果
```

## 设计检查清单

### 世界观
- [ ] 境界体系清晰、层次分明？
- [ ] 势力分布合理、有冲突潜力？
- [ ] 地理环境有辨识度？
- [ ] 资源体系自洽？

### 套路运用
- [ ] 开局是否吸引人？
- [ ] 逆袭是否有层次感？
- [ ] 爽点分布是否合理？

### 伏笔
- [ ] 体质伏笔是否有前后呼应？
- [ ] 身世伏笔是否埋设自然？
- [ ] 传承伏笔是否有因果？

### 创新点
- [ ] 是否有独特设定？
- [ ] 是否避免同质化？
- [ ] 是否有记忆点？
