# 穿越穿书题材模板

## 类型特征

穿越穿书小说是以穿越异世界、系统任务、攻略逆袭为核心，融合无限流、副本挑战与网文爽感的作品。

**核心要素：**
- 穿越设定：魂穿/身穿/重生到异世界
- 系统任务：任务驱动、积分兑换
- 攻略逆袭：改写原著命运
- 多世界：每个世界独立设定

**与系统流区别：**
- 系统是辅助工具
- 核心是穿越到原著世界
- 任务是攻略关键角色/改写剧情

**与玄幻区别：**
- 有明确原著剧情可预知
- 任务目标明确
- 多世界切换

## 力量体系

### 系统设定

```
系统类型：
- 任务系统：发布任务→完成奖励
- 签到系统：每日签到→累计奖励
- 扮演系统：扮演角色→获得能力
- 宠兽系统：收集宠物→培养进化
- 商城系统：积分兑换→稀有道具
- 加点系统：自由分配属性点

系统规则：
- 任务类型：日常/支线/主线/隐藏
- 惩罚机制：失败惩罚（抹杀/属性下降/道具扣除）
- 升级方式：完成任务/签到积累
- 商城定价：道具稀有度定价
```

### 能力获取

```
穿越前携带：
- 随身空间：储物/种植
- 系统面板：属性/任务/商城
- 原著记忆：剧情预知

穿越后获得：
- 原著角色能力
- 系统奖励能力
- 攻略对象能力

成长方式：
- 任务奖励：经验/道具/技能
- 攻略进度：好感度奖励
- 副本通关：装备/技能
- 隐藏成就：特殊奖励
```

### 世界设定

```
原著类型：
- 玄幻/仙侠：修炼升级
- 都市/言情：现代都市
- 古代/宫斗：历史背景
- 校园/青春：学校生活
- 末世/丧尸：末日生存
- 西幻/异世：魔法世界

世界规则：
- 原著剧情：可预知的走向
- 原著人物：关键角色设定
- 隐藏剧情：未写明但存在的秘密
- 原著结局：原本的HE/BE线
```

## 世界构成

### 多元世界

```
主世界（核心）：
- 现代社会/原生世界
- 系统发源地
- 最终回归地

副本世界（任务世界）：
- 玄幻世界：宗门林立、弱肉强食
- 都市世界：豪门恩怨、商战
- 古代世界：王朝更迭、江湖纷争
- 校园世界：青春成长、感情纠葛
- 末世世界：生存挑战、丧尸横行

特殊世界：
- 禁区/禁地：危险但奖励丰厚
- 隐藏世界：触发隐藏任务
- 真实世界：NPC自我意识觉醒
```

### 角色身份

```
穿书身份：
- 原著炮灰：被打压/被嘲笑/被利用
- 原著反派：人人喊打/立场对立
- 原著路人：默默无闻/可有可无
- 原著女主/女配：女频穿书

身份设定：
- 表面身份：原主的社会地位
- 实际身份：穿越者的真实来历
- 隐藏身份：特殊能力/背景
```

### 系统规则

```
任务设计：
- 主线任务：推进剧情必做
- 支线任务：可选完成
- 隐藏任务：触发条件特殊
- 日常任务：每日刷新

奖励机制：
- 经验值：提升等级
- 积分：商城消费
- 道具：装备/消耗品
- 技能：主动/被动技能
- 称号：特殊标识

惩罚机制：
- 抹杀：任务失败直接死亡
- 属性惩罚：降低属性
- 道具扣除：没收道具
- 好感度下降：攻略对象好感降低
```

## 经典套路

### 穿书炮灰逆袭流

```
经典开局：
- 穿成原著炮灰/恶毒女配
- 被原男主/女主打压
- 绑定逆袭系统
- 开始改写命运

关键转折：
- 攻略原男主/女主
- 截胡原主角机缘
- 打脸原主角
- 取代原主角位置
```

### 系统任务流

```
经典开局：
- 穿越到异世界
- 获得任务系统
- 开始做任务变强
- 逐渐揭开世界秘密

关键转折：
- 系统升级/进化
- 发现系统真相
- 任务失败危机
- 系统背后阴谋
```

### 攻略病娇/反派流

```
经典开局：
- 穿成反派/被反派针对
- 系统要求攻略目标
- 攻略对象从恨到爱
- 改写反派结局

关键转折：
- 攻略对象黑化/病娇
- 原著剧情崩坏
- 隐藏剧情触发
- 真爱vs任务
```

### 多人世界流

```
经典开局：
- 穿越到多个世界
- 每个世界有独立任务
- 世界之间有关联
- 逐渐发现真相

关键转折：
- 世界融合危机
- NPC觉醒
- 主神/管理员阴谋
- 回归原本世界
```

## 势力冲突

### 任务世界冲突

```
类型：
- 原著主角vs穿越者：资源争夺
- 攻略对象vs攻略对象：多角冲突
- 穿越者vs穿越者：竞争关系
- 原著势力：宗门/家族/国家

核心矛盾：
- 任务目标冲突
- 攻略对象归属
- 资源有限争夺
- 原著剧情vs改写
```

### 系统层面冲突

```
类型：
- 任务竞争：同一目标多个穿越者
- 系统对决：不同系统拥有者
- 主神博弈：高层势力争夺
- 玩家vs管理员：权限之争

核心矛盾：
- 任务完成优先级
- 系统升级资源
- 世界归属权
- 自由与规则
```

## 女性角色设计

### 穿书女性类型

```
原著女性：
- 原著女主：原本的女主角
- 原著女配：恶毒女配/炮灰女配
- 原著反派：与主角对立
- 原创女性：新加入的角色

攻略设计：
- 攻略难度：高低不同
- 攻略方式：不同策略
- 攻略结果：HE/BE/开放式
- 多攻略vs单攻略

设计原则：
- 女性角色要有独立人格
- 避免工具化攻略对象
- 攻略要有情感发展
- 尊重女性的自主选择
```
