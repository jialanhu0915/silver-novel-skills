# DND 奇幻 / 硬核 DND 题材模板

> **题材定位：** 服务于主动点名"DND"的读者和作者 —— 默认读者具备桌游相关认知，对多重宇宙、职业体系、九宫格阵营、法术位机制有期待。模板**保留 DND 风味**（职业身份认同、阵营哲学、资源管理、小队羁绊），但配套**雷区检查清单**防止小说变成桌游战报。

---

## 类型特征

DND 奇幻小说是以**多重宇宙架构**为背景、以**职业身份认同**为核心、以**九宫格阵营哲学**为道德框架、以**小队冒险**为叙事结构的作品。世界观根基来自《龙与地下城》桌游体系（D&D 5e / 经典模组），但允许完全自创位面。

**核心要素：**
- **多重宇宙**：位面、星界、物质位面、灵界等可自由组合
- **职业体系**：战士、法师、圣武士、盗贼、牧师、术士、德鲁伊、吟游诗人、武僧等
- **九宫格阵营**：守序善良 → 混乱邪恶 的行为逻辑纲领
- **法术机制**：环位 0-9、每日法术位准备、资源管理博弈
- **小队冒险**：地城探索、模组任务、阵营冲突、合作与背叛

**核心卖点：**
- 职业成长的节点感：从学徒到传奇的进阶
- 战术博弈的智谋：有限资源下的绝境翻盘
- 宝物与装备的获得感：+1 短剑到神器
- 阵营冲突的哲学碰撞：守序善良 vs 混乱邪恶的道德张力
- 探索未知的发现感：地城、遗迹、未知位面

**与玄幻（xuanhuan.md）/仙侠（xianxia.md）的根本区别：**

| 维度 | 玄幻/仙侠 | DND 奇幻 |
|------|----------|----------|
| 世界观 | 单一大陆 + 修真界分层 | **多重宇宙 + 多位面** |
| 力量体系 | 修为等级 + 灵气/法力 | **职业身份 + 法术位 + 阵营** |
| 核心矛盾 | 宗门争斗 / 正邪大战 | **小队冒险 + 阵营冲突 + 道德困境** |
| 主角定位 | 孤独求道者 / 天选之人 | **小队成员 / 冒险者** |
| 爽点侧重 | 突破碾压 / 装逼打脸 | **职业成长节点 + 战术智谋 + 道德抉择** |

## 力量体系

### 职业体系（核心创新）

DND 力量体系不是"修为等级"，而是**职业身份 + 能力组合 + 行为模式**的三位一体。

```
核心职业与定位：

战士（Martial）：
- 定位：近战输出、肉盾、武器大师
- 身份认同：纪律、武技、战术纪律
- 行为模式：直面冲突、用实力说话
- 关键约束：依赖装备与训练

法师（Wizard）：
- 定位：高伤害法术、远程施法、学者
- 身份认同：知识、研究、学术传统
- 行为模式：依赖法术位（资源有限）、备战型
- 关键约束：法术位耗尽 = 战斗力骤降

圣武士（Paladin）：
- 定位：近战 + 神术、坦克、道德标杆
- 身份认同：誓言、神祇、信仰体系
- 行为模式：受阵营约束（守序善良 = 不可滥杀）
- 关键约束：违背誓言 → 失去神术

盗贼（Rogue）：
- 定位：潜行、机关、爆发伤害
- 身份认同：街头智慧、自由、独立
- 行为模式：偷袭、先攻优势、闪避
- 关键约束：依赖环境与先手

牧师（Cleric）：
- 定位：治疗、辅助、神术施法者
- 身份认同：神祇信仰、治愈、引导
- 行为模式：随神祇领域变化（光明神 vs 战争神）
- 关键约束：信仰动摇 = 能力下降

术士（Sorcerer）：
- 定位：法术输出、先天血脉
- 身份认同：血脉传承、命运、天赋
- 行为模式：法术位少但威力强、不可学新法术
- 关键约束：血统限制（龙裔术士 ≠ 精灵术士）

德鲁伊（Druid）：
- 定位：自然法术、变形、野兽伙伴
- 身份认同：自然平衡、万物有灵
- 行为模式：变形战斗、自然领域能力加成
- 关键约束：金属铠甲禁用（传统约束）

吟游诗人（Bard）：
- 定位：辅助、控场、情报
- 身份认同：故事、艺术、社交
- 行为模式：魅力为先、技能广泛
- 关键约束：依赖表演与社交场合

武僧（Monk）：
- 定位：近战、闪避、拳法
- 身份认同：修行、戒律、武道
- 行为模式：徒手战斗、闪避强化、气功
- 关键约束：依赖气值（Ki）资源

扩展职业（可选）：
- 游侠（Ranger）：特定敌人加成、追踪、野外
- 术士（Sorcerer）：前文已列
- 邪术师（Warlock）：与异位面存在契约
- 魔战士（Magus）：法武合一
- 召唤师（Summoner）：位面召唤
```

**职业设计原则：**
- 职业不只是技能清单，更是**身份认同与世界观**
- 圣武士的守序善良不只是能放什么技能，更是他**如何做决策、如何面对道德困境**
- 法师不只是能施法，更是**学术传统的继承者**
- 职业身份影响 NPC 看待主角的方式（守序善良的圣武士 = 信任基础）

**职业 vs 修为的区别：**
- 修为：单线递进（炼气→筑基→金丹...）
- 职业：身份锚定（你永远首先是圣武士/法师/盗贼，然后才有等级）
- 这意味着 DND 主角的成长不是"修为越来越高"，而是"对这个职业的理解越来越深"

### 九宫格阵营（道德框架）

DND 标志性的阵营系统是**行为逻辑纲领**，不是道德标签。

```
九宫格定义（简表）：

      善良          中立          邪恶
守序  守序善良     守序中立     守序邪恶
中立  中立善良     绝对中立     中立邪恶
混乱  混乱善良     混乱中立     混乱邪恶

各阵营行为逻辑：

守序善良（LG）— 圣武士典型：
- 核心：荣誉 + 善良 + 规则
- 行为："荣誉即一切"
- 道德困境：法律与良善冲突时？（奴隶制合法的城邦）
- 主角参考：《黑暗精灵》崔斯特（早期）

守序中立（LN）— 审判者典型：
- 核心：秩序 > 一切
- 行为："规则就是规则"
- 道德困境：邪恶但合法的秩序？
- 主角参考：严格法师、戒律牧师

守序邪恶（LE）— 魔鬼典型：
- 核心：秩序 + 邪恶
- 行为："以灵魂为燃料发动血战、通过律法与契约治理领地"
- 道德困境：魔鬼也有"信用"——这是比混乱邪恶更高级的反派
- 主角参考：《血族》式的律法型大反派

中立善良（NG）：
- 核心：善良 > 法律/混乱
- 行为："该帮就帮"
- 道德困境：与守序善良的圣武士可能冲突

绝对中立（TN）— 德鲁伊典型：
- 核心：自然平衡
- 行为："不让天平倾斜"
- 道德困境：不介入邪恶？

中立邪恶（NE）— 阴谋家典型：
- 核心：自利 + 邪恶
- 行为："不择手段"
- 道德困境：比混乱邪恶更阴险

混乱善良（CG）— 游侠典型：
- 核心：善良 + 自由
- 行为："按良心，不按法律"
- 道德困境：守序善良觉得 CG 是惹麻烦

混乱中立（CN）：
- 核心：自由 > 一切
- 行为："我想怎样就怎样"
- 道德困境：不稳定的盟友

混乱邪恶（CE）— 恶魔典型：
- 核心：毁灭 + 自由
- 行为："烧杀抢掠，无序破坏"
- 道德困境：力量最直接的邪恶，但最无脑
```

**阵营设计原则：**
- **阵营是行为模式，不是标签**：作者必须能用 1-2 句话说出该角色"在这种情境下会怎么做"
- **阵营不是一成不变的**：圣武士可能堕落为守序邪恶（违反誓言后黑化），混乱中立可能稳定为守序善良（找到信仰）
- **同阵营 ≠ 同行为**：两个守序善良的圣武士可能因为信仰不同的神而做出相反决定
- **不要把阵营写成标签**：如果删掉阵营描述，行为完全不变 → 阵营就是装饰

**阵营冲突经典场景：**
- LG vs LE：道德碰撞（最经典的善恶对决）
- LG vs CG：法律与良心的冲突（解放奴隶 vs 遵守当地法律）
- LN vs CN：秩序与自由的永恒对立
- 队友之间阵营张力：圣武士（LG）+ 盗贼（CN）= 持续摩擦

### 职业 / 阵营自创方法论

上面给的 9 个核心职业 + 9 个经典阵营只是**基础素材**，完全自创一套职业体系 / 原创阵营变体是 DND 网文的核心竞争力。以下是 4 步方法论：

**步骤 1：确定职业主题 / 阵营主轴**

职业体系必须有一个核心主题，所有进阶命名都围绕这个主题展开。

```
主题示例：
- 「决斗者」：剑客 → 决斗者 → 仲裁者 → 圣剑使 → ...（围绕"一对一对决"展开）
- 「锻造师」：学徒 → 工匠 → 大师 → 宗师 → 传奇匠神 → ...（围绕"锻造神器"展开）
- 「守誓者」：立誓者 → 守誓者 → 仲裁者 → 大誓师 → 永恒之誓 → ...（围绕"不可破的誓言"展开）

阵营主轴示例：
- 守序善良的亚型：荣誉骑士（LG-H）vs 救赎圣武士（LG-S）—— 同一阵营的不同侧重
- 混乱善良的亚型：自由游侠（CG-F）vs 革命圣徒（CG-R）
```

**步骤 2：9 个职业名称的递进**

从学徒到传奇前，名称必须呈现"由学徒到大师"的递进：

```
递进逻辑：
- 学徒期：刚入门，对职业身份初步认同
- 初阶：能完成基础任务，对职业有实战理解
- 中阶：已立住脚，是该职业的合格代表
- 高阶：被同行认可，传授门徒
- 传奇前：达到凡人极限，离神祇只差一步

避免：
- 9 个名称全是形容词堆砌（"守剑者 → 守剑士 → 守剑师"——语义重复）
- 9 个名称主题跳跃（"学徒 → 战士 → 祭司"——内在逻辑断裂）
- 9 个名称都是"形容词 + 名词"（"光之剑客 → 暗之剑客"——缺乏职业身份认同）
```

**步骤 3：绑定职业身份代价**

**关键约束：每个职业的"代价"必须与职业主题绑定，不能泛化为"每升一级都掉血"。**

```
绑定示例（决斗者主题）：
- 学徒剑客：代价 = 无（普通人）
- 决斗者：代价 = 不能拒绝任何一对一决斗邀请（职业荣誉）
- 仲裁者：代价 = 必须见证每场重要决斗的公正
- 圣剑使：代价 = 自己不能主动发起决斗
- 传奇前：代价 = 已成为决斗的代名词，被仇家永远追杀

反例（不要这样）：
- 学徒：代价 = 偶尔会手抖
- 决斗者：代价 = 经常手抖
- 仲裁者：代价 = 总是手抖
（这三条是同一代价的重复，没有递进）
```

**步骤 4：设计职业 / 阵营克制关系**

DND 的克制不是单向的"高级压制低级"，而是**身份之间存在特殊对抗**。

```
克制关系示例：
- 圣武士 vs 吸血鬼：圣武士的"驱邪"对吸血鬼天然克制（职业身份克制）
- 混乱善良 vs 守序善良：道义拷问（不是战斗克制，是哲学克制）
- 法师 vs 术士：学术传统 vs 血脉天赋（身份认同的相互轻视）
- 盗贼 vs 圣武士：信任 vs 怀疑（小队羁绊的天然摩擦）

避免：
- 简单"高级 = 强"（失去剧情张力）
- 所有职业之间互相克制（无法设计情节冲突）
- 完全没有克制关系（沦为数值比拼）
```

### 法术与资源管理

**核心原则：不要直接写"环数""法术位"等桌游术语。** 用传统分级替代 + 通过实际效果展现。

```
等级转译（小说化）：
- 1-5 级 → 学徒（刚开始冒险的冒险者）
- 6-10 级 → 初阶（已立住脚的冒险者）
- 11-15 级 → 中阶（区域知名）
- 16-20 级 → 高阶（传奇前夕）
- 21+ 级 → 传奇（已超脱凡人）

法术环数转译（小说化）：
- 0 环（戏法）→ "他能做点小把戏：点亮灯火、清洁衣物、修补破损"
- 1-2 环 → "他施展了一个低阶法术"
- 3-5 环 → "他施放了强力法术"
- 6-7 环 → "他释放了大法师才能掌握的法术"
- 8-9 环 → "这是传奇法术，凡人难以企及"

法术位机制转译：
- "他还有几个法术位" → 删掉！
- 改写："他意识到今天已经施了太多次法术，再强行施法可能会反噬"
- "他的法力储备接近枯竭" / "他感觉到法力本源开始震颤"
- "他必须在休息后才能再次施展如此强大的法术"
```

**资源管理（核心爽点）：**
- DND 的核心魅力之一是**资源管理**（法术位、卷轴、药剂、气值）
- 主角用**有限资源**解决看似无解的困境
- 这就是 DND 版的"智谋爽点"

**小说化处理原则：**
- 法术的威力差异通过**实际效果描写**来体现 —— 高级法师的火球"更大、更热、更持久"是符合认知的
- 不要强调"3 级法师只有 3 倍威力，13 级法师有 13 倍威力"
- 让读者从描述中**自然感知**强弱差异，而非读数值

### 宝物与装备体系

```
宝物分级（小说化）：
- 普通 → "一把精钢长剑"
- +1 武器 → "一把带有微弱魔法附魔的精良武器"
- 魔法物品 → "蕴含魔法力量的物品"（不需要"+1"标注）
- 稀有 → "传说级武器"
- 传奇 → "神器"
- 独有 → "这个世界仅此一件的神器"

宝物类型：
- 武器：+1 长剑、+2 战斧、传奇巨剑
- 防具：板甲、盾牌、戒指防护
- 饰品：斗篷、护符、戒指
- 卷轴：一次性法术卷轴（资源管理神器）
- 药剂：治疗药水、抗性药水、变形药水
- 法器：法杖、魔杖、权杖

小说化处理原则：
- 宝物不仅是数值加成，更是**故事驱动力**
- "这把剑是某位传奇骑士的佩剑，传说他曾在某战役中用它斩下巨龙之首"
- 宝物的获得往往伴随代价（任务、牺牲、诅咒）
- 宝物与角色绑定（圣武士 vs 诅咒剑 = 道德困境）

神器示例（保留 DND 风味，不直接复制）：
- 破誓者（曾经的圣武士之剑，被诅咒后会逼主人违背誓言）
- 元素君王之冠（召唤元素的传奇饰品，但持有者会被元素意识侵蚀）
- 影界之靴（穿越影界的装备，但每次使用都会留下部分自我在影界）
```

**情节引擎设计原则：**

DND 宝物的真正价值不在数值，而在**驱动情节**。

```
宝物的三种情节功能：
- 线索（揭示世界观真相）：一把剑的剑柄上刻有古代神祇铭文
- 目标（核心追逐物 / 麦高芬）：神器 X 是整本书的追逐核心
- 代价（塑造角色弧光）：获得神器的代价是失去某个职业能力

宝物与角色阵营的冲突设计：
- 圣武士 + 诅咒剑：誓约要求不能持有邪恶之物，但此剑是唯一能救队友的武器
- 混乱善良盗贼 + 秩序法杖：法杖让他获得力量，但每次使用都强迫他遵守秩序
- 德鲁伊 + 金属神器：德鲁伊誓言禁止金属装备，但神器是拯救森林的唯一希望

宝物获取过程的职业考验：
- 法师必须放弃学术传统才能获得某法杖（与"知识传承"的法师身份根本冲突）
- 圣武士必须在"救人"与"持剑"之间选择（直接挑战他的核心誓约）
```

## 世界构成

### 多重宇宙架构

DND 的世界观是**多重宇宙**（Multiverse）。可以完全自创位面，不必绑定费伦。

```
经典位面架构：

物质位面（Material Plane）：
- 主角所在的主要位面
- 物理规则正常
- 可以是费伦、艾伯伦、龙枪、自创大陆

内层位面（Inner Planes）：
- 元素位面（火、风、水、土）
- 能量位面（正能量、负能量）
- 恶魔深渊 / 天使山巅
- 是元素生物、神祇眷族的家园

外层位面（Outer Planes）：
- 九层地狱（守序邪恶）
- 深渊（混乱邪恶）
- 七层天堂山（守序善良）
- 混沌海（混乱善良）
- 各神祇的领域

过渡位面：
- 星界（Astral Plane）：思维位面，色彩斑斓
- 灵界（Ethereal Plane）：与物质位面重叠
- 影界（Shadowfell）：物质位面的阴暗倒影
- 梦境境（Feywild）：物质位面的魔法倒影

自创位面建议：
- 龙枪本身就是一个独立世界，不需要费伦
- 完全自创的多重宇宙反而更灵活
- 关键是位面之间要有**可穿越性**（位面旅行、召唤、传送）
```

### 阵营势力格局

```
国家/城邦示例：

守序善良型城邦：
- 法律严明但公正
- 神祇信仰虔诚
- 圣武士骑士团驻扎
- 代表：银月城、博德之门类

混乱善良型城邦：
- 自治社区、自由港
- 民主决策或部落议事
- 盗贼公会与冒险者共存
- 代表：冒险者天堂类

守序邪恶型帝国：
- 法律至上，但服务于邪恶
- 魔鬼契约、灵魂交易
- 暴政但有效
- 代表：奈瑟斯、九狱类

混乱邪恶型领地：
- 部落、魔物巢穴
- 无政府状态下的暴力
- 代表：恶魔深渊、兽人部落

绝对中立型：
- 德鲁伊圣地、自然神殿
- 不参与外界纷争
- 代表：永聚岛
```

### 冒险生态

```
冒险者公会：
- 任务发布、资源对接
- 冒险者等级认证
- 是主角的核心社交场所

地城（Dungeons）：
- 古老遗迹、巨魔巢穴、神殿
- 分层结构（地下 1 层、地下 2 层...）
- 陷阱 + 怪物 + 谜题 + 战利品

模组式任务：
- 委托：找人/找物/清怪/调查
- 时限：3 天 / 1 周 / 1 个月
- 奖励：金币 + 经验 + 声望
- 风险：死亡 / 受伤 / 失败惩罚

政治与阴谋：
- 王国 vs 公会
- 神祇 vs 神祇
- 派系 vs 派系
- 主角在不同势力间周旋
```

## 经典套路

### 传统冒险流（《黑暗精灵》范式）

```
经典开局：
- 主角是新手冒险者（学徒级）
- 接到第一份委托（清剿哥布林巢穴）
- 组队：法师 + 战士 + 圣武士 + 盗贼 + 牧师
- 完成委托 → 升级 → 接更大的委托
- 逐步卷入更大的阴谋

关键转折：
- 委托背后的真相（如：哥布林是受某个势力操控）
- 队友的牺牲或背叛
- 主角的职业身份觉醒（"我为什么是圣武士"）
- 最终面对真正的反派（不仅是个哥布林王）

适用：单元剧结构、小队羁绊、经典奇幻冒险
参照：《黑暗精灵三部曲》《冰风谷三部曲》
```

### DND 规则解构流（《博德之门：我，九狱银行家》范式）

```
经典开局：
- 主角是 DND 知识者（玩家穿越/觉醒）
- 深度内化规则为世界观底层逻辑
- "骰子检定"升华为命运神骰的具象化意志
- "职业/种族/背景"转化为真实的生存逻辑
- "模组式冒险"解构为结构性叙事单元

关键转折：
- 主角用规则知识规避风险（提前准备 +5 长剑）
- 但面对真正的高阶冒险，规则知识也不够用
- 主角从"利用规则"成长到"理解规则背后的精神"
- 规则本身就是角色成长的隐喻

适用：硬核 DND 玩家向、深度设定党、想展示规则魅力
```

### 伪 DND / 轻 DND 流（《深渊主宰》《红袍法师》范式）

```
经典开局：
- 完全自创世界观
- 借用 DND 元素（种族、职业、阵营、法术）
- 但不绑定任何官方位面
- 简化或放弃严格的等级/规则体系
- "DND 风味 + 完全自由的世界观"

为什么这是主流选择？
- 真 DND 很难写成小说，低等级滥强手段多，高等级对战难写
- 自己大幅魔改的伪 DND 要好写得多
- 读者大多数是泛奇幻读者，不是 DND 硬核玩家

适用：绝大多数网文作者 —— 保留 DND 风味，但不受规则束缚
参照：《深渊主宰》《红袍法师》《法师故事》
```

### DND + 热门题材融合流

```
DND + 都市：现代背景下的 DND 元素
DND + 种田/领主：魔导工业化、领地建设
DND + 穿越/系统：主角带 DND 知识穿越异界
DND + 异兽文：主角是龙、恶魔等非人种族
DND + 恋爱：冒险小队中的感情线（参考《博德之门 3》的浪漫剧情）
```

## 势力冲突

### 阵营冲突（核心冲突类型）

```
守序善良 vs 守序邪恶：
- 经典善恶对决
- 圣武士 vs 魔鬼领主
- 道德立场明确但复杂（LE 也有自己的逻辑）

守序 vs 混乱：
- 法律 vs 自由
- 守序善良 vs 混乱善良 可能因此冲突
- 哲学深度的对立

善良 vs 邪恶：
- 同阵营内部未必一致
- 守序善良的圣武士可能觉得混乱善良是麻烦制造者
- 守序邪恶的魔鬼可能觉得混乱邪恶是没脑子的暴徒

中立派的选择：
- 绝对中立的德鲁伊可能不介入任何阵营冲突
- 但当平衡被打破时，必须选边站
```

### 派系冲突（冒险者 vs 政治）

```
冒险者公会 vs 王国：
- 冒险者的自由 vs 王国的法律
- 主角可能两边都要应付

神祇冲突：
- 不同神祇的教会互相竞争
- 神祇本身可能在对立阵营
- 主角可能服务于不同神祇的盟友

种族冲突：
- 人类王国 vs 精灵森林
- 矮人山脉 vs 兽人草原
- 提夫林（魔族混血）被所有种族歧视
```

### 跨位面冲突

```
位面入侵：
- 恶魔深渊的入侵
- 异位面存在的降临
- 凡人守土有责

召唤与契约：
- 邪术师与异位面存在的契约
- 召唤仪式失控
- 位面壁垒被打破

位面旅行：
- 主角穿梭于多个位面
- 不同位面有不同的规则
- 是冒险的升级版
```

## 女性角色设计

### 小队女性职业

```
女圣武士：
- 类型：高贵的誓言者
- 道德张力：守序善良的严格 vs 真实世界的灰色
- 弧光：是否在残酷世界中保持信仰？
- 反例：写成无趣的道德标杆 → 加人情味

女法师：
- 类型：学术传统的女性继承者
- 身份张力：在男权法师塔中的地位
- 弧光：从学术天才到真正的法师
- 反例：写成"女强文"工具 → 加内心戏

女盗贼：
- 类型：街头智慧的自由灵魂
- 身份张力：守序城邦中被通缉
- 弧光：信任问题（被背叛过？）
- 反例：写成花瓶 → 给独立动机

女牧师：
- 类型：信仰的承载者
- 身份张力：信仰是否被现实考验？
- 弧光：信仰动摇后的重建
- 反例：写成传声筒 → 加私心

女德鲁伊：
- 类型：自然守护者
- 身份张力：文明 vs 自然的永恒冲突
- 弧光：是否能为文明做出妥协？
- 反例：写成"自然之母" → 加个人化动机
```

### 跨阵营女性角色

```
女魔鬼（守序邪恶）：
- 类型：律法型反派
- 魅力点：魔鬼也有"信用"——比混乱邪恶更高级的反派魅力
- 弧光：在守序邪恶的逻辑中是否有人性残留？
- 经典参考：《血族》中的优雅反派

女恶魔（混乱邪恶）：
- 类型：暴力型反派
- 魅力点：原始的毁灭欲
- 弧光：混乱邪恶很少有弧光——更合适作为短期反派

女圣武士堕落者（守序善良 → 守序邪恶）：
- 经典堕落流
- 弧光：从信仰到怀疑到堕落的完整过程
- 关键：必须有可共情的堕落理由
- 反例：突然黑化 → 需要铺垫

女德鲁伊（绝对中立）：
- 类型：自然守护者
- 魅力点：万物有灵的哲学
- 弧光：在平衡被打破时必须做出选择
- 反例：写成冷漠 → 加人情味
```

### 弧光设计原则

```
职业弧光：
- 起步：对职业身份的初步认同（学徒阶段）
- 转折：对这个职业的真正理解（实战考验）
- 升华：成为这个职业的代表（不是数值最强，而是精神最强）
- 关键检查：弧光是否与"职业身份"绑定？

阵营弧光：
- 起步：稳定的阵营认同
- 转折：阵营信仰的动摇（道德困境）
- 升华：阵营的重塑或坚守
- 关键检查：是否避免了"突然改变阵营"的突兀？

跨位面弧光：
- 起步：只在物质位面活动
- 转折：穿越到其他位面，看到不同的规则
- 升华：理解多重宇宙的本质
- 关键检查：位面穿越是否有心理代价？
```

### 锚定物设计（防止人物扁平化）

```
正向锚定：
- 职业锚定：女圣武士锚定誓言（"我曾向骑士团长发过誓"）
- 阵营锚定：守序善良角色锚定同伴信任（"我答应过保护他们"）
- 神祇锚定：女牧师锚定神祇回应（"她的祈祷从未被忽视"）
- 城邦锚定：女法师锚定学术塔的传承（"那座塔见证了我所有的研究"）

反向锚定（适合邪教女主、堕落角色）：
- 执念型锚定：女盗贼锚定"曾被抛弃的孤儿院"
- 罪责型锚定：女堕落圣武士锚定"被我害死的队友"
- 复仇型锚定：女刺客锚定"灭门之仇"

关键约束：
- 锚定物不能"被抽走即崩溃"——必须有"锚定物的锚定物"
- 例：她锚定在"妹妹"，但当妹妹死亡后，她必须有第二层锚
- 反例：锚定物一破就崩溃 → 人物弧光单薄
- 检查：她的执念是否有"被满足的可能"？永远满足不了 = 单薄
```

### 独立目标（避免工具化）

```
女性同伴必须有独立目标，不能只服务于主角：

正向示例：
- 她有自己的职业进阶目标（即使与主角不同）
- 她有自己的阵营立场（可能与主角冲突）
- 她有自己的秘密（不为推动主角剧情而存在）
- 她有自己的冒险（与主角同时或交替进行）

检查清单：
- [ ] 她的目标是否在主角剧情之外能独立成立？
- [ ] 她的目标是否影响主角的决策？
- [ ] 如果删除她，主角剧情是否还成立？
  - 仍成立 → 她是工具人
  - 不成立 → 她有独立价值
```

## 爽点设计

DND 题材的爽点更接近**传统升级流**，但有自己独特的风味。

### 五类爽点

```
1. 职业成长的节点爽：
   - 从弱小的新手冒险者成长为传奇英雄
   - 获得新能力、新专长、新法术的节点感
   - 小说化处理：不要写成"升级了，获得新技能"
   - 正确：通过实战突破展现 —— 在生死关头突然领悟了某个法术的更高层运用
   - "他的导师在三十年前达到这个境界，而他在生死搏杀中找到了那条路"

2. 战术博弈的智谋爽：
   - 用有限的资源解决看似无解的困境
   - 展现主角的智谋和准备
   - 小说化处理：利用环境、情报、准备、队友配合
   - 反例："一个智商低、资源贫乏的低级法师反杀高智商法师"是不合理的
   - 正确："他用了三天时间研究地形，准备了七卷轴，在对方必经之路设下陷阱"

3. 宝物与装备的获得爽：
   - 获得神器、传奇装备的获得感
   - 小说化处理：宝物不仅是数值，更应该有故事和代价
   - "这把剑曾是某位传奇骑士的佩剑，他为了保护村庄而牺牲，剑的传承落到了他手中"

4. 阵营冲突的哲学爽：
   - 守序 vs 混乱、善良 vs 邪恶的哲学碰撞
   - 不同阵营角色的行为逻辑冲突
   - 小说化处理：不要把阵营写成标签，而要写成角色做出选择时的内在张力
   - "他知道放过这个敌人是更善良的选择，但他的誓言要求他必须斩草除根"

5. 探索未知的发现爽：
   - 地城探索、未知位面、远古遗迹
   - 保持 DND 冒险的探索感和发现感
   - "他在地下第七层发现了这座城市的真正起源，建造这座城市的人早已灭绝"
```

### 爽点节奏设计

```
章节内爽点分布：
- 开篇：资源紧张（紧张感）
- 中段：战术智谋（爽点）
- 高潮：职业突破或惨胜（释放）
- 结尾：钩子（下一个冒险的暗示）

长线爽点阶梯：
- 第 1-30 章：学徒级冒险，清剿、追踪、小规模冲突
- 第 31-80 章：初阶到中阶，卷入派系冲突，接触更高位面
- 第 81-150 章：中阶到高阶，参与位面战争，职业认同觉醒
- 第 151 章后：传奇级，多重宇宙级冲突，最终决战

注意：阶梯不绝对，可根据篇幅调整
```

## 雷区检查清单（务必避免）

> 这一节是 DND 小说最核心的写作约束。**网文 DND 最大的失败模式就是变成桌游战报。**

### 雷区 1：游戏味太浓

**表现**：直接写"力量检定""敏捷检定""知识检定加成""专长""兼职惩罚"等桌游术语。

**正确做法：**
- ❌ "他这段时间泡在图书馆，给他带来了多少多少检定加成，所以认出了这只魔兽"
- ✅ "他这段时间泡在图书馆，刚好看到相关资料，所以认出了这只魔兽"

**核心原则：卡（角色卡）这种东西只应该在作者手上。** 官方小说《黑暗精灵三部曲》里，绝不会有这种傻逼描述。崔斯特的双巧手是通过他父亲的对话展现的，读者自然知道"哦，他有双巧手专长"——**让故事本身展现设定，而不是直接念规则**。

### 雷区 2：等级/环数直接写进小说

**表现**：角色说"我是 7 级法师""这是 3 环法术"。

**正确做法**：
- 用叙事方式表达实力层次。"学徒→初级→中级→高级→传奇"比等级数字更自然。
- 法术的威力差异通过**实际效果**来体现。
- "他的法术让整间大厅都燃烧起来"比"他施展了 5 环火球术"更有感染力。

### 雷区 3：把 DND 规则当"真理"而非"工具"

**错误认知**："我要严格按照三宝书写小说"。

**正确认知**：三宝书是**游戏规则**，是将玩法数字化、回合化的妥协产物。"拿三宝书做参考，大抵相当于用《内燃机原理教程》当'文学创作课'的教辅"。好的 DND 小说看的是**人物和故事**，让内行的看门道、偶有只鳞片甲被认出来会心一笑；让外行的看热闹、没有任何基础也能看得津津有味。

### 雷区 4：开篇塞大量设定

**表现**：第一章就开始解释 DND 的世界观、位面结构、神系关系。

**正确做法**：
- **先立大致的设定框架**，但不要一次性塞太多。
- 参考《放开那个女巫》的开篇：第一章介绍主角穿越成王子、正在准备处死一个女巫（**正在发生的事**），然后通过原主记忆带出世界背景——**用行动带设定，而非用说明文带设定**。

### 雷区 5：高等级对战难写就强行回避

**表现**：主角永远在打小怪，避免高等级对战；或者让高等级永远不在场。

**正确做法**：
- DND 高等级对战的难题确实存在（"高等级能玩的滥强手段太多"）
- 但回避不是解决方案，而是问题 —— 读者会觉得世界观不完整
- 解决方案：
  1. 限制主角等级上限（传奇前即结束）
  2. 用位面穿越替代纯对战（位面本身的规则限制）
  3. 用 NPC 协战分担压力
  4. 用智谋和准备替代正面硬刚

**高等级战斗的具体技法（在 4 条解决方案基础上扩展）：**

```
5. 资源博弈法：
   - 高等级战斗降级为"谁先用完高环法术位"的资源战
   - 关键描写：法师不再数伤害，而是数剩余法术
   - 参照：《冰风谷》范式

6. 位面规则限制法：
   - 高等级战斗发生在灵界 / 星界，物理规则被改写
   - 无重力、无距离、无声音、思维可见
   - 关键描写：法术威力被位面规则"打折"或"放大"

7. 神祇介入法：
   - 高等级战斗实际是两个神祇的代理人战争
   - 主角只是棋子（甚至不知情）
   - 关键描写：神祇代行者出现，主角退场

8. 道德困境替代法：
   - 高等级战斗的真正冲突不在法术强度，而在"该不该打"
   - 例：圣武士面前的"敌人"其实是受控者，打他就等于杀无辜者
   - 关键描写：主角在最后一刻放下武器

9. 地形改造竞赛：
   - 高等级战斗前先互相改写战场
   - 一方用塑石术改变地形，另一方用气象咒反制
   - 关键描写：环境成为战力主角
```

### 雷区 6：阵营写成标签

**表现**：角色被标记为"LG 圣武士"，但行为与标签无关。

**正确做法**：
- 阵营必须在**具体决策场景**中展现
- "他知道这是守序善良的行为"——具体是什么行为？
- 反例：角色被称为圣武士，但遇到奴隶制度合法的城邦时没有任何道德挣扎
- 正确：圣武士在奴隶制度城邦的痛苦抉择（要不要遵守当地法律？）

## 结局设计

DND 题材的结局是整本书哲学的最高体现。爽点设计解决"过程中的爽感"，结局设计解决"终点如何收束"。

### 三种结局模型

**模型 1：传奇前即结束（《黑暗精灵》范式）**

```
- 故事在主角接近传奇时收束（不进入真正的传奇级）
- 高等级对战的难题被规避（"没到那个级别就不用写"）
- 主角的最终胜利是身份认同的完成，不是等级碾压
- 宝物可以传承给下一代，但主角本人不会"飞升"
- 读者理解：这是一个完整的故事，主角选择了自己的道路

适用：中长篇、单元剧+长线、想规避"高等级难写"的作者
```

**模型 2：多重宇宙开放结局（《博德之门》EP 范式）**

```
- 主角完成物质位面的主线，但选择离开物质位面继续冒险
- 留给读者想象空间（"他在灵界又经历了什么？"）
- 不写"飞升"或"成神"，但主角已经成为世界级传奇
- 续作 / 外传 / 同人空间大
- 读者理解：这不是结局，这是新冒险的起点

适用：长篇、IP 延伸、多重宇宙世界观、自有世界观的扩展
```

**模型 3：阵营哲学升华结局（《第一律法》范式）**

```
- 主角的最终胜利不是击败敌人，而是真正理解并贯彻自己的阵营信仰
- 例：守序善良圣武士在奴隶制城邦的选择（解放奴隶 vs 遵守法律）
- 他的选择让他失去了一切（地位、神术、队友），但他坚守了信仰
- 阵营觉醒代替等级碾压
- 读者理解：这不是"赢"，是"选择"

适用：硬核读者、文学性强、想深度挖掘阵营哲学
```

### 底线原则

```
无论选择哪种结局模型，必须遵守：

- DND 不是修仙，主角不必"飞升"
  - HE 不是"成为传奇"，而是"主角找到了自己的道路"
  - 主角不必"成神"或"飞升"，那不是 DND 的内核

- 阵营弧光优先于等级碾压
  - 主角的最终选择应该体现他的阵营信仰
  - 不是"我赢了"，而是"我做了符合我身份的选择"

- 宝物传承 > 数值继承
  - 神器不应在结局被"继承给儿子"，那等于否认了主角的弧光
  - 神器的最佳结局是被销毁、封印、或在有意义的地方传承
```

### 结局选择决策树

```
你的主角最在乎什么？
├── "我要成为传奇冒险者" → 模型 1（传奇前即结束）
├── "我要继续探索世界" → 模型 2（多重宇宙开放结局）
└── "我要坚守我的信仰" → 模型 3（阵营哲学升华结局）

你的读者群预期什么？
├── 爽感 + 完整故事 → 模型 1
├── 追读 + IP 延伸 → 模型 2
└── 哲学思辨 + 文学性 → 模型 3

篇幅规划？
├── 短篇（1-30 章）→ 模型 1 紧凑
├── 中篇（30-100 章）→ 模型 1 或 模型 2 灵活
└── 长篇（100+ 章）→ 模型 2 或 模型 3 需充分铺垫
```

## 设计检查清单

### 世界观
- [ ] 多重宇宙架构是否清晰？（至少有物质位面 + 1-2 个其他位面）
- [ ] 位面之间是否有可穿越性？（不是锁死在物质位面）
- [ ] 神祇体系是否明确？（至少有 3-5 个互相竞争的神祇）
- [ ] 是否避免了开篇塞大量设定？

### 职业体系
- [ ] 主角的职业身份是否明确？（不是"散修""野路子"）
- [ ] 职业是否影响 NPC 对主角的态度？
- [ ] 职业成长是通过**实战突破**展现而非"升级提示"？
- [ ] 是否避免了"升级了，获得新技能"的写法？

### 九宫格阵营
- [ ] 主角的阵营是否明确？
- [ ] 阵营是否在**具体决策场景**中展现？
- [ ] 是否避免了"阵营 = 标签"的写法？
- [ ] 阵营冲突是否产生了道德困境？
- [ ] 同阵营的角色是否行为一致？（不是 LG vs LE 同人）

### 法术机制
- [ ] 是否避免了直接写"环数"？
- [ ] 法术的威力差异是否通过**实际效果**展现？
- [ ] 资源管理（法术位、气值）是否有体现？
- [ ] 法术位耗尽 = 战斗力骤降 是否在小说中可见？

### 爽点
- [ ] 是否有职业成长的节点感？
- [ ] 是否有战术博弈的智谋展现？
- [ ] 是否有宝物获得的获得感？
- [ ] 是否有阵营冲突的哲学深度？
- [ ] 是否有探索未知的发现感？

### 雷区
- [ ] 是否避免了"力量检定""敏捷检定"等桌游术语？
- [ ] 是否避免了"我是 7 级法师""这是 3 环法术"的写法？
- [ ] 是否避免了"升级提示"？
- [ ] 是否避免了开篇塞大量设定？
- [ ] 高等级对战是否有解决方案（而非回避）？

### 人物
- [ ] 主角是否有职业身份认同？（不只是"我要变强"）
- [ ] 主角的阵营是否在决策中展现？
- [ ] 女性角色是否独立？
- [ ] 配角是否有自己的阵营立场？
- [ ] 反派是否有自己的逻辑？（不是纯粹的邪恶）

### 市场
- [ ] 是否保留了 DND 风味？（不是变成了纯西幻）
- [ ] 是否对非 DND 玩家也友好？（不需要 DND 知识也能读）
- [ ] 开篇是否有明确的核心卖点？
