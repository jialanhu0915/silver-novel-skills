# DND 奇幻 / 硬核 DND 题材模板

> **题材定位：** 服务于主动点名"DND"的读者和作者 —— 默认读者具备桌游相关认知，对多重宇宙、职业体系、九宫格阵营、法术位机制有期待。模板**保留 DND 风味**（职业身份认同、阵营哲学、资源管理、小队羁绊），但配套**雷区检查清单**防止小说变成桌游战报。

---

## 类型特征

DND 奇幻小说是以**多重宇宙架构**为背景、以**职业身份认同**为核心、以**九宫格阵营哲学**为道德框架、以**小队冒险**为叙事结构的作品。

**核心要素：**
- **多重宇宙**：位面、星界、物质位面、灵界等可自由组合
- **职业体系**：战士、法师、圣武士、盗贼、牧师、术士、德鲁伊、吟游诗人、武僧等
- **九宫格阵营**：守序善良 → 混乱邪恶 的行为逻辑纲领
- **法术机制**：环位 0-9、每日法术位准备、资源管理博弈
- **小队冒险**：地城探索、模组任务、阵营冲突、合作与背叛

**核心卖点：** 职业成长的节点感 / 战术博弈的智谋 / 宝物与装备的获得感 / 阵营冲突的哲学碰撞 / 探索未知的发现感

**与玄幻/仙侠的根本区别：**

| 维度 | 玄幻/仙侠 | DND 奇幻 |
|------|----------|----------|
| 世界观 | 单一大陆 + 修真界分层 | **多重宇宙 + 多位面** |
| 力量体系 | 修为等级 + 灵气/法力 | **职业身份 + 法术位 + 阵营** |
| 核心矛盾 | 宗门争斗 / 正邪大战 | **小队冒险 + 阵营冲突 + 道德困境** |
| 主角定位 | 孤独求道者 / 天选之人 | **小队成员 / 冒险者** |
| 爽点侧重 | 突破碾压 / 装逼打脸 | **职业成长节点 + 战术智谋 + 道德抉择** |

## 力量体系

### 职业体系（身份认同核心）

DND 力量体系不是"修为等级"，而是**职业身份 + 能力组合 + 行为模式**的三位一体。**职业不只是技能清单，更是身份认同与世界观**——圣武士的守序善良不只是能放什么技能，更是他如何做决策、如何面对道德困境。

**核心职业简表：**

| 职业 | 定位 | 身份认同 | 关键约束 |
|------|------|---------|---------|
| 战士 | 近战输出 / 肉盾 | 纪律、武技、战术纪律 | 依赖装备与训练 |
| 法师 | 高伤害法术 / 远程 | 知识、研究、学术传统 | 法术位耗尽 = 战斗力骤降 |
| 圣武士 | 近战 + 神术 / 坦克 | 誓言、神祇、信仰体系 | 违背誓言 → 失去神术 |
| 盗贼 | 潜行 / 爆发伤害 | 街头智慧、自由、独立 | 依赖环境与先手 |
| 牧师 | 治疗 / 辅助 | 神祇信仰、治愈、引导 | 信仰动摇 = 能力下降 |
| 术士 | 法术输出 / 先天血脉 | 血脉传承、命运、天赋 | 法术位少但威力强、不可学新法术 |
| 德鲁伊 | 自然法术 / 变形 | 自然平衡、万物有灵 | 金属铠甲禁用 |
| 吟游诗人 | 辅助 / 控场 / 情报 | 故事、艺术、社交 | 依赖表演与社交场合 |
| 武僧 | 近战 / 闪避 | 修行、戒律、武道 | 依赖气值（Ki）资源 |

**扩展职业：** 游侠（特定敌人加成 / 追踪）、邪术师（与异位面存在契约）、魔战士（法武合一）、召唤师（位面召唤）

**职业 vs 修为的根本区别：**
- 修为：单线递进（炼气→筑基→金丹...）
- 职业：身份锚定（你永远首先是圣武士/法师/盗贼，然后才有等级）
- DND 主角的成长不是"修为越来越高"，而是"对这个职业的理解越来越深"

### 九宫格阵营（道德框架）

DND 的阵营是**行为逻辑纲领**，不是道德标签。

**九宫格简表：**

| 阵营 | 核心 | 行为 | 道德困境 |
|------|------|------|---------|
| 守序善良（LG） | 荣誉 + 善良 + 规则 | 荣誉即一切 | 法律与良善冲突（奴隶制合法的城邦） |
| 守序中立（LN） | 秩序 > 一切 | 规则就是规则 | 邪恶但合法的秩序？ |
| 守序邪恶（LE） | 秩序 + 邪恶 | 以灵魂为燃料发动血战、通过律法与契约治理领地 | 魔鬼也有"信用"——比混乱邪恶更高级 |
| 中立善良（NG） | 善良 > 法律/混乱 | 该帮就帮 | 与守序善良的圣武士可能冲突 |
| 绝对中立（TN） | 自然平衡 | 不让天平倾斜 | 不介入邪恶？ |
| 中立邪恶（NE） | 自利 + 邪恶 | 不择手段 | 比混乱邪恶更阴险 |
| 混乱善良（CG） | 善良 + 自由 | 按良心，不按法律 | 守序善良觉得 CG 是惹麻烦 |
| 混乱中立（CN） | 自由 > 一切 | 我想怎样就怎样 | 不稳定的盟友 |
| 混乱邪恶（CE） | 毁灭 + 自由 | 烧杀抢掠，无序破坏 | 力量最直接的邪恶，但最无脑 |

**阵营设计原则：**
- **阵营是行为模式，不是标签**：作者必须能用 1-2 句话说出该角色"在这种情境下会怎么做"
- **阵营不是一成不变**：圣武士可能堕落为守序邪恶（违反誓言后黑化）
- **同阵营 ≠ 同行为**：两个守序善良的圣武士可能因信仰不同神而做出相反决定
- **不要把阵营写成标签**：如果删掉阵营描述，行为完全不变 → 阵营就是装饰

**阵营冲突经典场景：** LG vs LE（道德碰撞）/ LG vs CG（法律 vs 良心）/ LN vs CN（秩序 vs 自由）/ 队友之间阵营张力（圣武士 LG + 盗贼 CN）

### 职业 / 阵营自创方法论

9 个核心职业 + 9 个经典阵营只是**基础素材**，完全自创一套职业体系 / 原创阵营变体是 DND 网文的核心竞争力。以下是 4 步方法论：

**步骤 1：确定职业主题 / 阵营主轴**

职业体系必须有一个核心主题，所有进阶命名都围绕这个主题展开。

```
主题示例：
- 「决斗者」：剑客 → 决斗者 → 仲裁者 → 圣剑使 → ...（围绕"一对一对决"展开）
- 「锻造师」：学徒 → 工匠 → 大师 → 宗师 → 传奇匠神 → ...（围绕"锻造神器"展开）
- 「守誓者」：立誓者 → 守誓者 → 仲裁者 → 大誓师 → 永恒之誓 → ...（围绕"不可破的誓言"展开）

阵营主轴示例：
- 守序善良的亚型：荣誉骑士（LG-H）vs 救赎圣武士（LG-S）
- 混乱善良的亚型：自由游侠（CG-F）vs 革命圣徒（CG-R）
```

**步骤 2：9 个职业名称的递进**

从学徒到传奇前，名称必须呈现"由学徒到大师"的递进：学徒期（刚入门）→ 初阶（基础任务）→ 中阶（合格代表）→ 高阶（传授门徒）→ 传奇前（凡人极限）。

**避免：** 9 个名称全是形容词堆砌（语义重复）/ 9 个名称主题跳跃（内在逻辑断裂）/ 9 个名称都是"形容词 + 名词"（缺乏身份认同）。

**步骤 3：绑定职业身份代价**

**关键约束：每个职业的"代价"必须与职业主题绑定，不能泛化为"每升一级都掉血"。**

```
绑定示例（决斗者主题）：
- 学徒剑客：代价 = 无（普通人）
- 决斗者：代价 = 不能拒绝任何一对一决斗邀请
- 仲裁者：代价 = 必须见证每场重要决斗的公正
- 圣剑使：代价 = 自己不能主动发起决斗
- 传奇前：代价 = 已成为决斗的代名词，被仇家永远追杀

反例：学徒 = 偶尔手抖 / 决斗者 = 经常手抖 / 仲裁者 = 总是手抖（同一代价的重复，没有递进）
```

**步骤 4：设计职业 / 阵营克制关系**

DND 的克制不是单向的"高级压制低级"，而是**身份之间存在特殊对抗**。

```
克制关系示例：
- 圣武士 vs 吸血鬼：圣武士的"驱邪"对吸血鬼天然克制（职业身份克制）
- 混乱善良 vs 守序善良：道义拷问（不是战斗克制，是哲学克制）
- 法师 vs 术士：学术传统 vs 血脉天赋（身份认同的相互轻视）
- 盗贼 vs 圣武士：信任 vs 怀疑（小队羁绊的天然摩擦）

避免：简单"高级 = 强"（失去剧情张力）/ 所有职业之间互相克制（无法设计情节冲突）/ 完全没有克制关系（沦为数值比拼）
```

### 法术与资源管理

**核心原则：** 不要直接写"环数""法术位"等桌游术语。用传统分级替代 + 通过实际效果展现。

**等级转译（小说化）：**

| 桌游级 | 小说化表达 |
|--------|----------|
| 1-5 级 | 学徒（刚开始冒险的冒险者） |
| 6-10 级 | 初阶（已立住脚的冒险者） |
| 11-15 级 | 中阶（区域知名） |
| 16-20 级 | 高阶（传奇前夕） |
| 21+ 级 | 传奇（已超脱凡人） |

**法术环数转译（小说化）：**

| 环数 | 小说化表达 |
|------|----------|
| 0 环（戏法） | "他能做点小把戏：点亮灯火、清洁衣物、修补破损" |
| 1-2 环 | "他施展了一个低阶法术" |
| 3-5 环 | "他施放了强力法术" |
| 6-7 环 | "他释放了大法师才能掌握的法术" |
| 8-9 环 | "这是传奇法术，凡人难以企及" |

**法术位机制转译：**
- ❌ "他还有几个法术位" → 删掉！
- ✅ "他意识到今天已经施了太多次法术，再强行施法可能会反噬"
- ✅ "他的法力储备接近枯竭" / "他感觉到法力本源开始震颤"
- ✅ "他必须在休息后才能再次施展如此强大的法术"

**资源管理（核心爽点）：** DND 的核心魅力之一是**资源管理**（法术位、卷轴、药剂、气值）。主角用**有限资源**解决看似无解的困境——这就是 DND 版的"智谋爽点"。

### 宝物与装备体系

**宝物分级（小说化）：**

| 桌游级 | 小说化表达 |
|--------|----------|
| 普通 | "一把精钢长剑" |
| +1 武器 | "一把带有微弱魔法附魔的精良武器" |
| 魔法物品 | "蕴含魔法力量的物品"（不需要"+1"标注） |
| 稀有 | "传说级武器" |
| 传奇 | "神器" |
| 独有 | "这个世界仅此一件的神器" |

**宝物类型：** 武器（+1 长剑 / 传奇巨剑）/ 防具（板甲 / 盾牌 / 戒指防护）/ 饰品（斗篷 / 护符 / 戒指）/ 卷轴（一次性法术卷轴，资源管理神器）/ 药剂（治疗药水 / 抗性药水 / 变形药水）/ 法器（法杖 / 魔杖 / 权杖）

**小说化处理原则：**
- 宝物不仅是数值加成，更是**故事驱动力**
- "这把剑是某位传奇骑士的佩剑，传说他曾在某战役中用它斩下巨龙之首"
- 宝物的获得往往伴随代价（任务、牺牲、诅咒）
- 宝物与角色绑定（圣武士 vs 诅咒剑 = 道德困境）

**神器示例（保留 DND 风味，不直接复制）：**
- 破誓者（曾经的圣武士之剑，被诅咒后会逼主人违背誓言）
- 元素君王之冠（召唤元素的传奇饰品，但持有者会被元素意识侵蚀）
- 影界之靴（穿越影界的装备，但每次使用都会留下部分自我在影界）

**情节引擎设计原则：**

DND 宝物的真正价值不在数值，而在**驱动情节**。

```
宝物的三种情节功能：
- 线索（揭示世界观真相）：一把剑的剑柄上刻有古代神祇铭文
- 目标（核心追逐物 / 麦高芬）：神器 X 是整本书的追逐核心
- 代价（塑造角色弧光）：获得神器的代价是失去某个职业能力

宝物与角色阵营的冲突设计：
- 圣武士 + 诅咒剑：誓约要求不能持有邪恶之物，但此剑是唯一能救队友的武器
- 混乱善良盗贼 + 秩序法杖：法杖让他获得力量，但每次使用都强迫他遵守秩序
- 德鲁伊 + 金属神器：德鲁伊誓言禁止金属装备，但神器是拯救森林的唯一希望

宝物获取过程的职业考验：
- 法师必须放弃学术传统才能获得某法杖（与"知识传承"的法师身份根本冲突）
- 圣武士必须在"救人"与"持剑"之间选择（直接挑战他的核心誓约）
```

## 世界构成

### 多重宇宙架构

DND 的世界观是**多重宇宙**（Multiverse）。可以完全自创位面，不必绑定费伦。

| 位面类型 | 特点 | 典型代表 |
|---------|------|---------|
| 物质位面 | 物理规则正常，主角主要活动位面 | 费伦、艾伯伦、龙枪、自创大陆 |
| 内层位面 | 元素 / 能量 / 恶魔 / 天使 | 火 / 风 / 水 / 土元素位面、恶魔深渊、天使山巅 |
| 外层位面 | 各大神祇领域 + 阵营极端 | 九层地狱、混沌海、七层天堂山、各神祇领域 |
| 过渡位面 | 与物质位面重叠 / 思维位面 | 星界、灵界、影界、梦境境 |

**自创位面建议：** 龙枪本身就是一个独立世界；完全自创的多重宇宙反而更灵活；关键是位面之间要有**可穿越性**。

### 阵营势力格局

| 势力类型 | 特征 | 代表 |
|---------|------|------|
| 守序善良型城邦 | 法律严明但公正、神祇信仰虔诚、圣武士骑士团驻扎 | 银月城、博德之门类 |
| 混乱善良型城邦 | 自治社区、自由港、民主决策、盗贼公会与冒险者共存 | 冒险者天堂类 |
| 守序邪恶型帝国 | 法律至上但服务于邪恶、魔鬼契约 / 灵魂交易、暴政但有效 | 奈瑟斯、九狱类 |
| 混乱邪恶型领地 | 部落 / 魔物巢穴、无政府状态下的暴力 | 恶魔深渊、兽人部落 |
| 绝对中立型 | 德鲁伊圣地、自然神殿、不参与外界纷争 | 永聚岛 |

### 冒险生态

冒险者公会（任务发布 / 等级认证 / 主角核心社交场所）+ 地城（古老遗迹 / 分层结构 / 陷阱 + 怪物 + 谜题 + 战利品）+ 模组式任务（委托 / 时限 / 奖励 / 风险）+ 政治与阴谋（王国 vs 公会 / 神祇 vs 神祇 / 派系 vs 派系）。

## 经典套路

### 传统冒险流（《黑暗精灵》范式）

经典开局（新手冒险者接到第一份委托 → 组队法师/战士/圣武士/盗贼/牧师 → 完成委托升级 → 逐步卷入更大阴谋）；关键转折（委托背后的真相 / 队友的牺牲或背叛 / 主角的职业身份觉醒 / 最终面对真正的反派）。**适用：** 单元剧结构、小队羁绊、经典奇幻冒险。

### DND 规则解构流（《博德之门：我，九狱银行家》范式）

主角是 DND 知识者（玩家穿越/觉醒），深度内化规则为世界观底层逻辑。"骰子检定"升华为命运神骰的具象化意志；"职业/种族/背景"转化为真实的生存逻辑；"模组式冒险"解构为结构性叙事单元。**适用：** 硬核 DND 玩家向、深度设定党。

### 伪 DND / 轻 DND 流（《深渊主宰》《红袍法师》范式）

完全自创世界观，借用 DND 元素（种族、职业、阵营、法术）但不绑定任何官方位面，简化或放弃严格的等级/规则体系。**为什么是主流选择：** 真 DND 很难写成小说，低等级滥强手段多，高等级对战难写；自己大幅魔改的伪 DND 要好写得多；读者大多数是泛奇幻读者。**适用：** 绝大多数网文作者。

### DND + 热门题材融合流

DND + 都市（现代背景下的 DND 元素）/ DND + 种田/领主（魔导工业化、领地建设）/ DND + 穿越/系统（主角带 DND 知识穿越异界）/ DND + 异兽文（主角是龙、恶魔等非人种族）/ DND + 恋爱（冒险小队中的感情线）。

## 势力冲突

### 阵营冲突（核心冲突类型）

```
守序善良 vs 守序邪恶：经典善恶对决，圣武士 vs 魔鬼领主，道德立场明确但复杂
守序 vs 混乱：法律 vs 自由，哲学深度的对立
善良 vs 邪恶：同阵营内部未必一致（守序善良觉得混乱善良是麻烦制造者）
中立派选择：绝对中立的德鲁伊可能不介入任何阵营冲突，但平衡被打破时必须选边站
```

### 派系冲突（冒险者 vs 政治）

冒险者公会 vs 王国（自由 vs 法律）/ 神祇冲突（不同神祇的教会互相竞争）/ 种族冲突（人类王国 vs 精灵森林 / 矮人山脉 vs 兽人草原 / 提夫林被所有种族歧视）。

### 跨位面冲突

位面入侵（恶魔深渊的入侵 / 异位面存在的降临）+ 召唤与契约（邪术师与异位面存在的契约 / 召唤仪式失控）+ 位面旅行（不同位面有不同的规则 / 是冒险的升级版）。

## 女性角色设计

### 职业类型简表

| 职业 | 身份张力 | 弧光核心 | 反例 |
|------|---------|---------|------|
| 女圣武士 | 守序善良的严格 vs 灰色世界 | 是否在残酷世界中保持信仰？ | 写成无趣的道德标杆 → 加人情味 |
| 女法师 | 男权法师塔中的地位 | 从学术天才到真正的法师 | 写成"女强文"工具 → 加内心戏 |
| 女盗贼 | 守序城邦中被通缉 | 信任问题（被背叛过？） | 写成花瓶 → 给独立动机 |
| 女牧师 | 信仰是否被现实考验？ | 信仰动摇后的重建 | 写成传声筒 → 加私心 |
| 女德鲁伊 | 文明 vs 自然的永恒冲突 | 是否能为文明做出妥协？ | 写成"自然之母" → 加个人化动机 |

### 跨阵营女性角色

| 阵营 | 魅力点 | 弧光 |
|------|-------|------|
| 女魔鬼（守序邪恶） | 律法型反派，魔鬼也有"信用"——比混乱邪恶更高级 | 在守序邪恶的逻辑中是否有人性残留？ |
| 女恶魔（混乱邪恶） | 原始的毁灭欲 | 混乱邪恶很少有弧光——更合适作为短期反派 |
| 女圣武士堕落者（LG → LE） | 经典堕落流 | 从信仰到怀疑到堕落的完整过程（必须有可共情的堕落理由） |
| 女德鲁伊（绝对中立） | 万物有灵的哲学 | 在平衡被打破时必须做出选择 |

### 弧光设计原则

```
职业弧光：起步（对职业身份初步认同）→ 转折（对职业的真正理解）→ 升华（成为职业的代表）
阵营弧光：起步（稳定的阵营认同）→ 转折（阵营信仰的动摇）→ 升华（阵营的重塑或坚守）
跨位面弧光：起步（只在物质位面活动）→ 转折（穿越到其他位面）→ 升华（理解多重宇宙的本质）
```

### 锚定物设计（防止人物扁平化）

```
正向锚定：职业锚定（圣武士锚定誓言）/ 阵营锚定（守序善良角色锚定同伴信任）/ 神祇锚定（牧师锚定神祇回应）/ 城邦锚定（法师锚定学术塔的传承）

反向锚定（适合邪教女主、堕落角色）：执念型锚定（盗贼锚定"曾被抛弃的孤儿院"）/ 罪责型锚定（堕落圣武士锚定"被我害死的队友"）/ 复仇型锚定（刺客锚定"灭门之仇"）

关键约束：锚定物不能"被抽走即崩溃"——必须有"锚定物的锚定物"。她的执念是否有"被满足的可能"？永远满足不了 = 单薄。
```

### 独立目标（避免工具化）

女性同伴必须有独立目标，不能只服务于主角。她有自己的职业进阶目标（即使与主角不同）/ 她有自己的阵营立场（可能与主角冲突）/ 她有自己的秘密（不为推动主角剧情而存在）/ 她有自己的冒险（与主角同时或交替进行）。

```
检查清单：
- [ ] 她的目标是否在主角剧情之外能独立成立？
- [ ] 她的目标是否影响主角的决策？
- [ ] 如果删除她，主角剧情是否还成立？（仍成立 → 工具人；不成立 → 有独立价值）
```

## 爽点设计

DND 题材的爽点更接近**传统升级流**，但有自己独特的风味。

### 五类爽点

| 类型 | 核心 | 小说化处理 |
|------|------|----------|
| 职业成长节点爽 | 从弱小到传奇 | 通过实战突破展现，不要"升级了，获得新技能" |
| 战术博弈智谋爽 | 用有限资源解决无解困境 | 利用环境、情报、准备、队友配合 |
| 宝物获得爽 | 获得神器、传奇装备 | 宝物不仅是数值，更应该有故事和代价 |
| 阵营冲突哲学爽 | 守序 vs 混乱、善良 vs 邪恶的哲学碰撞 | 不要把阵营写成标签，而要写成内在张力 |
| 探索未知发现爽 | 地城探索、未知位面、远古遗迹 | 保持 DND 冒险的探索感和发现感 |

### 爽点节奏设计

```
章节内爽点分布：开篇（资源紧张）→ 中段（战术智谋）→ 高潮（职业突破或惨胜）→ 结尾（钩子）

长线爽点阶梯：
- 第 1-30 章：学徒级冒险，清剿、追踪、小规模冲突
- 第 31-80 章：初阶到中阶，卷入派系冲突，接触更高位面
- 第 81-150 章：中阶到高阶，参与位面战争，职业认同觉醒
- 第 151 章后：传奇级，多重宇宙级冲突，最终决战
```

## 雷区检查清单（务必避免）

> 这一节是 DND 小说最核心的写作约束。**网文 DND 最大的失败模式就是变成桌游战报。**

### 雷区 1：游戏味太浓

**表现：** 直接写"力量检定""敏捷检定""知识检定加成""专长""兼职惩罚"等桌游术语。

**正确做法：**
- ❌ "他这段时间泡在图书馆，给他带来了多少多少检定加成，所以认出了这只魔兽"
- ✅ "他这段时间泡在图书馆，刚好看到相关资料，所以认出了这只魔兽"

**核心原则：卡（角色卡）这种东西只应该在作者手上。** 官方小说《黑暗精灵三部曲》里，绝不会有这种傻逼描述。崔斯特的双巧手是通过他父亲的对话展现的，读者自然知道"哦，他有双巧手专长"——**让故事本身展现设定，而不是直接念规则**。

### 雷区 2：等级/环数直接写进小说

**表现：** 角色说"我是 7 级法师""这是 3 环法术"。

**正确做法：**
- 用叙事方式表达实力层次。"学徒→初级→中级→高级→传奇"比等级数字更自然。
- 法术的威力差异通过**实际效果**来体现。
- "他的法术让整间大厅都燃烧起来"比"他施展了 5 环火球术"更有感染力。

### 雷区 3：把 DND 规则当"真理"而非"工具"

**错误认知：** "我要严格按照三宝书写小说"。

**正确认知：** 三宝书是**游戏规则**，是将玩法数字化、回合化的妥协产物。"拿三宝书做参考，大抵相当于用《内燃机原理教程》当'文学创作课'的教辅"。好的 DND 小说看的是**人物和故事**，让内行的看门道、偶有只鳞片甲被认出来会心一笑；让外行的看热闹、没有任何基础也能看得津津有味。

### 雷区 4：开篇塞大量设定

**表现：** 第一章就开始解释 DND 的世界观、位面结构、神系关系。

**正确做法：**
- **先立大致的设定框架**，但不要一次性塞太多。
- 参考《放开那个女巫》的开篇：第一章介绍主角穿越成王子、正在准备处死一个女巫（**正在发生的事**），然后通过原主记忆带出世界背景——**用行动带设定，而非用说明文带设定**。

### 雷区 5：高等级对战难写就强行回避

**表现：** 主角永远在打小怪，避免高等级对战；或者让高等级永远不在场。

**正确做法：** DND 高等级对战的难题确实存在，但回避不是解决方案，而是问题 —— 读者会觉得世界观不完整。

**5 种核心解决方案：**
1. **限制等级上限** —— 传奇前即结束（见结局设计）
2. **位面规则限制** —— 高等级战斗发生在灵界/星界，物理规则被改写（无重力、无距离）
3. **资源博弈法** —— 高等级战斗降级为"谁先用完高环法术位"的资源战
4. **神祇介入法** —— 高等级战斗实际是两个神祇的代理人战争，主角只是棋子
5. **道德困境替代法** —— 高等级战斗的真正冲突不在法术强度，而在"该不该打"的伦理问题

### 雷区 6：阵营写成标签

**表现：** 角色被标记为"LG 圣武士"，但行为与标签无关。

**正确做法：**
- 阵营必须在**具体决策场景**中展现
- "他知道这是守序善良的行为"——具体是什么行为？
- 反例：角色被称为圣武士，但遇到奴隶制度合法的城邦时没有任何道德挣扎
- 正确：圣武士在奴隶制度城邦的痛苦抉择（要不要遵守当地法律？）

## 结局设计

DND 题材的结局是整本书哲学的最高体现。爽点设计解决"过程中的爽感"，结局设计解决"终点如何收束"。

### 三种结局模型

**模型 1：传奇前即结束（《黑暗精灵》范式）** —— 故事在主角接近传奇时收束（不进入真正的传奇级），高等级对战的难题被规避，主角的最终胜利是身份认同的完成。**适用：** 中长篇、单元剧+长线、想规避"高等级难写"的作者。

**模型 2：多重宇宙开放结局（《博德之门》EP 范式）** —— 主角完成物质位面的主线，但选择离开物质位面继续冒险，留给读者想象空间。**适用：** 长篇、IP 延伸、多重宇宙世界观。

**模型 3：阵营哲学升华结局（《第一律法》范式）** —— 主角的最终胜利不是击败敌人，而是真正理解并贯彻自己的阵营信仰。阵营觉醒代替等级碾压。**适用：** 硬核读者、文学性强、想深度挖掘阵营哲学。

### 底线原则

- DND 不是修仙，主角不必"飞升" —— HE 是"主角找到了自己的道路"，不是"成为传奇"
- 阵营弧光优先于等级碾压 —— 主角的最终选择应体现他的阵营信仰
- 宝物传承 > 数值继承 —— 神器不应在结局被"继承给儿子"，最佳结局是被销毁、封印或在有意义的地方传承

### 结局选择决策树

```
你的主角最在乎什么？
├── "我要成为传奇冒险者" → 模型 1
├── "我要继续探索世界" → 模型 2
└── "我要坚守我的信仰" → 模型 3

你的读者群预期什么？
├── 爽感 + 完整故事 → 模型 1
├── 追读 + IP 延伸 → 模型 2
└── 哲学思辨 + 文学性 → 模型 3

篇幅规划？
├── 短篇（1-30 章）→ 模型 1 紧凑
├── 中篇（30-100 章）→ 模型 1 或 模型 2 灵活
└── 长篇（100+ 章）→ 模型 2 或 模型 3 需充分铺垫
```
