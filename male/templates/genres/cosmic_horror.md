# 宇宙恐怖 / Cosmic Horror 题材模板

> **题材定位：** 借用宇宙恐怖的哲学（人类渺小、接触即疯狂、真相不可承受），但素材库可自由选择 —— 既可致敬克苏鲁神话本体系，也可借用哲学后完全自创。

---

## 类型特征

宇宙恐怖小说是以**不可名状之物**为核心、以**理智消耗**为代价、以**真相揭露**为驱动的作品。哲学根基来自洛夫克拉夫特，但网文化后允许融合中式修仙、都市怪谈、SCP、维多利亚时代等元素。

**核心要素：**
- **不可名状**：超越人类认知的存在，不可直视、不可言说、不可战胜
- **理智机制**：接触、调查、晋升都会消耗精神（SAN 值 / 污染度 / 沉沦值）
- **真相揭露**：世界表面合理，深层恐怖；每揭一层都是双刃剑
- **凡人渺小**：最强调查员在古神面前也只是稍大的蚂蚁
- **仪式与禁忌**：接触有规则，违反规则 = 立刻疯狂

**核心卖点：**
- 真相震撼：揭开的秘密越深，读者震撼越强
- 理智拉扯：SAN 值归零前的惊险博弈
- 序列晋升：传统升级流的爽感外壳 + 理智消耗的代价内核
- 惨胜快感：付出惨烈代价换取微小胜利，比碾压更有回味
- 同伴羁绊：疯狂世界中的人性微光

**与传统恐怖（horror.md）的根本区别：**

| 维度 | 传统恐怖 (horror.md) | 宇宙恐怖 (cosmic_horror.md) |
|------|---------------------|---------------------------|
| 反派 | 鬼/僵尸，**可封印可反杀** | 古神/不可名状，**不可战胜** |
| 主角成长 | 道术等级提升 | 序列/途径晋升，**代价是理智下降** |
| 爽点 | 捉鬼反杀、逃生成功 | 真相揭露、惨胜、救赎 |
| 哲学底色 | 正邪对立 | 灰色地带（调查员 vs 邪教都是凡人） |
| 大众化处理 | 中式民俗爽文 | 反类型，可正剧化 |

## 力量体系

### 理智机制（核心创新）

```
理智值（SAN 值 / 污染度 / 沉沦值）：

消耗来源：
- 直接接触不可名状之物（注视、聆听、接触）
- 获得序列/途径晋升（每升一级都更接近疯狂）
- 研读禁忌典籍（知识即毒药）
- 长时间处于污染区域
- 见证同伴疯狂或死亡

恢复方式（困难）：
- 远离污染源
- 锚定物（亲友、信念、记忆）
- 仪式净化（代价昂贵）
- 时间流逝（极慢）

理智阈值与状态：
- 90-100：清醒，普通人状态
- 70-89：轻微异常，偶尔闪回/噩梦
- 50-69：明显异常，开始出现幻觉、强迫行为
- 30-49：严重异常，认知扭曲、价值观崩塌
- 10-29：临界疯狂，丧失自我
- 0-9：不可逆疯狂/堕落

关键设计原则：
- **理智机制必须自洽**：每个消耗来源有明确触发条件
- **恢复方式必须困难**：否则理智机制沦为装饰
- **理智下降要可见**：让读者通过主角行为观察到恶化
- **理智归零 ≠ 死亡**：可能是更恐怖的结局（成为眷族、堕落为怪物）
```

### 序列 / 途径体系（参考《诡秘之主》）

```
序列设计原则：
- 每条序列有 9 个阶段（序列 9 ~ 序列 1），序列 0 = 旧日 / 古神位
- 每个序列对应一种"非人化代价"（序列 7 = 学徒，代价是记忆衰退）
- 高序列对低序列有"神秘学压制"（无需战斗，弱者直接疯狂）
- 晋升仪式：必须满足特定条件（地点、时间、祭品）

经典 22 序列（诡秘体系，可自创）示例：
- 暴君 / 猎人 / 红祭司 / 黑夜 / 死神 / 战神
- 学徒 / 偷盗者 / 秘祈人 / 观众 / 读心者 / 怪物
- 囚犯 / 疯子 / 律师 / 歌颂者 / 水手 / 观众
- 太阳 / 仲裁人 / 审判者
- 魔女 / 耕种者 / 药师

自定义示例（中式融合）：
- 道士 → 法师 → 天师 → 仙人 → 道祖（每个阶段都可能成仙或疯狂）
- 守密人 → 守望者 → 守夜人 → 守灵者 → 守墓人

关键约束：
- **晋升 ≠ 单纯变强**：每一次晋升都让主角离"人"更远
- **高序列必须孤独**：知道太多真相，无法向普通人解释
- **同序列不同人** = 不同道路的选择题（道德困境）

### 序列自创方法论

上面给的"自定义示例"只是示例，完全自创一套序列体系门槛极高。以下是 4 步方法论：

**步骤 1：确定序列主题**

序列体系必须有一个核心主题，所有 9 级命名都围绕这个主题展开。

```
主题示例：
- 「看守」：守夜人 → 守望者 → 守门人 → ...（围绕"看守黑暗"展开）
- 「水手」：水手 → 风暴预言者 → 深渊航行者 → ...（围绕"深海航行"展开）
- 「医师」：草药医 → 病理解剖师 → 灵魂缝合师 → ...（围绕"治愈 / 解剖"展开）
```

**步骤 2：设计 9 级命名的逻辑递进**

从序列 9 到序列 1，名称必须呈现"由人到非人"的递进：

```
递进逻辑：
- 序列 9：人类身份（学生 / 学徒 / 守夜人）
- 序列 7-6：觉醒期（执行者 / 战斗者 / 持剑者）
- 序列 5-4：扭曲期（异化开始，名称变得抽象）
- 序列 3-2：超越期（已非人，名称接近概念）
- 序列 1：神之座（与神话存在同列）

避免：
- 9 个名称全是形容词堆砌（"守夜者 → 守望者 → 守卫者"——语义重复）
- 9 个名称主题跳跃（"学生 → 战士 → 祭司"——内在逻辑断裂）
```

**步骤 3：绑定"非人化代价"**

**关键约束：每个序列的"非人化代价"必须与序列主题绑定，不能泛化为"每升一级都掉 SAN"。**

```
绑定示例（水手主题）：
- 序列 9 水手：代价 = 对海洋有轻微感知（普通人无感）
- 序列 7 风暴预言者：代价 = 听到海洋的"低语"，无法在陆上安眠
- 序列 5 深渊航行者：代价 = 身体开始异化（鳃裂 / 蹼 / 低温耐受）
- 序列 3 深海祭祀：代价 = 已无法完全在陆地存活
- 序列 1 深渊之拥：代价 = ???（与深海存在合一的临界）

反例（不要这样）：
- 序列 9：代价 = 偶尔噩梦
- 序列 7：代价 = 经常噩梦
- 序列 5：代价 = 总是噩梦
（这三条是同一代价的重复，没有"非人化"递进）
```

**步骤 4：设计压制 / 克制关系**

神秘学压制不是单向的"高序列压制低序列"那么简单。

```
压制关系示例：
- 序列 5 对序列 8：有"位阶压制"（高序列对低序列天然威压）
- 序列 7 对序列 6：可能有"克制"（某些序列对另一些序列有特殊优势）
  - 例：「学者」克制「战士」（知识即力量）
  - 例：「守秘者」被「真相追寻者」克制（信息战）

避免：
- 简单"高级 = 强"（失去剧情张力）
- 所有序列之间互相克制（无法设计情节冲突）
- 完全没有克制关系（沦为数值比拼）
```

### 仪式与禁忌

```
接触规则示例：
- 不可直视沉睡者（直接疯狂）
- 不可在月圆之夜提及某名字（引来眷族）
- 不可擅自抄录某段经文（文字本身带污染）
- 不可在镜子前停留超过 X 秒（看见不该看的）

仪式类型：
- 召唤仪式：召唤眷族/低序列存在（高风险高回报）
- 封印仪式：暂时封印某物（不可永久）
- 净化仪式：恢复理智（消耗稀有材料）
- 晋升仪式：序列进阶（需满足条件 + 理智消耗）

禁忌层级：
- 表层禁忌：违反 = 受伤/降智
- 中层禁忌：违反 = 永久理智损伤
- 深层禁忌：违反 = 直接疯狂或死亡
- 终极禁忌：违反 = 引来旧日注视（整本书级别事件）
```

## 世界构成

### 三层世界模型

```
表层世界（守序）：
- 普通人社会：不知道不可名状的存在
- 日常生活的合理性：物理规则、社会秩序
- 主角初始身份：普通人、边缘调查员、神秘学学生

隐藏层（守秘者）：
- 守秘组织：维持表层世界的"正常"
- 调查员/处理者：处理超自然事件但对外保密
- 邪教/崇拜者：主动接触不可名状（寻求力量或解脱）
- 灰色势力：游走于守秘与崇拜之间

不可名状层（神话）：
- 古神沉睡地：人类禁区
- 眷族活动区：低序列威胁
- 污染扩散区：理智持续下降
- 旧日注视区：进入即终结
```

### 势力格局

```
守秘阵营：
- 官方组织：超自然事件处理局（参考 SCP 基金会 / 诡秘值夜者）
  - 严格守序：彻底封锁所有超自然信息
  - 温和守秘：有限公开、有限封锁
- 民间组织：教会、退隐门派、古老家族
  - 各自掌握部分真相
  - 互相之间信息不共享

邪教阵营：
- 主动崇拜：追求晋升/解脱/复仇
- 失控眷族：失去理智成为怪物（已无自我）
- 幕后黑手：更高序列者操纵低序列者

中立灰色：
- 投机者：利用超自然力量谋利
- 旁观者：知道真相但不参与
- 守秘逃兵：因理智崩溃而退出

不可名状层：
- 沉睡古神：极少直接登场
- 高序列眷族：主要反派
- 低序列眷族：日常威胁
- 污染扩散体：无智慧的污染源
```

## 经典套路

### 调查员流（克苏鲁 TRPG 范式）

```
经典开局：
- 主角是普通人或边缘调查员
- 接到第一个案件（看似普通的失踪/死亡）
- 调查过程发现超自然痕迹
- 真相逐步揭开，主角理智开始消耗
- 最终面对不可名状，做出两难选择

关键转折：
- 案件背后的案件（阴谋套阴谋）
- 同伴的牺牲或背叛
- 理智临界点的惊险博弈
- 最终真相：比案件本身更恐怖的存在

适用：硬核宇宙恐怖、致敬 Lovecraft、单元剧+长线
```

### 序列途径流（《诡秘之主》范式）

```
经典开局：
- 主角意外获得序列 9 能力（穿越/重生/觉醒）
- 拜入守秘组织，开始系统晋升
- 通过一次次事件+仪式逐步晋升
- 每晋升一级，接触更深真相
- 序列越高，敌人越强，代价越大

关键转折：
- 同序列不同道路的选择（道德困境）
- 晋升仪式的特殊要求（情感/人性的抉择）
- 来自更高序列者的关注
- 旧日注视的逼近

适用：长篇、网文爽感强、世界观宏大
```

### 中式融合流（《道诡异仙》范式）

```
经典开局：
- 中式世界观（修真/民国/现代）作为表层
- 主角接触"中式克苏鲁"事件（民俗恐怖+宇宙真相）
- 修仙体系与理智机制并存
- 修真者在古神面前一样渺小

关键转折：
- 修仙的真相 = 修真者本就在不可名状注视下修炼
- 民俗仪式（跳大神、出马仙、茅山）背后的真实
- 中式理智污染（走火入魔 = 理智归零的本土化）
- 道与魔的边界模糊

适用：东方读者共鸣强、修真+恐怖融合、网文黄金赛道
```

### 都市怪谈流

```
经典开局：
- 现代都市背景，普通人主角
- 接触都市传说/网络怪谈
- 发现传说背后是真实污染源
- 主角被卷入守秘者 vs 邪教之争
- 普通人如何在大能博弈中生存

关键转折：
- 都市传说中的真相揭露
- 普通人的理智保护机制（无知是福）
- 主角被迫选择成为调查员还是遗忘者
- 都市日常的"正常"背后是什么

适用：现代背景、入门门槛低、读者代入强
```

## 势力冲突

### 调查员 vs 邪教

```
核心矛盾：
- 调查员：维持表层世界秩序，封锁真相
- 邪教：追求真相/力量/解脱
- 双方都是凡人，都可能理智崩溃
- 真正的敌人是不可名状本身

道德困境：
- 调查员封锁真相 = 剥夺普通人知情权？
- 邪教追求真相 = 推动人类进化还是加速毁灭？
- 主角站哪一边都是灰色选择

经典冲突类型：
- 案件争夺：同一事件双方都要介入
- 叛徒问题：组织内部有人被邪教渗透
- 灰色同行：手段相同但目的不同的调查员
- 共情反派：邪教成员也是有故事的人
```

### 守秘组织内部路线之争

```
典型路线分歧：
- 鹰派：彻底封锁，包括清洗知情者
- 鸽派：有限公开，争取盟友
- 革命派：推翻现有秩序，重建真相社会
- 隐世派：远离世俗，专心研究

经典冲突：
- 主角卷入路线之争
- 不同路线代表不同价值观
- 没有绝对正确，只有代价不同
```

### 不可名状 vs 凡人

```
不可名状的真正威胁：
- 不可战胜（不是打不死，是无法理解）
- 不可直视（接触 = 疯狂）
- 不可沟通（没有谈判可能）
- 不可逃避（真相一旦知道就无法遗忘）

凡人应对方式：
- 守秘（不知道就没事）
- 封印（暂时隔离）
- 利用（借低序列眷族之力）
- 牺牲（用部分人的理智换取整体安全）
- 接受（与不可名状共存，但放弃部分人性）
```

## 女性角色设计

### 调查员女性

```
类型：
- 冷静调查员：高理智、克制、专业
- 疯狂调查员：高能力但已濒临崩溃
- 退役调查员：因理智问题退出，转为导师
- 守秘者：掌握部分真相但不直接行动

设计原则：
- 她的理智状态是核心特征（不只能力高低）
- 她的"锚定物"是什么？（亲友、信念、记忆）
- 她与不可名状的关系？（见证者？幸存者？受害者？）
```

### 邪教女性

```
类型：
- 高阶祭司：理智归零但保持优雅
- 堕落者：曾经是调查员，背叛投敌
- 救赎者：曾入邪教，现回头
- 共情反派：有自己的悲剧和逻辑

设计原则：
- 邪教不等于邪恶，可能有自己的"道理"
- 她的堕落过程有铺垫（不是突然黑化）
- 她的疯狂状态可以是角色魅力（克系女主经典）
```

### 眷族 / 不可名状层

```
类型：
- 半眷族：人类身 + 部分异化
- 完美眷族：外形仍人类但已无自我
- 反向眷族：异化但保留部分人性（更恐怖）
- 共生体：与主角绑定（克系女主经典套路）

设计原则：
- 异化过程要可见、有节奏
- 她的"人性残留"是角色亮点
- 与主角的关系涉及理智风险（接触她 = 消耗）
```

### 弧光设计（与男性角色同等深度）

```
调查员弧光：
- 起步：理智稳定，信仰体系清晰
- 转折：接触第一个真相，信仰动摇
- 升华：选择保留部分人性 or 接受非人化
- 关键检查：弧光是否与她的"锚定物"绑定？

邪教弧光：
- 起步：正常人，因某个事件堕入邪教
- 转折：发现邪教的"道理"其实有自洽逻辑
- 升华：要么回归（痛苦）、要么彻底堕落（毁灭）
- 关键检查：她的堕落过程是否有可共情的理由？

眷族弧光：
- 起步：人类身，被接触后开始异化
- 转折：异化过程不可逆，进入"半人半神"状态
- 升华：人性残留 vs 神性吸引的二选一
- 关键检查：异化过程是否可见、有节奏？
```

### 锚定物设计（防止人物扁平化）

```
正向锚定（亲友 / 信念 / 记忆）：
- 锚定物不能"被抽走即崩溃"——必须有"锚定物的锚定物"
- 例：她锚定在"妹妹"，但当妹妹死亡后，她必须有第二层锚
- 反例：锚定物一破就崩溃 → 人物弧光单薄

反向锚定（执念 / 仇恨 / 罪责）：
- 适合邪教女主、堕落角色
- 锚定物被"解决"反而是触发崩溃的条件
- 例：她靠"为父母复仇"活着，复仇完成后必须找到新锚
- 检查：她的执念是否有"被满足的可能"？永远满足不了 = 单薄

关键约束：锚定物的设计要让她的"崩溃"或"坚持"都有戏剧张力
```

### 独立目标（避免工具化）

```
女性同伴必须有独立目标，不能只服务于主角：

正向示例：
- 她有自己的晋升序列（即使与主角不同）
- 她有自己的调查目标（不是只跟随主角的案件）
- 她有自己的秘密（不为推动主角剧情而存在）

检查清单：
- [ ] 她的目标是否在主角剧情之外能独立成立？
- [ ] 她的目标是否影响主角的决策？
- [ ] 如果删除她，主角剧情是否还成立？
  - 仍成立 → 她是工具人
  - 不成立 → 她有独立价值
```

## 爽点重构（关键创新）

> **传统升级打脸不适用。** 宇宙恐怖的爽点必须重构，否则要么变四不像，要么哲学崩塌。

### 六类爽点设计

```
1. 真相揭露的震撼爽：
   - 不是"打赢了"，而是"看见了真相"
   - 层层剥洋葱，每一层都震撼
   - 关键揭露后主角的认知崩塌（理智快速下降）
   - 读者震撼 ≠ 主角胜利

2. 序列晋升的代价爽：
   - 晋升 = 离"人"更远 = 离真相更近
   - 晋升仪式涉及道德抉择（牺牲某人？放弃某记忆？）
   - 晋升后主角开始"非人化"，但能力大增
   - 爽感来自"我变强了"+"我付出了不可逆的代价"

3. 惨胜的回味爽：
   - 击败眷族 ≠ 解决威胁，可能引来更大存在
   - 付出惨烈代价（同伴死亡、理智归零、身体异化）
   - 换取微小的胜利（救下某人、封印某物、揭露部分真相）
   - 读者回味：不是"爽"，是"震撼 + 唏嘘"

4. 借力打力的智慧爽：
   - 不直接对抗，用更弱的不可名状挡更强的
   - 利用不同眷族之间的矛盾
   - 利用守秘组织和邪教的冲突
   - 主角像下棋，读者看主角智商压制

5. 同伴羁绊的人性爽：
   - 在疯狂世界中的人性微光
   - 锚定物的力量（为保护某人而坚持）
   - 同伴牺牲的悲壮
   - 团队羁绊的温暖（与冰冷主题形成反差）

6. 反杀低级眷族的纯粹爽：
   - 让主角偶尔痛快打怪（低级眷族）
   - 维持网文爽感节奏
   - 但提醒读者：这些只是杂兵，真正的恐怖在后面
```

### 爽点节奏设计

```
章节内爽点分布：
- 开篇：理智拉扯（紧张）
- 中段：真相揭露（震撼）
- 高潮：惨胜/反杀（释放）
- 结尾：钩子（下一个不可名状的暗示）

长线爽点阶梯：
- 第 1-30 章：接触表层威胁（低级眷族、案件）
- 第 31-80 章：晋升序列，接触中层真相
- 第 81-150 章：卷入守秘组织 vs 邪教大战
- 第 151 章后：直视古神（结局导向）

注意：阶梯不绝对，可根据篇幅调整
```

## 结局设计

宇宙恐怖网文的结局是整本书哲学的最高体现。爽点重构解决"过程中的爽感"，结局设计解决"终点如何收束"。

### 三种结局模型

**模型 1：开放式（诡秘之主式）**

```
- 主角成为序列 0 或类似高位
- 旧日仍在，威胁永恒
- 主角"赢"了当前的危机，但宇宙恐怖的压迫感未减
- 读者理解：这场战斗结束了，下一场永远会来

适用：长篇、追读导向、IP 延伸
```

**模型 2：牺牲式（道诡异仙式）**

```
- 主角以自身为代价封印 / 延缓某个不可名状
- 不可名状未被消灭，只是被压制
- 主角可能"飞升" / "成神" / "消散"——但不是战胜
- 读者理解：主角的选择是"两害相权取其轻"

适用：中长篇、情感共鸣强的作品、悲剧向读者
```

**模型 3：循环式**

```
- 主角发现一切早已注定
- 自己可能是上一轮循环的遗物 / 复刻 / 失败的前任
- 古神不是被"击败"的，而是被"等待"的
- 读者理解：恐怖不是某个事件，而是结构性的永恒

适用：哲学向、文学性强、反类型作品
读者预期低，不适合纯爽文读者
```

### 底线原则

```
无论选择哪种结局模型，必须遵守：

- 古神不可被消灭
  - HE 不是"打败了古神"
  - HE 是"主角接受了这个真相，并选择了某种活下去的方式"
  - 终极威胁必须永恒存在（否则宇宙恐怖哲学崩塌）

- 不可名状的"在场"必须延续到结局
  - 即使主角成为序列 0，他/她仍然在旧日的注视下
  - 主角的"赢"不能削弱宇宙恐怖的压迫感

- 读者的"爽感"应来自主角的抉择，不是来自古神的死亡
```

### 结局选择决策树

```
你的主角最在乎什么？
├── "保护身边的人" → 模型 2（牺牲式）
├── "理解这个世界的真相" → 模型 1（开放式）
└── "打破命运的循环" → 模型 3（循环式）

你的读者群预期什么？
├── 爽感 + 追读 → 模型 1
├── 情感共鸣 + 共情 → 模型 2
└── 哲学思辨 + 反类型 → 模型 3

篇幅规划？
├── 短篇（1-30 章）→ 模型 2 紧凑
├── 中篇（30-100 章）→ 模型 1 灵活
└── 长篇（100+ 章）→ 模型 3 需充分铺垫
```

## 设计检查清单

### 理智机制
- [ ] 理智消耗来源是否有明确触发条件？
- [ ] 理智恢复方式是否足够困难？
- [ ] 主角的理智状态是否在章节中可见？
- [ ] 理智归零后是否设计了非死亡结局（堕落、异化）？
- [ ] 理智机制是否被滥用？（主角绝境掉血又恢复是雷区）

### 不可名状设计
- [ ] 不可名状之物是否真的有压迫感？（不是打不死的小怪）
- [ ] 是否避免了"古神被主角嘴炮说服"的雷区？
- [ ] 接触规则是否明确、自洽？
- [ ] 眷族分级是否清晰？
- [ ] 旧日 / 序列 0 是否被滥用？（轻易登场会失去压迫感）

### 力量体系
- [ ] 序列 / 途径设计是否自洽？（晋升代价合理）
- [ ] 是否避免了"高序列 = 单纯数值变大"？
- [ ] 不同序列之间的克制关系是否清晰？
- [ ] 主角的"非人化"是否有可见描写？

### 爽点
- [ ] 是否重构了爽点（非简单升级打脸）？
- [ ] 惨胜是否有真实代价？
- [ ] 真相揭露是否层层递进？
- [ ] 同伴羁绊是否温暖而不过度？
- [ ] 是否给了低级眷族让主角偶尔爽快打怪？

### 世界观
- [ ] 三层世界模型是否清晰？
- [ ] 守秘组织 vs 邪教 vs 不可名状的三角关系是否平衡？
- [ ] 灰色地带是否有充分展开？（调查员 vs 邪教无绝对善恶）
- [ ] 中式 / 维多利亚 / 都市等表层世界观是否一致？

### 人物
- [ ] 主角弧光是否清晰？（普通人 → 调查员 → 半神 → ??）
- [ ] 女性角色是否独立？（不只作为男主锚定物）
- [ ] 反派（邪教 / 眷族）是否有自己的逻辑？
- [ ] 同伴是否有独立故事线？
- [ ] 女性角色弧光是否与男性角色同等深度？（不只承担"陪伴 / 锚定"功能）
- [ ] 女性角色锚定物是否有层次？（单一锚定 = 一破就崩）
- [ ] 女性角色是否有独立目标？（删除她后剧情仍成立 → 工具人）

### 哲学
- [ ] 是否避免了"主角战胜古神"的雷区？
- [ ] 真相揭露是否保持震撼？
- [ ] 是否在网文爽感与宇宙恐怖哲学之间找到平衡？
- [ ] 结局是否保持宇宙恐怖内核？（HE 也好 BE 也好，不可名状仍在）

### 结局
- [ ] 结局模型是否明确选择（开放式 / 牺牲式 / 循环式）？
- [ ] 决策树是否被参考？（主角诉求 / 读者预期 / 篇幅）
- [ ] 古神不可消灭原则是否遵守？（不是被"打败"而是被"接受"或"延缓"）
- [ ] 不可名状的"在场"是否延续到结局最后一章？
- [ ] 读者爽感来源是主角抉择而非古神死亡？
