# 女性角色设计指南

## 核心原则

**女性角色不是奖品或工具，她们是和男性角色一样完整的个体。**

每个重要女性角色都应该有：
- 独立的动机和欲望
- 自己的成长线
- 完整的性格
- 属于自己的选择

---

## 问题批判

### ❌ 禁止的写法

| 写法 | 问题 |
|------|------|
| "红颜知己" | 把女性当战利品 |
| "可爱小姨子/小秘" | 把女性当辅助工具 |
| "高冷女总裁后真香" | 只有"真香"没有独立人格 |
| "默默付出不求回报" | 只有奉献没有自我 |
| "傻白甜被保护" | 毫无独立能力 |

### ✅ 正确的写法

每个女性角色都应该有：
1. **她自己的目标** - 不是"辅佐男主"，而是她想要什么
2. **她自己的困境** - 她自己要解决的问题
3. **她自己的选择** - 她做的决定会影响剧情
4. **她自己的成长** - 不是被男主"改造"，而是自我成长

## 女性角色类型重设计

### 旧版（问题版）

```
❌ 高冷总裁：白富美，最初看不起后真香
❌ 青梅竹马：从小一起长大，默默付出
❌ 可爱小姨子：调皮可爱，助攻
❌ 职场女强：独立自主，并肩作战
```

### 新版（设计指南）

#### 1. 野心型

| 要素 | 描述 |
|------|------|
| 核心 | 她有自己想实现的目标 |
| 困境 | 她的野心与现实/性别限制冲突 |
| 发展 | 她如何平衡野心与情感/道德 |
| 示例 | 一位想要上位的女修士，她的手段和原则 |

#### 2. 独立型

| 要素 | 描述 |
|------|------|
| 核心 | 她不依赖任何人 |
| 困境 | 她的独立让她难以信任他人 |
| 发展 | 她如何学会适度依靠他人 |
| 示例 | 一位独来独往的女杀手，有自己的信条 |

#### 3. 复杂型

| 要素 | 描述 |
|------|------|
| 核心 | 她不是单纯的好人或坏人 |
| 困境 | 她的选择充满道德模糊 |
| 发展 | 她如何在灰色地带生存 |
| 示例 | 一位为了复仇不惜一切的女主角 |

#### 4. 成长型

| 要素 | 描述 |
|------|------|
| 核心 | 她有明显的成长空间 |
| 困境 | 她的性格有缺陷需要克服 |
| 发展 | 她通过事件逐渐成熟 |
| 示例 | 一位被宠坏的世家小姐，经历变故后成长 |

## 避免的套路

### ❌ "女强却爱上男主后变弱"

这是最常见的崩坏：

```
❌ 开头：女总裁雷厉风行
❌ 爱上男主后：变得患得患失，失去决策能力
❌ 理由：为了爱情
```

**正确做法**：她的爱情不应该削弱她原有的特质。

### ❌ "默默付出型"

```
❌ 她爱他，所以不求回报
❌ 她爱他，所以放弃自己的机会
❌ 她爱他，所以隐忍一切
```

**正确做法**：她有自己的需求和底线。

### ❌ "助攻工具人"

```
❌ 小姨子天然助攻男主
❌ 闺蜜天然帮忙追女主
❌ 女仆天然崇拜男主
```

**正确做法**：每个角色都有自己的立场和动机。

## 关系设计

### 不是"获得"而是"选择"

| ❌ 旧写法 | ✅ 新写法 |
|----------|----------|
| 男主"获得"了多少红颜 | 女主主动选择了男主 |
| 女主数量 = 成就 | 每段感情都有意义 |
| 追到女主 = 胜利 | 关系发展需要经营 |

### 关系应该是对等的

```
✅ 她有能力保护自己
✅ 她有自己的话语权
✅ 她能影响剧情走向
✅ 她的选择有后果
```

---

## 台词设计检查

设计女性角色台词时，检查：

1. **这句话是否只有"因为爱他"这一个理由？**
   - ❌ "我这样做都是因为爱你"
   - ✅ "我这样做是因为我觉得这是对的，而且碰巧我也爱你"

2. **她的台词是否能体现出她有自己的想法？**
   - ❌ "你决定就好"
   - ✅ "我同意你的方向，但细节上我有这些建议..."

3. **她的反应是否合理？**
   - ❌ 被伤害后立刻原谅
   - ✅ 被伤害后有愤怒、挣扎、最终选择（可能是原谅也可能是离开）
