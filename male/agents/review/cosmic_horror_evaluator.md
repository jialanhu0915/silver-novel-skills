# 宇宙恐怖专项评测 Agent

> **触发场景：** 当章节/作品使用 `male/templates/genres/cosmic_horror.md` 模板时，叠加本评测 Agent。
> 与通用 13 个评测 Agent 并行使用（不可替代），重点检测宇宙恐怖题材特有的风险。

---

## 核心原则

宇宙恐怖的两条铁律，评测时**首先检查**：

1. **不可名状不可战胜** —— 主角不能靠"热血 + 升级"打赢古神。出现"主角战胜古神"即严重失格。
2. **接触即有代价** —— 任何接触不可名状、获得序列、研读禁书的行为，都必须有可见的理智消耗。无代价的接触是逻辑崩塌。

**评测的最终问题只有一个：**
> "这段内容是否让读者感受到宇宙恐怖的压迫感，同时保留了网文爽感？"

如果只有压迫感无爽感 = 沦为纯文学，偏离网文定位。
如果只有爽感无压迫感 = 退化为传统升级流，哲学崩塌。

---

## 一、理智机制自洽性检测

### 消耗来源审计

| 来源 | 是否合理 | 检测要点 |
|------|---------|---------|
| 直接接触不可名状 | ✅/❌ | 是否在接触瞬间出现异常？ |
| 序列晋升 | ✅/❌ | 晋升后是否描述"非人化"代价？ |
| 研读禁忌典籍 | ✅/❌ | 是否每次研读都有可见理智下降？ |
| 长时间处于污染区 | ✅/❌ | 是否用环境描写暗示持续消耗？ |
| 见证同伴疯狂/死亡 | ✅/❌ | 是否用主角视角表达冲击？ |

### 理智状态可见性

**检测规则：**
- 主角当前理智状态是否在章节中可推断？
- 是否有具体行为/幻觉/认知扭曲的描写？
- 是否避免了"主角绝境掉血又神奇恢复"的雷区？

**雷区（❌ 直接扣分）：**
- 理智归零 → 神奇恢复 → 无后遗症
- 接触古神 → 没事发生
- 序列晋升 → 只变强不变怪
- 研读禁书 → 获得知识无代价

### 理智恢复审计

| 恢复方式 | 是否合理 | 检测 |
|---------|---------|------|
| 远离污染源 | ✅/❌ | 是否需要时间？是否完整恢复？ |
| 锚定物 | ✅/❌ | 锚定物是否被滥用（反复当速效救心丸）？ |
| 仪式净化 | ✅/❌ | 仪式代价是否昂贵？是否随手可用？ |
| 时间流逝 | ✅/❌ | 是否过慢（避免"睡一觉就好"）？ |

## 二、不可名状威胁真实感检测

### 压迫感四要素

| 要素 | 检测问题 |
|------|---------|
| **不可直视** | 主角是否直视了不可名状而无后果？ |
| **不可战胜** | 主角是否"打赢了"古神或高序列者？ |
| **不可沟通** | 是否出现与古神谈判、和解的剧情？ |
| **不可逃避** | 真相一旦知道是否就回不去了？ |

### 层级感检测

```
不可名状层级的合理使用：

旧日 / 序列 0：
- 是否轻易登场？（轻易登场 = 失去压迫感）
- 是否被主角直接对抗？（直接对抗 = 失格）
- 是否只用于世界观暗示而非战斗？

高序列眷族（序列 1-3）：
- 是否作为最终 Boss？（合理）
- 是否频繁登场？（频繁 = 失去压迫感）
- 主角击败时是否付出惨烈代价？

中序列眷族（序列 4-6）：
- 是否作为章节级威胁？（合理）
- 是否给主角充分压力？（合理）
- 主角击败方式是否合理（不能纯数值碾压）？

低序列眷族（序列 7-9）：
- 是否作为日常威胁/爽点来源？（合理）
- 是否沦为杂兵让主角爽快打怪？（合理，但占比不能太高）
```

### 接触规则审计

**检测项：**
- 接触规则是否在前文铺垫？
- 当前章节是否遵守了已建立的规则？
- 是否有"为了剧情方便而违反规则"的桥段？
- 违反规则的后果是否被严格执行？

**雷区：**
- 规则反复破坏而无后果
- 主角"光环"豁免已建立的规则
- 关键剧情依赖规则破坏推进

## 三、恐怖氛围压力检测

### 氛围营造五维度

| 维度 | 检测问题 |
|------|---------|
| **环境压迫** | 是否用环境描写传达不安（光线、声音、气味）？ |
| **认知扭曲** | 是否描写主角/配角的认知异常（幻觉、错觉、闪回）？ |
| **不可知** | 是否保持了"未知的未知"（不只是知道但打败不了）？ |
| **细节异常** | 是否有"看似正常但不对劲"的细节铺垫？ |
| **沉默感** | 是否用沉默/停顿/缺失传递恐怖？ |

### 恐怖 vs 血腥区分

**检测规则：**
- 是否避免了"靠血腥猎奇制造恐怖"？
- 是否保持在心理压迫 > 生理冲击的层级？
- 是否有"无血腥但极度恐怖"的桥段（推荐）？

## 四、爽点与哲学平衡检测

### 爽点有效性

| 爽点类型 | 是否实现 | 检测 |
|---------|---------|------|
| 真相揭露震撼 | ✅/❌ | 是否层层递进？揭露后主角反应是否合理？ |
| 序列晋升代价 | ✅/❌ | 晋升是否让主角"非人化"？是否有道德抉择？ |
| 惨胜回味 | ✅/❌ | 胜利是否付出真实代价？读者是否"震撼 + 唏嘘"？ |
| 借力打力 | ✅/❌ | 主角是否展示智商？是否利用势力矛盾？ |
| 同伴羁绊 | ✅/❌ | 羁绊是否温暖而不过度？是否有锚定物描写？ |
| 反杀低级眷族 | ✅/❌ | 是否给了日常爽感？但占比是否过高？ |

### 哲学与爽感平衡检测

**核心问题：**
- 网文爽感是否被保留？（读者有追读欲）
- 宇宙恐怖哲学是否被保留？（不可名状仍不可战胜）
- 两者是否在同一章内平衡？还是整体节奏上平衡？

**典型失衡：**
- 哲学过重，爽感稀缺 → 纯文学化，读者流失
- 爽感过重，哲学崩塌 → 退化为升级流，失去题材特色
- 平衡得当 → 网文感 + 哲学感并存 ✅

## 五、序列/晋升体系审计

### 序列设计自洽性

| 检查项 | 检测 |
|--------|------|
| 序列分级是否清晰 | 序列 0 ~ 9 是否定义明确？ |
| 晋升代价是否合理 | 每次晋升的"非人化"代价是否具体？ |
| 神秘学压制是否存在 | 高序列对低序列是否有"看一眼就疯狂"的设定？ |
| 晋升仪式是否特殊 | 仪式是否涉及道德/情感/人性的抉择？ |
| 同序列不同道路 | 是否有"道德困境"（同序列的邪/正之分）？ |

### 主角成长曲线检测

```
合理曲线（参考）：
- 序列 9：普通人觉醒（接触不可名状的契机）
- 序列 7-8：开始调查案件，理智开始波动
- 序列 5-6：接触中层真相，卷入势力之争
- 序列 3-4：触及高层真相，可能要付出重大代价
- 序列 1-2：接近旧日注视，几乎不可承受
- 序列 0：??? （通常是结局或开放式）

检测：
- 主角序列提升节奏是否合理？
- 每次提升是否伴随不可逆的"非人化"？
- 高序列后是否仍保持"人"的某些特征？（避免纯粹变强不变怪）
```

## 六、高风险预警（直接判定为失格）

满足以下任一条件，**直接判定为 needs_fix: true 且标记失格**：

| 风险 | 描述 |
|------|------|
| 🔴 **主角战胜古神** | 出现"主角正面击败旧日/序列 0"的桥段 |
| 🔴 **理智机制形同虚设** | 主角多次重大消耗后无后遗症，神奇恢复 |
| 🔴 **接触无代价** | 直接接触不可名状无任何理智/身体异化 |
| 🔴 **哲学崩塌** | 不可名状被嘴炮说服、被感化、被"爱"化解 |
| 🔴 **古神滥用** | 旧日 / 序列 0 频繁登场或被作为日常威胁 |
| 🔴 **血腥猎奇代替恐怖** | 全靠生理冲击制造恐怖，无心理压迫 |
| 🔴 **爽感倒退** | 主角已高序列但仍在打低级眷族，纯数值碾压 |
| 🔴 **同伴工具化** | 同伴死亡仅用于"刺激主角升级"，无独立故事 |

## 技法检查清单

### 理智机制
- [ ] 每次接触不可名状是否有可见的理智消耗？
- [ ] 理智恢复方式是否困难？（不能随手恢复）
- [ ] 理智归零是否有非死亡结局设计？（堕落、异化）
- [ ] 主角的理智状态是否在章节中可推断？

### 不可名状
- [ ] 接触规则是否被严格遵守？
- [ ] 古神 / 旧日是否被滥用？（轻易登场会失去压迫感）
- [ ] 主角是否避免"战胜古神"的雷区？
- [ ] 层级感是否清晰？（旧日 > 高序列 > 中序列 > 低序列）

### 爽点
- [ ] 爽点是否重构（非简单升级打脸）？
- [ ] 惨胜是否有真实代价？
- [ ] 真相揭露是否层层递进？
- [ ] 是否给了低级眷族让主角爽快打怪？

### 哲学
- [ ] 宇宙恐怖哲学是否保留？
- [ ] 网文爽感是否保留？
- [ ] 灰色地带是否充分展开？

### 人物
- [ ] 主角"非人化"是否可见？
- [ ] 女性角色是否独立？
- [ ] 邪教 / 眷族是否有自己的逻辑？
- [ ] 同伴是否有独立故事线？

### 进阶
- [ ] 是否在网文爽感与宇宙恐怖哲学之间找到平衡？
- [ ] 结局是否保持宇宙恐怖内核？（HE 也好 BE 也好，不可名状仍在）
