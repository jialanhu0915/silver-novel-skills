# DND 奇幻专项评测 Agent

> **触发场景：** 当章节/作品使用 `male/templates/genres/dnd.md` 模板时，叠加本评测 Agent。
> 与通用 13 个评测 Agent 并行使用（不可替代），重点检测 DND 题材特有的风险。
> **本评测与模板视角不同：** 模板告诉作者"怎么写"，本评测告诉评审"怎么判定"。两边的雷区列表不可互替。

---

## 核心原则

DND 的两条铁律，评测时**首先检查**：

1. **桌游感 ≠ 小说感** —— 桌游规则可以被魔改、世界可以自创，但"写的是小说还是桌游战报"是唯一硬标准。
2. **DND 风味要保留** —— 职业身份认同、九宫格阵营、小队羁绊、资源管理、位面探索是 DND 风味的核心元素，缺失任意 2 个即题材错位。

**评测的最终问题有两个：**

> **主问题：** "这段内容是否在保持 DND 风味的同时，不沦为桌游战报？"
> **次问题：** "网文读者的爽感（职业成长 / 战术智谋 / 探索发现）是否被保留？"

如果只有 DND 风味无小说感 = 沦为桌游战报，失格。如果只有小说感无 DND 风味 = 退化为普通玄幻升级流，题材错位。两者兼顾 = 合格的 DND 奇幻小说。

---

## 一、桌游术语红线检测（一票否决项）

> 本维度是 DND 评测的最高优先级。一旦触碰红线，整章直接判为 needs_fix: true 且标记失格。

**术语黑名单：**

```
❌ 检定类：力量检定、敏捷检定、知识检定、豁免检定、难度等级（DC）
❌ 等级类：职业等级（"我是 7 级法师"）、兼职、专长
❌ 法术类：法术环数（"这是 3 环法术""我只能施展到 5 环"）、法术位（"他还剩 2 个法术位"）、每日次数
❌ 数值类：+1/+2/+3 武器防具数值标注
❌ 战斗类：先攻值、攻击加值、伤害骰
```

**判定方式：** 出现即标记为**高风险**。如果累计超过 3 处，**直接判为失格**，需全局替换。

| 情况 | 判定 |
|------|------|
| 术语黑名单出现 1-2 处 | 标记高风险，needs_fix: true，fix_priority: high |
| 术语黑名单出现 ≥ 3 处 | 直接判失格，needs_fix: true，fix_priority: high |
| 术语被注释化（如"作者注：此处对应 5 环火球术"） | 不算违规 |
| 术语在元叙事/系统提示中 | 不算违规 |

## 二、职业身份认同深度检测

**评分：** 优（职业身份是角色的第二人格）/ 中（职业主要通过战斗能力体现）/ 差（职业只是个标签）

**检测问题：**
- [ ] 主角的职业是否影响了他的世界观、说话方式、决策逻辑？
- [ ] 法师是否像学者一样思考问题（重视知识、依赖法术位、备战型）？
- [ ] 圣武士是否在道德困境中痛苦权衡（守序善良 vs 灰色情境）？
- [ ] NPC 是否因为主角的职业而区别对待他？（圣武士到处被信任，盗贼被警惕）

**雷区：** 职业只体现在战斗中 / 法师像战士一样冲动施法 / 圣武士在奴隶制合法城邦毫无道德挣扎 / 盗贼像骑士一样光明正大

---

## 三、九宫格阵营哲学检测

**评分：** 优（阵营在 ≥3 个具体决策场景中展现且有道德困境）/ 中（阵营被提及但未驱动决策）/ 差（阵营只是标签）

**检测问题：**
- [ ] 角色在 ≥ 3 个具体决策场景中的行为是否符合其阵营？
- [ ] 主角的阵营是否与配角阵营产生过摩擦/冲突？
- [ ] 是否至少出现过 1 次"阵营信仰 vs 现实利益"的道德抉择？
- [ ] 同阵营的不同角色是否因为个体差异做出不同选择？（不是 LG vs LE 同人）

**阵营冲突经典场景检测（应出现 ≥1-2 类）：**

```
- LG vs LE：道德碰撞（最经典的善恶对决）
- LG vs CG：法律 vs 良心（解放奴隶 vs 遵守当地法律）
- LN vs CN：秩序 vs 自由
- 同阵营内部：圣武士（LG）+ 牧师（LG）vs 神祇冲突
```

> **文化转译说明：** 上述示例偏西方 DND 传统。中国网文语境下，可将"奴隶制合法城邦"转换为"宗门视魔族为死敌但主角发现魔族也有善良个体"等。核心是"**阵营信仰 vs 现实利益**"的抉择机制，不限于特定文化背景。

**雷区：** 阵营只出现在角色介绍中 / 阵营冲突 = 单纯战斗 / 在奴隶制合法城邦无道德挣扎 = 标签驱动 / 所有同阵营角色行为完全一致

---

## 四、资源管理叙事化检测

**评分：** 优（资源管理是每场冲突的核心张力）/ 中（偶尔提及资源消耗但不影响核心剧情）/ 差（资源无限或从不影响剧情）

**检测问题：**
- [ ] 主角是否有**有限资源**的感知？（法术还剩多少、药剂还有几瓶、体力消耗如何）
- [ ] 资源耗尽是否带来**真实的紧张感**？
- [ ] 资源分配是否成为**战术决策的核心**？
- [ ] 是否有"他今天已经施展了三次法术，能感觉到法力源开始震颤"这类叙事化呈现？

**资源管理信号词：**

```
应出现："法力源震颤""法力本源开始动摇""他感觉到体内魔力在枯竭"
       "药剂已经见底""这是最后一瓶""气值（Ki）耗尽，他不得不退守"
应避免："他还剩 3 个法术位""他还有 2 次施法机会""今日法术位未用尽"
```

**雷区：** 法术位耗尽无后果 / 道具永不耗尽 / 资源管理从未影响战斗结果 / 直接写"法术位""气值"等术语

---

## 五、小队羁绊质量检测

**评分：** 优（每个队友都有独立弧光、独立目标、独立秘密）/ 中（队友有基本性格但本质是"任务工具"）/ 差（队友就是战斗挂件）

**检测问题：**
- [ ] 队友是否有**独立的动机、秘密和目标**？
- [ ] 队友之间的**阵营摩擦**是否产生剧情张力？
- [ ] 删除某个队友，主线剧情是否仍完整成立？（仍成立 → 工具人；不成立 → 有独立价值）
- [ ] 队友是否有**自己的弧光**？（不是只服务于主角成长）

**队友职业互补检测（应出现 ≥2-3 类）：**

```
战斗互补：战士（肉盾）+ 盗贼（输出）+ 法师（远程）+ 牧师（治疗）
阵营互补：圣武士（LG）+ 盗贼（CN）的道义摩擦
技能互补：法师（探查）+ 游侠（追踪）+ 德鲁伊（自然知识）
性格互补：吟游诗人（社交）+ 武僧（自律）+ 术士（冲动）
```

**雷区：** 队友无独立动机 / 阵营摩擦只发生在战斗中 / 队友死亡只用于"刺激主角升级" / 队友无独立弧光

---

## 六、DND 风味保留检测

> **核心问题：** 这是"换了名字的玄幻升级文"，还是真正的 DND 奇幻？

**前 30 章内，以下 5 个核心元素应至少出现 2 个：**

| 元素 | 检测 |
|------|------|
| **职业身份认同** | 主角是否因职业身份做出独特决策？（不是单纯的"修为等级"） |
| **九宫格阵营决策** | 主角是否因阵营信仰做出道德抉择？ |
| **小队协作** | 是否有 ≥ 3 人的冒险小队？队友是否有互补关系？ |
| **资源管理** | 主角是否需要管理法术位/气值/药剂等有限资源？ |
| **位面探索** | 故事是否涉及位面/地城/遗迹的探索？（不止是平原地形） |

**伪 DND 自洽性检测：**
- 作者是否明确选择了"伪 DND"路线？
- 魔改是否自洽？（种族、职业、阵营的内部逻辑一致）
- 魔改是否影响了 DND 风味？（完全抛弃职业身份 = 不是 DND 了）

**原教旨 DND 路线合理性：**
- 是否对非 DND 玩家友好？（不需要 DND 知识也能读）
- 是否避免了术语堆砌？（检定/环数/兼职等直接出现 = 失格）

**雷区：** 标注 DND 但只是"换皮玄幻" / 主角是个"升级者"无职业身份 / 队友全是工具人 / 法力无限 / 故事从头到尾在一个平原打怪

---

## 七、战斗非回合制检测

**评分：** 优（有分工、配合、变数与环境互动）/ 中（有环境描写但行动仍按顺序）/ 差（明显按"先攻顺序"轮换）

**检测问题：**
- [ ] 战斗是否给人"回合制"的感觉？（你打一下，我打一下，轮流来）
- [ ] 是否有**环境互动**？（地形改造、天气利用、视野遮蔽）
- [ ] 是否有**借力打力**？（同伴配合、敌方互斗）
- [ ] 是否有**战术配合**？（不是各打各的，而是有协同）
- [ ] 行动顺序是否看起来自然，而非"按先攻值排序"？

**回合制判定三示例：**

```
✅ 完全合规（不触发红线）：
战士冲入敌阵吸引注意 → 盗贼趁乱绕后突袭 → 法师在后方吟唱法术，
施法时被流矢干扰导致偏移 → 牧师借机包扎战士伤口。
（分工、配合、变数同步出现，符合战斗逻辑）

🔴 触发警告（高风险）：
盗贼先行动——偷袭命中 → 战士再行动——挥砍命中 →
法师最后行动——施放火球术 → 敌人回合：向战士反击。
（顺序机械、无环境互动、无同步配合，明显是先攻排序描红）

⚠️ 合规但有优化空间：
战士和盗贼配合牵制，法师趁机施法，但整体节奏略显程式化。
（不触发红线，但可在 notes 中标注建议）
```

**雷区：** 战斗是简单的"按顺序出招"流水账 / 队友各打各的 / 无环境因素参与 / 法师施法从不被干扰

## 八、宝物情节引擎检测

**评分：** 优（宝物的获得/代价/使用构成完整微弧光，与职业认同和阵营信仰绑定）/ 中（有简单获得过程但代价不足）/ 差（宝物直接掉落/购买/赠送，无过程无代价）

**宝物三种情节功能覆盖度检测：**

| 功能 | 检测 |
|------|------|
| **线索** | 宝物是否揭示世界观真相？（如剑柄铭文暗示远古神祇） |
| **目标** | 宝物是否是冒险的核心追逐物 / 麦高芬？（如神器 X 是整本书追逐目标） |
| **代价** | 宝物是否有代价？（诅咒 / 道德困境 / 位面干扰 / 失去某职业能力） |

**评分标准：** 三种功能全覆盖 → 优 / 仅 2 种 → 中 / 仅 1 种或 0 种 → 差

**检测问题：**
- [ ] 获得宝物是否有**过程**（任务、牺牲、探索）？
- [ ] 宝物是否有**代价**？（诅咒、道德困境、位面干扰）
- [ ] 宝物是否成为**情节引擎**而非单纯数值提升？
- [ ] 宝物是否与角色的职业认同或阵营信仰绑定？（圣武士 + 诅咒剑 / 法师 + 学术法杖）

**雷区：** 神器/稀有装备直接捡到 / 宝物只提供数值加成 / 宝物的代价与角色弧光完全脱钩 / 同一件宝物被多人轻松使用

---

## 九、高风险预警（直接判定为失格）

满足以下任一条件，**直接判定 needs_fix: true 且标记失格**：

| 风险 | 描述 |
|------|------|
| 🔴 **桌游术语泛滥** | 术语黑名单出现 ≥ 3 处（检定/环数/兼职/法术位等） |
| 🔴 **标签驱动决策** | 角色行为完全由阵营标签或数值驱动，缺乏内心冲突或情境推理 |
| 🔴 **题材错位** | 前 30 章内 DND 核心元素（职业认同/阵营/小队/资源/位面）少于 2 个，本质是玄幻换皮 |
| 🔴 **开局设定倾倒** | 第一章超过 30% 篇幅是纯世界观/规则说明文 |
| 🔴 **回合制战斗描红** | 战斗明显按"先攻顺序"轮换，无环境/战术/变数 |
| 🔴 **资源无限** | 主角法力/道具无限，从未因资源耗尽陷入困境 |
| 🔴 **宝物直接掉落** | 神器/稀有装备直接捡到，无过程、无代价、无情节引擎 |
| 🔴 **阵营崩塌** | 主角在奴隶制合法城邦毫无挣扎，或同阵营角色完全同人化 |

## 十、可接受缺陷（网文语境下的弹性空间）

> **重要声明：** 以下维度不计入核心评分，允许弹性。评测时不作为失分项。

| 维度 | 弹性边界 |
|------|---------|
| **位面正统性** | 不必按官方 DND 位面架构（星界/灵界/九狱）。完全自创位面允许。 |
| **升级节奏** | 快穿升级或慢热成长均可，只要节奏稳定（不要 50 章升 1 级、3 章升 10 级）。 |
| **种族/职业完整性** | 不必全种族全职业齐全。聚焦 2-3 种族、3-4 职业且重点展开即可。 |
| **法术命名方式** | 自创法术名（"伊格尼斯的烈焰"）或经典名（"火球术"）均可，不走极端游戏式命名即可。 |
| **道德困境深度** | 不要求每章深度哲学思考。道德困境作为点缀而非主线即可。 |
| **高等级战斗复杂度** | 可以选择：①传奇前结束 ②位面规则限制 ③智谋+环境替代。核心是不强行写空洞高等级对波。 |

**处理方式：** 这些维度不进入 `problems` 列表，但可在 `notes` 字段中作为风格标注。

## 评测执行

### 输入

- 待评测章节内容
- 题材模板（必须是 `male/templates/genres/dnd.md`）
- 当前章节在整体剧情中的位置
- 主角当前职业等级 + 当前阵营 + 当前小队组成

### 输出 JSON

```json
{
  "genre_alignment": "高/中/低",
  "tabletop_jargon": {
    "violations": 0,
    "examples": []
  },
  "class_identity": "优/中/差",
  "alignment_philosophy": "优/中/差",
  "resource_management": "优/中/差",
  "party_dynamics": "优/中/差",
  "battle_naturalness": "优/中/差",
  "treasure_engine": "优/中/差",
  "dnd_flavor_retained": true,
  "problems": [],
  "high_risk_warnings": [],
  "needs_fix": true/false,
  "fix_priority": "high/medium/low",
  "suggestions": []
}
```

### 触发逻辑

**needs_fix: true** 当：
- 出现任一"高风险预警"（直接判定）
- `genre_alignment: 低`
- `tabletop_jargon.violations ≥ 3`
- 任一维度评分 = 差
- `alignment_philosophy = 差` 且 阵营张力未出现（双 false 即阵营沦为标签）
- `dnd_flavor_retained = false`（风味显著流失）
- 任意 ≥ 3 个维度评分 = 中（避免"六十分万岁"的作品通过评测）

**needs_fix: false** 当：
- 无高风险预警
- `genre_alignment ≥ 中`
- 桌游术语违规 < 3 处
- 各维度评分 ≥ 中
- 评分 = 中 的维度 < 3 个

**fix_priority** 判定：
- `high`：失格级问题（高风险预警触发）
- `medium`：重要维度评分 = 差但未失格，或 ≥ 3 个维度 = 中
- `low`：仅风格优化空间

### genre_alignment 计算逻辑

`genre_alignment` 是"核心元素覆盖率 + 伪 DND 自洽性"的综合判定：

| 核心元素数（5 个中为 true 数） | 伪 DND 自洽性 | genre_alignment |
|---------------------------|--------------|----------------|
| ≥ 4 | 自洽 | **高** |
| 2-3 | 自洽 | **中** |
| ≥ 4 | 不自洽 | **中**（有执行问题） |
| 2-3 | 不自洽 | **中** |
| ≤ 1 | 任何 | **低** |

> 注：5 个核心元素指职业身份认同 / 九宫格阵营决策 / 小队协作 / 资源管理 / 位面探索。
