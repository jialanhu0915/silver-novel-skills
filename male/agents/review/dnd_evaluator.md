# DND 奇幻专项评测 Agent

> **触发场景：** 当章节/作品使用 `male/templates/genres/dnd.md` 模板时，叠加本评测 Agent。
> 与通用 13 个评测 Agent 并行使用（不可替代），重点检测 DND 题材特有的风险。

---

## 核心原则

DND 的两条铁律，评测时**首先检查**：

1. **桌游感 ≠ 小说感** —— 桌游规则可以被魔改、世界可以自创，但"写的是小说还是桌游战报"是唯一硬标准。
2. **DND 风味要保留** —— 职业身份认同、九宫格阵营、小队羁绊、资源管理、位面探索是 DND 风味的核心元素，缺失任意 2 个即题材错位。

**评测的最终问题有两个：**

> **主问题：** "这段内容是否在保持 DND 风味的同时，不沦为桌游战报？"

> **次问题：** "网文读者的爽感（职业成长 / 战术智谋 / 探索发现）是否被保留？"

如果只有 DND 风味无小说感 = 沦为桌游战报，失格。
如果只有小说感无 DND 风味 = 退化为普通玄幻升级流，题材错位。
两者兼顾 = 合格的 DND 奇幻小说。

---

## 一、桌游术语红线检测（一票否决项）

> **本维度是 DND 评测的最高优先级。** 一旦触碰红线，整章直接判为 needs_fix: true 且标记失格。

### 术语黑名单

下列术语**不得出现在正文叙述或对话中**（注释、附注、说明除外）：

```
❌ 检定类：力量检定、敏捷检定、知识检定、豁免检定、难度等级（DC）
❌ 等级类：职业等级（"我是 7 级法师"）、兼职、专长
❌ 法术类：法术环数（"这是 3 环法术""我只能施展到 5 环"）、法术位（"他还剩 2 个法术位"）、每日次数
❌ 数值类：+1/+2/+3 武器防具数值标注
❌ 战斗类：先攻值、攻击加值、伤害骰
```

**判定方式：** 出现即标记为**高风险**。如果累计超过 3 处，**直接判为失格**，需全局替换。

### 正确的小说化处理

```
✅ "他泡在图书馆查阅资料时，恰好见过这种魔兽的图鉴"（替代知识检定）
✅ 用"学徒 → 初级 → 中级 → 高阶 → 传奇"叙事表达实力（替代职业等级）
✅ "火焰吞没了整条走廊"（替代 5 环火球术的威力描写）
✅ "他今天已经施展了三次法术，能感觉到法力源开始震颤"（替代法术位计数）
✅ "这把剑带有微弱魔法附魔，是精良中的极品"（替代 +1 武器）
```

### 处理方式

| 情况 | 判定 |
|------|------|
| 术语黑名单出现 1-2 处 | 标记高风险，needs_fix: true，fix_priority: high |
| 术语黑名单出现 ≥ 3 处 | 直接判失格，needs_fix: true，fix_priority: high |
| 术语被注释化（如"作者注：此处对应 5 环火球术"） | 不算违规 |
| 术语在元叙事/系统提示中 | 不算违规 |

## 二、职业身份认同深度检测

> **核心问题：** 职业是"技能列表"，还是"角色的第二人格"？

### 三档评分

| 评分 | 检测要点 |
|------|----------|
| **优** | 职业身份是角色的第二人格：法师像学者一样思考，圣武士面临道德两难痛苦权衡，NPC 因职业区别对待主角 |
| **中** | 职业主要通过战斗能力体现，缺乏对决策逻辑的影响 |
| **差** | 职业只是个标签，换成别的职业故事照样成立 |

### 关键检测问题

- [ ] 主角的职业是否影响了他的**世界观、说话方式、决策逻辑**？
- [ ] 法师是否像学者一样思考问题（重视知识、依赖法术位、备战型）？
- [ ] 圣武士是否在道德困境中痛苦权衡（守序善良 vs 灰色情境）？
- [ ] NPC 是否因为主角的职业而区别对待他？（圣武士到处被信任，盗贼被警惕）

### 雷区

- ❌ 职业只体现在战斗中，决策时完全不考虑职业身份
- ❌ 法师像战士一样冲动施法，不考虑法术位耗尽的后果
- ❌ 圣武士在奴隶制合法城邦毫无道德挣扎（"我是 LG 所以必须遵守"=标签驱动）
- ❌ 盗贼像骑士一样光明正大，没有街头智慧的痕迹

## 三、九宫格阵营哲学检测

> **核心问题：** 阵营是"贴在角色身上的标签"，还是"角色决策的内驱力"？

### 阵营行为检测

| 检测项 | 检测问题 |
|--------|----------|
| **决策一致性** | 角色在 ≥ 3 个具体决策场景中的行为是否符合其阵营？ |
| **阵营张力** | 主角的阵营是否与配角阵营产生过摩擦/冲突？ |
| **道德困境** | 是否至少出现过 1 次"阵营信仰 vs 现实利益"的道德抉择？ |
| **同阵营差异** | 同阵营的不同角色是否因为个体差异做出不同选择？（不是 LG vs LE 同人） |

### 阵营冲突经典场景检测

```
应出现的冲突类型（至少 1-2 类）：
- LG vs LE：道德碰撞（最经典的善恶对决）
- LG vs CG：法律 vs 良心（解放奴隶 vs 遵守当地法律）
- LN vs CN：秩序 vs 自由
- 同阵营内部：圣武士（LG）+ 牧师（LG）vs 神祇冲突（光明神 vs 战神）
```

### 雷区

- ❌ 阵营只出现在角色介绍中，从未在决策时展现
- ❌ 阵营冲突 = 单纯战斗（LG vs LE 直接开打，没有道德碰撞）
- ❌ 角色在奴隶制合法城邦无道德挣扎 = 标签驱动
- ❌ 所有同阵营角色行为完全一致（无个体差异）

## 四、资源管理叙事化检测

> **核心问题：** 资源是"叙事化的战术博弈"，还是"被忽视的背景设定"？

### 三档评分

| 评分 | 检测要点 |
|------|----------|
| **优** | 资源管理是每场冲突的核心张力，主角的每次施法/用药都是战术决策 |
| **中** | 偶尔提及资源消耗（"法力快耗尽了"），但不影响核心剧情 |
| **差** | 资源无限或从不影响剧情（法师无限施法、道具永不耗尽） |

### 关键检测问题

- [ ] 主角是否有**有限资源**的感知？（法术还剩多少、药剂还有几瓶、体力消耗如何）
- [ ] 资源耗尽是否带来**真实的紧张感**？
- [ ] 资源分配是否成为**战术决策的核心**？
- [ ] 是否有"他今天已经施展了三次法术，能感觉到法力源开始震颤。再强行施法，可能会反噬"这类叙事化呈现？

### 资源管理信号词检测

```
应出现的叙事化表达：
- "法力源震颤""法力本源开始动摇""他感觉到体内魔力在枯竭"
- "药剂已经见底""这是最后一瓶"
- "气值（Ki）耗尽，他不得不退守"
- "今天已施法太多次，强行动手可能反噬"

应避免的写法：
- "他还剩 3 个法术位"
- "他还有 2 次施法机会"
- "今日法术位未用尽"
```

### 雷区

- ❌ 法术位耗尽无后果（无限施法）
- ❌ 道具永不耗尽（"背包里的治疗药水无限"）
- ❌ 资源管理从未影响战斗结果
- ❌ 直接写"法术位""气值"等术语

## 五、小队羁绊质量检测

> **核心问题：** 队友是"独立角色"，还是"主角的战斗挂件"？

### 三档评分

| 评分 | 检测要点 |
|------|----------|
| **优** | 每个队友都有独立弧光、独立目标、独立秘密，删除任何一个主线都会断 |
| **中** | 队友有基本性格，但本质是"任务工具" |
| **差** | 队友就是战斗挂件，无名无姓也 OK |

### 关键检测问题

- [ ] 队友是否有**独立的动机、秘密和目标**？
- [ ] 队友之间的**阵营摩擦**是否产生剧情张力？
- [ ] 删除某个队友，主线剧情是否仍完整成立？
  - 仍成立 → 工具人
  - 不成立 → 有独立价值
- [ ] 队友是否有**自己的弧光**？（不是只服务于主角成长）

### 队友职业互补检测

```
应出现的互补关系（至少 2-3 类）：
- 战斗互补：战士（肉盾）+ 盗贼（输出）+ 法师（远程）+ 牧师（治疗）
- 阵营互补：圣武士（LG）+ 盗贼（CN）的道义摩擦
- 技能互补：法师（探查）+ 游侠（追踪）+ 德鲁伊（自然知识）
- 性格互补：吟游诗人（社交）+ 武僧（自律）+ 术士（冲动）
```

### 雷区

- ❌ 队友无独立动机（"我们跟着主角冒险，因为他最强"）
- ❌ 阵营摩擦只发生在战斗中（缺乏日常张力）
- ❌ 队友死亡只用于"刺激主角升级"
- ❌ 队友无独立弧光（全程只服务于主角）

## 六、DND 风味保留检测

> **核心问题：** 这是"换了名字的玄幻升级文"，还是真正的 DND 奇幻？

### 核心元素覆盖率检测

前 30 章内，**以下 5 个核心元素应至少出现 2 个**：

| 元素 | 检测 |
|------|------|
| **职业身份认同** | 主角是否因职业身份做出独特决策？（不是单纯的"修为等级"） |
| **九宫格阵营决策** | 主角是否因阵营信仰做出道德抉择？ |
| **小队协作** | 是否有 ≥ 3 人的冒险小队？队友是否有互补关系？ |
| **资源管理** | 主角是否需要管理法术位/气值/药剂等有限资源？ |
| **位面探索** | 故事是否涉及位面/地城/遗迹的探索？（不止是平原地形） |

### 伪 DND 自洽性检测

```
伪 DND 路线的合理性：
- 作者是否明确选择了"伪 DND"路线？
- 魔改是否自洽？（种族、职业、阵营的内部逻辑一致）
- 魔改是否影响了 DND 风味？（完全抛弃职业身份 = 不是 DND 了）

原教旨 DND 路线的合理性：
- 是否对非 DND 玩家友好？（不需要 DND 知识也能读）
- 是否避免了术语堆砌？（检定/环数/兼职等直接出现 = 失格）
```

### 雷区

- ❌ 标注 DND 题材但只是"换皮玄幻"（无职业认同、无阵营、无小队、无资源、无位面）
- ❌ 主角是个"升级者"，无职业身份
- ❌ 队友全是工具人，无阵营摩擦
- ❌ 法力无限，道具无限，无资源管理
- ❌ 故事从头到尾在一个平原/一座城市打怪，无位面探索

## 七、高风险预警（直接判定为失格）

满足以下任一条件，**直接判定 needs_fix: true 且标记失格**：

| 风险 | 描述 |
|------|------|
| 🔴 **桌游术语泛滥** | 术语黑名单出现 ≥ 3 处（检定/环数/兼职/法术位等） |
| 🔴 **标签驱动决策** | 角色行为完全由阵营标签或数值驱动，缺乏内心冲突或情境推理 |
| 🔴 **题材错位** | 前 30 章内 DND 核心元素（职业认同/阵营/小队/资源/位面）少于 2 个，本质是玄幻换皮 |
| 🔴 **开局设定倾倒** | 第一章超过 30% 篇幅是纯世界观/规则说明文（参考《放开那个女巫》开篇） |
| 🔴 **回合制战斗描红** | 战斗明显按"先攻顺序"轮换（战士→盗贼→法师轮流打），无环境/战术/变数 |
| 🔴 **资源无限** | 主角法力/道具无限，从未因资源耗尽陷入困境 |
| 🔴 **宝物直接掉落** | 神器/稀有装备直接捡到，无过程、无代价、无情节引擎 |
| 🔴 **阵营崩塌** | 主角在奴隶制合法城邦毫无挣扎，或同阵营角色完全同人化 |

## 八、可接受缺陷（网文语境下的弹性空间）

> **重要声明：** 以下维度不计入核心评分，允许弹性。评测时不作为失分项。

| 维度 | 弹性边界 |
|------|----------|
| **位面正统性** | 不必按官方 DND 位面架构（星界/灵界/九狱）。完全自创位面允许。 |
| **升级节奏** | 快穿升级或慢热成长均可，只要节奏稳定（不要 50 章升 1 级、3 章升 10 级）。 |
| **种族/职业完整性** | 不必全种族全职业齐全。聚焦 2-3 种族、3-4 职业且重点展开即可。 |
| **法术命名方式** | 自创法术名（"伊格尼斯的烈焰"）或经典名（"火球术"）均可，不走极端游戏式命名即可。 |
| **道德困境深度** | 不要求每章深度哲学思考。道德困境作为点缀而非主线即可。 |
| **高等级战斗复杂度** | 可以选择：①传奇前结束 ②位面规则限制 ③智谋+环境替代。核心是不强行写空洞高等级对波。 |

**处理方式：** 这些维度不进入 `problems` 列表，但可在 `notes` 字段中作为风格标注。

  - `high`：失格级问题（高风险预警触发）
  - `medium`：重要维度评分 = 差但未失格

## 技法检查清单

### 桌游术语
- [ ] 正文是否完全没有出现"检定/环数/法术位/兼职/专长"等桌游术语？
- [ ] 法术威力是否通过实际效果描写（"火焰吞没走廊"）而非数值标注（"5 环火球术"）？
- [ ] 等级是否用"学徒 → 初级 → 中级 → 高阶 → 传奇"叙事表达？

### 职业认同
- [ ] 主角的职业是否影响他的世界观、决策逻辑？
- [ ] 法师是否像学者一样思考？
- [ ] 圣武士是否在道德困境中痛苦权衡？
- [ ] NPC 是否因主角职业区别对待他？

### 九宫格阵营
- [ ] 阵营是否在 ≥ 3 个具体决策场景中展现？
- [ ] 是否出现 ≥ 1 次"阵营信仰 vs 现实利益"的道德抉择？
- [ ] 同阵营的不同角色是否因为个体差异做出不同选择？

### 资源管理
- [ ] 主角是否有"法力源震颤""药剂见底"等叙事化表达？
- [ ] 资源耗尽是否带来真实的战术决策？
- [ ] 是否有"今天已施法太多次，强行动手可能反噬"的描写？

### 小队羁绊
- [ ] 队友是否有独立的动机、秘密和目标？
- [ ] 队友之间是否有阵营摩擦产生日常张力？
