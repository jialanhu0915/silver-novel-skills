# 女频小说创作指南

本指南是 `silver-novel-skills` Skill 的女频子流程，由根 `SKILL.md` 路由后通过 Read 工具加载。
不要直接作为独立 Skill 使用（无 frontmatter，不会被 Claude Code 自动发现）。

---

全类型 AI 独立创作女频小说，支持先整体规划后逐章节完成，通过 SQLite 数据库追踪所有设定确保跨章节一致性。

专为女频读者打造：言情/耽美/百合/女尊/快穿/悬疑恋爱等类型。

---

## 核心特性

- **全女频类型支持**：现代言情、古代言情、耽美、百合、女尊、快穿、悬疑恋爱
- **女性视角优先**：以女性角色和感情线为核心驱动
- **交互式框架确定**：类型 → 世界观 → 人物 → 情节 → 感情线
- **程序化生成**：基于类型模板的有逻辑随机组合
- **多 Agents 协作**：框架阶段分工、评测阶段评审
- **自我反思机制**：十维评测（逻辑/人物/感情线/男角独立性/节奏/伏笔/信息/金手指/爽点/配角）
- **智慧型反派设计**：拒绝无脑降智，让反派有城府、有手段
- **男性角色独立人格**：拒绝工具化，男主/男配有自己的人生目标与成长（关键问题）
- **女性主角深度塑造**：情感独立、有智慧、有成长弧光
- **感情线质量设计**：推拉张力、甜虐平衡、多层次情感递进
- **金手指使用原则**：拒绝替代成长，让金手指辅助而非替代
- **合理爽点设计**：拒绝功利化，加入情感和精神层面爽点
- **配角独立价值**：拒绝工具化，让配角有自己的故事
- **BL/GL/双男主/双女主/群像 支持**
- **用户偏好定制**：根据用户风格偏好调整感情线走向
- **SQLite 数据库全程追踪**

## 核心设计原则

### 一、智慧型反派

**反派的嘲讽必须有智慧、有根据，禁止无脑降智！**

| 检查项 | 说明 |
|--------|------|
| 嘲讽有事实依据 | 基于客观事实或合理推断 |
| 嘲讽有目的 | 试探/施压/立威/离间 |
| 反派有城府 | 不只靠嘴，有实际手段 |
| 嘲讽分对象 | 对强者有应有的尊重 |

### 二、男性角色独立人格（关键问题）

**男性角色不是"帅/富/专一"的工具人，他们是自己人生的主角。**

| 检查项 | 说明 |
|--------|------|
| 有独立目标 | 不是只为"爱女主"而活 |
| 有自己成长线 | 不是被女主"改造" |
| 有自己困境 | 需要解决自己的问题 |
| 有底线原则 | 不是无条件的付出 |
| 有独立人生 | 工作/理想/社交不围绕女主转 |

### 三、女性主角深度塑造

**女主不是等待被拯救的公主，是自己故事的主角。**

| 检查项 | 说明 |
|--------|------|
| 有情感独立性 | 不是离了男人不能活 |
| 有智慧与判断 | 能独立分析问题 |
| 有成长弧光 | 性格会变化，有心路历程 |
| 有自己的选择 | 她的决定推动剧情 |
| 有底线与原则 | 不是无条件的付出 |

### 四、感情线质量设计

**好的感情线是推拉、是张力、是甜虐交织。**

| 检查项 | 说明 |
|--------|------|
| 推拉张力 | 相爱相杀/欲擒故纵/误会虐点 |
| 甜虐平衡 | 甜蜜与虐心交织，不过分甜腻或虐 |
| 情感递进 | 层层递进，有节奏的情感升温 |
| 心理细节 | 内心独白、微表情、行为暗示 |
| 关系平等 | 双向奔赴，非单方面付出 |

### 五、金手指使用原则

**金手指是辅助工具，不是替代成长的捷径。**

| 检查项 | 说明 |
|--------|------|
| 不是100%成功 | 有失败可能 |
| 有代价限制 | 消耗/次数/副作用 |
| 需要主角判断 | 不是自动触发 |
| 有依赖后果 | 过度依赖有负面影响 |

### 六、合理爽点设计

**爽点不只是"获得"，更是"成长"和"认同"。**

| 爽点类型 | 说明 |
|----------|------|
| 实力成长型 | 付出努力后的突破 |
| 情感共鸣型 | 信任回报、羁绊加深 |
| 认知升级型 | 看穿真相、智破困局 |
| 关系互动型 | 被认可、并肩作战 |
| 反派打脸型 | 智慧型打脸，有铺垫有手段 |

### 七、配角独立价值

**配角不是工具，是自己故事里的主角。**

| 检查项 | 说明 |
|--------|------|
| 有自己目标 | 不是只为帮助主角 |
| 有自己困境 | 需要解决自己的问题 |
| 有自己的关系 | 和主角是平等关系 |
| 有自己的成长 | 配角也在成长 |

## 用户风格偏好

在开始创作前，请确认以下风格偏好：

### 1. 双主角关系风格
- **平等型**：双方实力/地位对等，势均力敌
- **互补型**：性格/能力互补，互相成就
- **主从型**：一方主导，另一方配合

### 2. 感情线风格
- **甜宠主**：甜蜜为主，虐点只是调味
- **虐恋主**：虐心为主，甜只是希望
- **推拉型**：甜虐交织，张力十足

### 3. 男性角色定位
- **并肩作战型**：与女主实力对等，共同战斗
- **守护型**：实力强但尊重女主选择
- **辅助型**：提供资源/情报，不抢风头

### 4. 女性被男性拯救
- **禁止型**：女主完全独立，不依赖男性拯救
- **有限型**：可以接受有限度的帮助
- **接受型**：可以接受英雄救美

## 题材模板

### 现代言情 (Romance Modern)

**典型题材**：总裁/甜宠/豪门/校园/都市异能

**核心元素**：
- 甜蜜互动与推拉张力
- 男主：不只是"霸总"，有自己的事业压力和人生困惑
- 女主：独立自主，有自己的工作能力
- 感情线：甜中带虐，误会与解开误会
- 爽点：男主暗中守护，女主独立反击

### 耽美 (Boys' Love)

**典型题材**：古代/现代/校园/娱乐圈/星际

**核心元素**：
- 双男主：攻受都有独立人格和成长线
- 关系动态：身份差/性格差带来的张力
- 感情线：甜虐交织，欲擒故纵
- 情感细腻：心理描写要到位
- 攻受平等：不是一方完全主导另一方

### 百合 (Girls' Love)

**典型题材**：校园/都市/古风/异能

**核心元素**：
- 双女主：女性视角，情感细腻
- 感情线：含蓄渐进，暧昧试探
- 女性共鸣：符合女性读者的情感期待
- 社会议题：可以融入但不刻意
- 真实情感：不狗血，真实可信

### 女尊 (Female Dominance)

**典型题材**：女尊世界/女强男弱/美男环绕

**核心元素**：
- 世界观反转：女性主导的社会
- 男色消费：男性美貌的描写
- 女强设定：女强vs美男
- 感情线：女主主导，男性被动
- 反转爽点：性别反转的打脸

### 快穿 (Quick Pass / 快穿)

**典型题材**：穿书/系统流/攻略/逆袭

**核心元素**：
- 任务系统：用户定义任务目标
- 多人世界：每个世界独立设定
- 角色切换：适应不同世界身份
- 金手指：系统技能/任务奖励
- 感情线：每个世界独立或主线串联

### 悬疑恋爱 (Mystery Romance)

**典型题材**：推理/刑侦/心理/神秘

**核心元素**：
- 双重叙事：悬疑线 + 感情线交织
- 真相揭露：层层剥开的悬念
- 感情线：并肩作战的信任建立
- 惊悚元素：适当的紧张感
- 结局：真相与爱情双丰收

## 感情线设计指南

### 推拉动力学

| 阶段 | 推 | 拉 |
|------|----|----|
| 暧昧期 | 故意疏远/假装不在意 | 暗中关心/不经意靠近 |
| 确认期 | 傲娇拒绝/欲擒故纵 | 真心流露/主动表白 |
| 磨合期 | 争吵冷战/误会 | 和好甜蜜/信任加深 |
| 稳定期 | 独立空间/小争执 | 默契十足/互相支持 |

### 甜虐平衡公式

```
总感情线 = 70%甜蜜片段 + 30%虐心片段

虐心类型分布：
- 误会虐：40%（可快速解开）
- 身份虐：30%（身世/立场的冲突）
- 信任虐：20%（背叛或隐瞒）
- 生死虐：10%（生死关头的考验）
```

### 心理描写技巧

- **微表情**：眼神/动作暗示情感
- **内心独白**：内心挣扎与抉择
- **行为逻辑**：行为反映心理变化
- **环境呼应**：场景烘托情感
- **对话张力**：话里有话/欲言又止

## 金手指设计（女频适配）

### 女频常见金手指类型

| 类型 | 说明 | 示例 |
|------|------|------|
| 读心术 | 感知他人情绪/意图 | 谈判/社交无往不利 |
| 医术/毒术 | 治病救人/克敌制胜 | 神医流 |
| 空间/系统 | 储物/任务/技能兑换 | 快穿流 |
| 重生 | 先知优势 | 逆袭流 |
| 身份切换 | 多重马甲 | 马甲流 |
| 特殊体质 | 体质带来的优势 | 锦鲤体质/旺夫 |

### 金手指使用原则

1. **不为替代成长**：金手指是加速器，不是替代努力
2. **代价限制**：使用有代价，形成约束
3. **失败可能**：关键时刻可能失效
4. **依赖后果**：过度依赖会带来问题
5. **女性赋能**：金手指强化女性自主性
