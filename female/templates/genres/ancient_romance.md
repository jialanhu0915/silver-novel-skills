# 古代言情 (Ancient Romance) 创作模板

## 目标读者
喜欢古风背景、宫廷权谋、宅斗智谋的女性读者

## 核心元素
- 古代背景（明清、唐宋等）
- 言情+智斗双线并行
- 权力博弈与情感纠葛
- 传统文化意境

---

## 一、核心卖点与特色

### 古言三大魅力

| 卖点 | 说明 | 写法要点 |
|------|------|----------|
| **意境塑造** | 以景寄情，让读者"入画" | 写景为写情服务，月色/落花/风雪 |
| **智斗美学** | 四两拨千斤的酣畅感 | 信/话/礼/茶皆为筹码 |
| **历史想象力** | 大势中的小人物破局 | 王朝更迭/君臣博弈 |

### 2025年趋势信号
- 《探花》"女扮男装科举入仕+职场感"验证老梗新写
- 重生宅斗路线（《雁回时》）市场号召力依然强劲
- 大女主清醒独立而非无脑开挂

---

## 二、细分类型与套路

### 类型一：宅斗文

**焦点：** 大家族内部后宅之争

**关系网络：**
- 妻妾、嫡庶、妯娌、婆媳
- 构成网状矛盾结构

**典型升级结构（50万字）：**

| 阶段 | 字数 | 核心内容 | 敌方势力 |
|------|------|----------|----------|
| 蛰伏隐忍 | 0-10万 | 开局困境设定，庶女/继女/落魄嫡女/冲喜新娘 | 嫡母打压/姐妹陷害/恶仆欺主 |
| 锋芒初露 | 10-30万 | 经济战/舆论战/婚姻战多维反击 | 扩大至家族层面 |
| 巅峰对决 | 30-50万 | 清算血仇/身份逆转/权力更迭 | 皇宫/皇权势力 |

**地图演进：** 恶奴→姨娘→庶妹→家族→男主家族（多为皇宫）

---

### 类型二：宫斗文

**焦点：** 后宫权力斗争

**升级线路：** 宫女→美人→嫔→妃→贵妃→皇后

**独特元素：**
- 后宫与前朝权力联动
- 娘家势力博弈
- 皇子斗争

---

### 类型三：朝堂权谋类

**特点：** 女主直接切入朝堂生态

**经典设定：**
- 女扮男装考取功名
- 切入摄政王与皇权之间的漩涡
- "职场感"植入古言

**代表作：**《探花》

---

### 类型四：江湖/悬疑类

**特点：** 古言+幻想元素融合

**设定方向：**
- 异能设定（如梦境操控）
- 先婚后爱+悬疑解谜
- 大女主成长线

**代表作：**《白篱梦》

---

## 三、身份设定模式

### 穿越/重生路线

| 路线 | 女主处境 | 核心爽点 |
|------|----------|----------|
| **穿越女强** | 被休弃/受尽屈辱/魂穿 | 改变屈辱状态，渣男回头打脸 |
| **重生女强** | 曾为原配/遭遇背叛/重生 | 改变历史悲剧，挽回失去一切 |

### 身份对位法则

```
穿成庶女 → 必然有个厉害的主母
穿成嫡女 → 必然有个厉害的姨娘
穿成有后母的嫡女 → 必然有厉害的主母+厉害的姨娘
不管嫡出庶出 → 必然有心高气傲的姐妹（靶子）
```

### 家族设定两大型

| 类型 | 女主处境 | 结局走向 |
|------|----------|----------|
| **守护型** | 家族没落但家人保护 | 守护家族，自立自强，家族崛起 |
| **破坏型** | 家族强大但被虐待 | 奋起反抗，推倒家族 |

## 四、情感线特有模式

### 经典情感模式

| 模式 | 简介 | 情感张力 |
|------|------|----------|
| **先婚后爱** | 开局直接嫁入/冲喜/替嫁 | 相处中渐生情愫 |
| **重生复仇+情感拉锯** | 重生不为恋爱，为复仇 | 全盘算计中意外萌芽 |
| **女扮男装+职场共事** | 男子身份共事，情感滋生 | 男主心理煎熬+真相揭露反转 |
| **强取豪夺+智慧对抗** | 男主位高权重，女主弱势 | 压迫中赢回尊重和独立感情 |

### 男主魅力公式

**性格矛盾型：**
- 表面温润，实则残忍
- 表面多情，实则纯情
- 表面邪魅，实则冷酷

**身残强大型：**
- 身有残缺但只手遮天

## 五、核心爽点体系

### 打脸节点

| 爽点 | 场景 | 打脸对象 |
|------|------|----------|
| 寿宴揭假账 | 公开场合 | 嫡母 |
| 诗会才艺碾压 | 宴会 | 庶妹/竞争对手 |
| 医术救人 | 救活关键人物 | 获得家族庇护 |
| 身份揭穿 | 假千金暴露/真千金确认 | 颠覆正统 |
| 滴血验亲 | 公开对峙 | 嫡系正统 |
| 追妻火葬场 | 渣男回头/悔悟 | 情感爽点 |

### 起承转合节奏

```
起：开局被欺辱（积累情绪）
承：暗中积累筹码（期待反转）
转：公开打脸（情绪爆炸）
合：确立地位（爽点落袋）
```

**节奏要点：** 从受辱到打脸，中间缓释5集左右是最佳节奏

## 六、金手指类型

| 金手指 | 说明 | 示例 |
|--------|------|------|
| **医术/毒术/蛊术** | 救人/杀人的实用技能 | 《君九龄》医术即生存哲学 |
| **空间/药园** | 穿越携带辅助空间 | 改善家境，化解家族矛盾 |
| **预知能力** | 做梦预知吉凶 | 《首辅娇娘》规避风险 |
| **身份揭露** | 假千金/真千金逐步揭露 | 形成反转爽点 |
| **异能** | 如梦境操控 | 《白篱梦》镜体 |
| **组合金手指** | 穿越重生+先婚后爱+男主残疾等 | 注意叠加数量≠可用性 |

## 七、写作注意事项

### 常见误区

| 误区 | 正确做法 |
|------|----------|
| 女配开篇强后变弱 | 让核心女配贯穿始终，保持强大 |
| 男主出场太慢 | 前3万字集中男女主戏码，最好直接出场 |
| 男主被迫纳妾 | 女频读者对感情忠诚度极其敏感 |
| 女主过度依附男性 | 独立技能线、事业线、成长路径 |
| 敌人越来越多无情感钩子 | 每个反派设立应有情感根源 |
| 时代感违和 | 避免现代词汇，保持古典用词和礼法体系 |

### 核心原则

**双线并行：** 事业线（升级）和感情线必须同步推进，缺一则失衡

**反派设计：** 女配必须足够强大，阴谋迭起才能维持长文

## 八、与现有框架的衔接

### 与 charactercraft.md 的衔接
- 女性角色四大要素：独立目标/成长线/自主选择/底线
- 女配也需有独立目标和成长线

### 与 plotarchitect.md 的衔接
- 关键节点设计：打脸型节点需要有递进铺垫
- 金手指设计：每个阶段只需一种金手指侧面

### 与 climax_design.md 的衔接
- 女频爽点类型：情感共鸣型、羁绊确认型
- 宅斗打脸型爽点需要节奏把控

## 技法检查清单

### 身份设定
- [ ] 身份设定符合"对位法则"
- [ ] 家族设定清晰（守护型/破坏型）
- [ ] 金手指有独特性，非堆砌

### 情感线
- [ ] 双线并行（事业+感情）
- [ ] 情感模式有独特张力
- [ ] 男主性格有矛盾反差
- [ ] 无男主花心/纳妾问题

### 爽点设计
- [ ] 打脸节点有起承转合
- [ ] 打脸间隔节奏合理（约5集）
- [ ] 事业线有财富/身份/技能提升
- [ ] 敌人强大且有情感钩子

### 文风
- [ ] 意境塑造（以景写情）
- [ ] 无现代词汇违和
- [ ] 礼法体系可信
