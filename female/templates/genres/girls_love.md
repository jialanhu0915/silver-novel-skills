# 百合（GL）类型模板

**目标读者：** 喜欢女女恋爱的女性读者
**核心元素：** 双女主、情感深度、女性视角

---

## 百合说明

- 女性之间的恋爱，主要为女性读者创作
- 两个角色都是女性，没有基于性别的权力不平衡
- 重点关注情感连接和女性体验
- 女性视角是核心——真实地探索女性之间的吸引

---

## 百合细分类型

### 类型一：青春校园

- **设定：** 高中、大学
- **基调：** 纯真、青春、情感觉醒
- **重点：** 初恋、成长、自我发现
- **典型：** 一方发现吸引，另一方不知情

### 类型二：现代都市

- **设定：** 当代城市生活
- **重点：** 事业、独立、寻找爱情
- **平衡：** 职业生活与爱情兼顾
- **典型：** 职场相遇或邻居

### 类型三：治愈温暖

- **基调：** 温暖、情感情愈
- **背景：** 一方或双方都有过往创伤
- **重点：** 共同度过困难、相互支持
- **典型：** 失忆/抑郁/家庭创伤

### 类型四：时尚模特

- **设定：** 娱乐圈
- **重点：** 光鲜、竞争、恋爱
- **典型：** 从对手到恋人的竞争对手

| 重点 | 常注重激情 | 常注重情感亲密 |

## 百合专项注意事项

### 1. 真实女性体验

- 探索真实的女性吸引
- 不是男性幻想版本
- 处理：社会压力、家庭期望、自我发现

### 2. 女性情谊

- 谨慎处理女性友谊
- 区分：亲密友谊vs浪漫兴趣
- 包含：不只是恋爱中的女性纽带

### 3. 情感深度

- 百合常注重情感亲密
- 身体亲密：暗示而非明示
- 重点：情感连接第一

### 4. 社会背景

- 家庭压力（尤其在传统设定中）
- 社会偏见（谨慎处理）
- 结局：挣来的，不是魔法般的接受

## 百合角色类型

### 女主类型A（主动型）

- 在恋爱中采取主动
- 有清晰的目标和野心
- 可能在外表上更"男性化"，也可能不是
- 示例：CEO、运动员、自信的艺术家

### 女主类型B（成长型）

- 在故事中发现吸引
- 最初更犹豫
- 通过关系成长
- 示例：晚熟者、温室花朵

### 关系动态

- 一方可能更有经验
- 不同性格创造张力
- 双方都应该有独立生活
- 平衡：在一起的时间vs独立成长

- [ ] 双方都有独立目标？
- [ ] 社会背景处理得当？
- [ ] 情感连接是首要的？
- [ ] 用户偏好风格保持一致？

## 当前趋势（2025）

### 向深度与类型融合迈进

2025年被称为"gl井喷之年"，但一部成熟的百合作品不仅需要甜，更需要构建有深度、可信的女性宇宙。

**核心变化**：
- 突破单纯的"轻飘飘的甜"，向更复杂的情感与自我身份认同探索
- 晋江文学城于2023年设立"无CP＋"独立分站，职场晋升、专业成长类作品点击量同比增长47%
- 越来越多GL作品开始强调"因她是她"而爱，拒绝用男性的喜欢来衬托女主魅力

## 如何写出真实的女性情感

### 核心原则

| 原则 | 说明 |
|------|------|
| 强调共鸣 | 女女之间的情感来自于沉浸式的心理历程，拒绝"假大空"的甜 |
| 先让读者"找到自己喜欢她" | 不要着急推进感情线，在相互理解的过程中建立可信的感情基础 |
| 情绪真实胜于情节的戏剧化 | 不一定非要"末日逃生"才能证明爱意，生活化的陪伴与细节共鸣更能打动GL读者 |

### 拒绝虐女逻辑

- 不应该把男性的凝视、羞辱或背弃作为推动女女情感发展的工具
- 如果故事中有虐，必须是女主之间纯粹的情感哲学，而不是男性视角的伤害

---

## 避免男性凝视的写法

### 避雷指南

| 问题 | 说明 |
|------|------|
| 聚焦内在动力 | 将叙事重点放在女主个人成长经历、事业推进与情感碰撞上 |
| 减轻男角色戏份 | 部分GL核心读者对男性角色较为敏感，创作时可适当控制男性角色的戏份 |
| 提升女性自主性 | 女性角色的魅力不靠外界的"认可"或"追捧"来证明，而依靠对自身的清醒认知与行动的意志力 |
| 拒绝"仍想讨好男读者" | 全身心地描绘女性的欲望与梦想 |

## 写作注意事项

| 问题 | 后果 | 解决方案 |
|------|------|----------|
| 感情线缺失 | 百合文感情线是灵魂，万万不能没有 | 剧情可以没有，感情线必须有 |
| 虐过多 | 百合读者偏爱吃"甜食"，过多虐会劝退读者 | 甜为主，虐适度 |
| 男性角色"抢戏" | gl里很忌讳出现男的，轻微"百破"都会导致差评 | 尽量减少男性角色戏份 |
| 用男性爱慕来衬托女主魅力 | 读者反感 | 不要描写男配调戏/喜欢女主 |
| "闺蜜式甜" | 两个女主感情停留在"好闺闺"层面 | 必须有独属于情侣的化学反应 |
