# 女频仙侠 (Female Xianxia) 创作模板

## 目标读者
喜欢修仙题材、仙侠情感、东方幻想的女性读者

## 核心元素
- 仙侠世界观（修真/修仙）
- 情感纠葛与人物成长并行
- 拯救与自我救赎主题
- 东方幻想美学

---

## 一、核心卖点与特色

### 女频仙侠魅力

| 卖点 | 说明 |
|------|------|
| **仙凡差距的情感张力** | 身份对立下的禁忌之恋 |
| **拯救与自我救赎** | 从"为世界牺牲"到"为我重生破局" |
| **双线升级** | 事业线（修仙）+ 感情线并进 |

### 叙事转型趋势

**传统叙事：** "为了世界我必须牺牲"——宿命裹挟，以牺牲换爱情

**现代转型：** "为了我自己我要重生破局"——从世界系向玩家心态转型

---

## 二、细分类型

### 类型一：师徒恋类

**核心设定：** 师尊×弟子的权力落差

**张力来源：**
- 权力不平等（师尊拥有绝对权威）
- 道德困境（师徒伦常成为禁忌）
- 外部压力（仙魔对立/世界存亡）

**经典设定：** 无情剑修+门规森严 > 你情我愿（更有戏剧张力）

---

### 类型二：仙魔恋/跨种族类

**核心设定：** 上神×魔尊/妖王的身份对立

**张力来源：**
- 正邪不两立的宏大背景
- 情感博弈vs立场对立
- 平衡"英雄救美"套路

---

### 类型三：重生逆袭类

**核心设定：** 前世记忆+命运改写

**叙事结构：**
- 为何被杀？（前世悲剧）
- 被谁背叛？（找出真凶）
- 错过哪些机缘？（今生截胡）
- 重生后步步为营

**爽点：** 重生线索埋设+命运改写节奏

---

### 类型四：满级大佬降维打击类

**核心设定：** 女主出场即巅峰

**特点：**
- 美强但不惨
- 凡间审判恶人如碾压
- 不需要逆袭成长

**代表作：**《云渺》神女云渺

---

### 类型五：团宠+科学修仙类

**核心设定：** 现代思维+修仙世界

**特点：**
- 穿书降智修仙文
- 学术思维科学修仙
- 团宠小师妹

**代表作：**《小师妹天生反骨》

---

### 类型六：硬核升级流类

**核心设定：** 扎实修炼+步步筹谋

**特点：**
- 修炼体系硬核
- 从练气到筑基到国师
- 姨娘到修真者身份转变

**代表作：**《陈姨娘她修真回来了》

## 三、力量体系设定

### 境界设计原则

| 设计要点 | 说明 |
|----------|------|
| **层级分明** | 炼气→筑基→金丹→元婴→化神（4-5层为宜） |
| **每层细分** | 初期/中期/后期 |
| **配套权力** | 徒子徒孙→正式弟子→受器重弟子→掌门 |

### 法宝与排名体系

女频仙侠核心元素：
- 法宝排名
- 神器排名
- 宠物排名
- 提供"横向竞争"爽感

### 灵根体系

**基础设定：** 五种基础灵根+变异灵根

**常见写法：** 五种基础灵根满值+冰/雷/风变异灵根满值

### 创新方向示例

```
先天五灵根 → 同修天/畜生/饿鬼/地狱/人道
以玄学术法沟通五界灵息
借因果业力淬炼道基
```

## 四、情感特有模式

### 师徒禁忌的张力设计

**多重冲突叠加：**
1. 权力不平等（师尊决定权）
2. 道德困境（伦常禁忌）
3. 外部压力（世界存亡）

**核心原则：** CP双方人设必须有足够内外冲突

### 三生三世情感延宕

**范式：** 轮回转世，多次错过

**每世意义：**
- 不同情感状态
- 不同命运转折
- 拉长情感期待长度

## 五、核心爽点

| 爽点类型 | 说明 | 写法要点 |
|----------|------|----------|
| **逆袭爽** | 废柴灵根到强者碾压 | 以排名体系和境界突破衡量 |
| **打脸爽** | 宗门大比/秘境闯关公开碾压 | 真正实力展现 |
| **飞升渡劫爽** | 生死关卡突破 | 渡劫+天赐奇遇（功法/法宝/身份） |
| **战斗策略爽** | 阵法破解/环境利用/五行相克 | 突出智谋，避免无脑堆数值 |

## 九、写作注意事项

### 常见误区

| 误区 | 正确做法 |
|------|----------|
| 师徒虐恋过度套路化 | 反思"救世"叙事套路 |
| 修炼潦草/情感过度 | 两条线阶段性互相推进 |
| "世界系"过重 | 避免个体情感承载过多政治隐喻 |
| 能力体系粗暴 | 层次分明，升级有逻辑 |
| 金手指同质化 | 空间/系统/重生已泛滥，需挖掘独特内核 |
| 人物脸谱化 | 赋予道德困境和情感纠葛 |
| 文风违和 | 避免现代词汇，多用四字短语/诗词引用 |

### 双线交织原则

```
感情线新进展 → 解锁事业线新资源/新机遇
事业线突破 → 制造感情线新矛盾/新靠近契机
```

两线不能各自为政！

## 十、与现有框架的衔接

### 与 charactercraft.md 的衔接
- 女性角色四大要素：独立目标/成长线/自主选择/底线
- 师尊/男主也需有独立目标和深度

### 与 plotarchitect.md 的衔接
- 关键节点设计：境界突破/渡劫飞升
- 金手指设计：代价原则（每个阶段一种金手指）

### 与 climax_design.md 的衔接
- 女频爽点类型：成长型爽点+情感型爽点
- 战斗爽点需要智谋元素

### 与 worldbuilder.md 的衔接
- 力量体系设计：境界/灵根/功法自洽
- 世界观构建：仙凡差距/门派势力
