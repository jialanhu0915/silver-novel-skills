# 耽美（BL）类型模板

**目标读者：** 喜欢男男恋爱的女性读者
**核心元素：** 双男主、浪漫张力、情感深度

## 耽美说明

- 起源于日本（Boys' Love），在中国发展为耽美
- 主要由女性作者为女性读者创作
- 男角色应被塑造成完整的男人，而非女性化的男人
- 重点关注情感连接和关系动态

---

## 耽美细分类型

### 类型一：现代甜宠

| 方面 | 描述 |
|------|------|
| **设定** | 现代、都市 |
| **基调** | 轻松、浪漫、HE（幸福结局） |
| **重点** | 日常生活、甜蜜时刻、关系发展 |
| **示例** | 室友、同事、青梅竹马 |

### 类型二：古代武侠

| 方面 | 描述 |
|------|------|
| **设定** | 武侠世界、古代中国 |
| **元素** | 修仙、门派、江湖 |
| **示例** | 武侠门派中的师兄弟关系 |

### 类型三：校园纯爱

| 方面 | 描述 |
|------|------|
| **设定** | 高中、大学 |
| **基调** | 纯真、青春、初恋 |
| **重点** | 情感觉醒、共同成长 |

### 类型四：虐恋情深

| 方面 | 描述 |
|------|------|
| **基调** | 虐心、激烈、可能有悲剧元素 |
| **冲突** | 家庭反对、社会压力、秘密 |
| **可能涉及** | 分手、分离、牺牲 |

### 类型五：玄幻冒险

| 方面 | 描述 |
|------|------|
| **设定** | 异世界、超自然 |
| **元素** | 系统、穿越、超能力 |
| **重点** | 特殊环境下的关系发展 |

## 攻受关系动态

> **重要：** 攻受是叙事角色分工，不是性格强弱！

### 攻方

- **传统叙事角色：** 追求方、主动方
- **不一定：** 性格上更强势
- **可以是：** 温柔、天真、古怪

### 受方

- **传统叙事角色：** 承受方、被追求方
- **不一定：** 性格上更弱势
- **可以是：** 强势、凶猛、能干

### 核心原则

两个角色都必须是有各自目标、性格和成长弧光的完整的人。

## 关系阶段设计

### 阶段一：相遇

- 初次相遇的方式
- 强烈印象（正面或负面）
- 难忘时刻

### 阶段二：相识

- 互相了解
- 发现差异
- 初步吸引

### 阶段三：暧昧

- 推拉
- 吃醋时刻
- 擦肩而过
- 身体张力

### 阶段四：确认

- 感情被承认
- 开始亲密
- 建立信任

### 阶段五：磨合

- 关系挑战
- 价值观差异浮现
- 亲密加深

### 阶段六：升华

- 深度羁绊
- 共同度过危机
- 准备好结局

---

## 耽美专项注意事项

### 1. 亲密戏处理

- 通常暗示优于明示
- 重点在情感，不只是身体
- 使用"雾里看花"技巧（暗示而非展示）
- 保持在内容规范内

### 2. 社会背景

- 可能包括：家庭排斥、社会偏见
- 谨慎处理：不要太黑暗，不要轻视
- 结局：挣来的，不是魔法般解决

### 3. 角色声音

- 两个男人听起来都应该是男人
- **避免：** 女性化的言语模式、"娘受"刻板印象
- **可以包括：** 攻受不同的言语风格但都男性化

### 4. 配角处理

- 常包括：女性朋友、家庭成员、竞争对手
- **功能：** 情节推动，不只是搞笑担当
- **必须是：** 有完整性格的

- [ ] 两个角色都像是完整的男人？

## 当前趋势（2025）

### 强强文的全面崛起

随着性别平等意识的增强，越来越多的耽美读者开始不满于女性在异性恋关系中的弱势地位。耽美强强文使用多种策略建构了攻受之间的对等关系：

| 设定 | 说明 |
|------|------|
| 精神共鸣 | 强调双方共有的强大精神力 |
| 平等爱情 | 以共同成长、灵魂沟通为特点 |
| 双向奔赴 | 双方都有独立目标，势均力敌 |

### 世界观的深度构架

BL文与穿越/仙侠/无限流等题材深度融合已成主流。

**代表作**：《魔道祖师》《天官赐福》等，以丰富的世界观构架和深刻的人物塑造不断刷新原耽行业的天花板。

## 感情线设计

### 经典感情线路径

```
邂逅 → 结识 → 相处 → 暧昧 → 误解 → 考验 →（短暂分别、重逢）→ 解开心结 → 告白 → 在一起
```

### 攻受关系平衡

| 类型 | 爽点 | 说明 |
|------|--------|------|
| 强攻弱受 | 凌虐的爽感 | 攻方在"施虐与保护"中同时满足控制欲与安全感 |
| 强强设定 | 灵魂共鸣 | 现代趋势，注重精神上的平等与互补 |
| 主动/被动转换 | 角色魅力 | 可"主攻"或"主受"为主视角，保持双方独立性 |

---

## 避免"娘受化"

"娘受化"本质是将传统异性恋中的"女方人设"生硬套用在男性角色上。

### 避雷策略

| 策略 | 说明 |
|------|------|
| 打破性别刻板标签 | 强"受"可以具有在特定领域中的力量与坚韧 |
| 保持人物复杂性 | 魅力不仅仅来自于"被保护"或"依附"攻方 |
| 双向欣赏 | 无论攻还是受，都把双方人物塑造为都有值得欣赏的地方 |

---

## 写作注意事项

| 问题 | 后果 | 解决方案 |
|------|------|----------|
| 感情逻辑劣于剧情逻辑 | 感情流的核心是感情，不是剧情 | 感情逻辑优先 |
| "零情感演技" | 主角像沉默寡言者+隔岸观火的路人 | 该表达时要表达 |
| 攻梦女倾向 | 作者对某一方的偏爱导致爆雷 | 保持双方平衡 |
| 设定不自洽 | 世界观（尤其高魔/无限流）逻辑感丧失 | 设定必须自洽 |
| 低俗黄暴 | 失去心理层面的读者满足 | 写"肉"要有技巧，既要有感官刺激，更要有心理层面满足 |

### 核心口诀

> 感情逻辑优于剧情逻辑；双方都要是完整的男人；魅力来自性格而非"被保护"
