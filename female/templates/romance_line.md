# 女频小说感情线设计指南

感情线是女频小说的核心驱动力，不像男频小说以战斗/升级为主。

核心原则：感情线不只是"两个人在一起"——它是一个有阶段、有张力、有情感共鸣的动态过程。

---

## 1. 感情线六阶段

### 阶段一：相遇

- 初次相遇的方式
- 强烈第一印象（正面或负面）
- 难忘时刻
- 篇幅：通常1-3章

### 阶段二：相识

- 日常互动
- 发现差异
- 轻度好奇/张力
- 篇幅：3-10章

### 阶段三：暧昧

- 甜的核心阶段
- 推拉动态
- 吃醋时刻
- 擦肩而过
- 篇幅：5-15章（核心甜蜜期）

### 阶段四：确认

- 感情被承认
- 但仍有障碍
- 公开vs私下动态
- 篇幅：3-8章

### 阶段五：磨合

- 关系挑战
- 价值观差异浮现
- 建立信任
- 可能有小分手
- 篇幅：5-12章

### 阶段六：升华

- 深度羁绊建立
- 共同度过重大危机
- 准备好结局
- 篇幅：3-8章

## 2. 推拉技巧

### 推 - 创造距离

- 假装冷漠/推开/消失
- 目的：制造焦虑、提升价值
- 风险：太多会让读者沮丧

### 拉 - 创造亲近

- 关心/暧昧/靠近
- 目的：回报情感投入
- 风险：太多会失去张力

### 健康的推拉模式

```
推："我不需要你"（但肢体语言相反）
拉："其实……我担心你"
推："算了，我要走了"
拉："等等……请留下"
```

## 3. 甜点设计

### 甜的类型

| 类型 | 描述 | 示例 |
|------|------|------|
| 细节糖 | 细心的关怀举动 | 记住偏好、注意变化 |
| 场景糖 | 浪漫场景 | 雨天窗前、星光下等 |
| 语言糖 | 令人心动的话语 | 表白、承诺 |
| 肢体糖 | 身体亲密 | 牵手、拥抱、亲吻 |

### 甜点原则

- 质量重于数量：一个有意义的甜 > 十个肤浅的甜
- 铺垫重要：甜前的张力放大甜蜜
- 因角色而异：什么对这个角色来说是甜的
- 多样化：混合不同类型的甜

---

## 4. 虐点设计

### 虐的类型

| 类型 | 描述 |
|------|------|
| 误会虐 | 误解（最多3-5章内解开） |
| 分离虐 | 分离（通常是距离，不是死亡） |
| 选择虐 | 在爱情和其他之间被迫选择 |
| 牺牲虐 | 自我牺牲（但不能太廉价） |

### 虐点原则

- 必须服务于剧情/角色成长
- 不应该是无尽的苦难
- 需要适当的解决（大虐后必须大甜）
- 避免：三观不正虐（为了剧情强迫角色变傻）

---

## 5. 节奏：3:1甜虐原则

每3个甜，就要有1个虐
每个重大虐段落后，要跟一个重大甜段落

### 好的节奏模式

```
第10章：甜（初吻）
第11章：甜（约会）
第12章：甜（小推-小担心）
第13章：虐（误会）
第14章：虐（分离）
第15章：解决 → 大甜（重逢）
```

## 6. 感情线追踪

### 数据库架构

```sql
CREATE TABLE romance_lines (
    id INTEGER PRIMARY KEY,
    novel_id INTEGER,
    character_pair VARCHAR(100),
    current_stage VARCHAR(50),
    tension_level INTEGER,
    sweet_level INTEGER,
    conflict_level INTEGER,
    last_development VARCHAR(200),
    next_target VARCHAR(200)
);
```

### 阶段指标

| 阶段 | 张力 | 甜度 | 冲突 |
|------|------|------|------|
| 相遇 | 中 | 低 | 低 |
| 相识 | 中高 | 低中 | 低 |
| 暧昧 | 高 | 中高 | 中 |
| 确认 | 中高 | 高 | 中 |
| 磨合 | 中 | 中 | 高 |
| 升华 | 中 | 高 | 中 |

## 7. 与剧情的关系

感情线应该推动剧情，而不只是伴随剧情：
- 外部冲突考验关系
- 关系选择影响剧情方向
- 两者都应该感觉不可或缺

### 融合模式

```
剧情事件 → 考验感情 → 角色成长 → 更深羁绊
感情发展 → 创造赌注 → 提升剧情赌注
```

## 8. 检查清单

- [ ] 清晰的感情阶段进展？
- [ ] 推拉动态保持？
- [ ] 甜虐比例平衡（3:1）？
- [ ] 双方都推动感情线？
- [ ] 感情线影响剧情？
- [ ] 用户偏好风格体现（甜为主/虐为主/推拉型）？
- [ ] 甜点类型多样化（细节/场景/语言/肢体）？
- [ ] 虐段有适当解决？
- [ ] 每个阶段有明确的标志？

## 快速参考：阶段过渡

```
相遇 → 相识：第一次有意义的互动
相识 → 暧昧：第一次吃醋或亲密时刻
暧昧 → 确认：相互表白或公开承认
确认 → 磨合：第一次严重冲突
磨合 → 升华：共同度过重大危机
升华 → 结局：幸福结局达成
```
