# 女频小说规划阶段提示词

## 核心规划理念

**男频规划顺序：** 世界观 → 角色 → 情节 → 金手指
**女频规划顺序：** 角色/感情线 → 世界观支持 → 情节服务感情

女频小说的核心是**人物情感发展**和**关系演变**，世界观和情节都是为感情线服务的。

---

## 阶段一：类型确定

### 1. 类型选择（女频）

请选择小说类型（可多选）：
```
1. 言情：现代都市、甜/虐、霸道总裁/温暖治愈/破镜重圆
2. 耽美(BL)：双男主、攻受关系、甜/虐
3. 百合(GL)：双女主、她力量、感情真实性
4. 女尊：性别反转、女强世界、男色消费
5. 快穿/穿书：系统流、攻略、多世界（任务由你确认）
6. 悬疑恋爱：推理+感情、甜蜜惊悚
7. 其他：如有其他类型请说明
```

### 2. 主角模式选择

请选择主角模式：
```
A. 单主角（女频默认）
B. 双男主（用于BL）
C. 双女主（用于GL）
D. 群像（3+主角）
```

### 3. 风格基调

请选择风格基调：
```
- 简洁明快：文字精炼，节奏快
- 古典雅致：文风古朴，意境深远
- 华丽绚烂：辞藻华丽，描写细腻
- 沉重严肃：深刻严肃，虐心虐身
- 轻松幽默：搞笑日常，甜宠为主
- 黑暗压抑：压抑氛围，虐恋情深
```

### 4. 篇幅目标

```
- 短篇：1-5万字
- 中篇：5-20万字
- 长篇：20-50万字
- 超长篇：50万字以上
```

```

B. 守护型：男性主动保护，女性有自主能力
C. 辅助型：男性提供资源/情感支持，女性主导
```

### 8. 女主依赖拯救的接受度

请选择你偏好的被拯救桥段：
```
A. 禁止：女主必须自救或智慧脱困
B. 少量允许：危机时刻可以有援手，但不能是常态

```

## 阶段四：核心角色设计

### 女频主角设计要求

**女主设计要素（必须有）：**

| 要素 | 说明 |
|------|------|
| 性格缺陷 | 封闭型/多疑型/逃避型/傲慢型 |
| 价值观底线 | 她不会做什么 |
| 内心恐惧 | 她害怕什么 |
| 成长弧光 | 她的情感/认知突破 |

**女主类型：**
- 野心型：有自己的目标，野心与现实冲突
- 独立型：不依赖任何人，有自己的信条
- 复杂型：道德灰色地带，选择充满困境
- 成长型：性格有缺陷，通过事件逐渐成熟

### 男主设计要素（如有）

| 要素 | 说明 |
|------|------|
| 独立目标 | 他有自己的目标（不是只为女主） |
| 性格深度 | 有缺点，有脆弱的一面 |
| 成长弧光 | 他会改变/成长 |

### 反派设计要求

反派必须是有智慧的角色，禁止无脑降智！

| 反派类型 | 特征 |
|----------|------|
| 智慧型 | 有城府，嘲讽有理有据 |
| 傲慢型 | 出身高贵，傲慢有资本 |
| 阴险型 | 笑里藏刀，背后算计 |
| 立场型 | 冲突源于立场，非善恶 |

反派嘲讽检查清单：
- [ ] 嘲讽有事实依据
- [ ] 嘲讽有目的（试探/施压/立威/离间）
- [ ] 反派有实际手段
- [ ] 对强者有尊重

### 角色关系网络设计

详细设计请参考：`shared/templates/character_relationship_network.md`

请设计角色之间的关系网络，包括：

| 关系类型 | 示例 |
|----------|------|
| 血缘关系 | 家族、亲子、兄弟姐妹 |
| 情感关系 | 爱情、暧昧、知己 |
| 对立关系 | 对手、仇敌、立场冲突 |
| 复杂关系 | 亦敌亦友、旧识、命运纠缠 |

关系设计四大原则：
- 关系要有意义（有历史背景和冲突潜力）
- 关系要有张力（表面vs背后、利益vs情感）
- 关系要动态（随剧情发展变化）
- 关系要平等（配角不是工具）

关系网络输出：
```
## 角色关系网络

### 核心关系
- [角色A] <--[关系类型]--> [角色B]
  - 初始状态：
  - 当前状态：
  - 发展趋势：上升/下降/波动/稳定

### 隐藏关系（如有）
- [角色X] <--[关系类型]--> [角色Y]
  - 伏笔埋设位置：
  - 揭示时机：
```

## 阶段五：感情线设计

### 感情阶段设计

请设计感情线阶段（6阶段）：

| 阶段 | 预期章节 | 用户偏好 |
|------|---------|---------|
| 相遇 | Ch 1-3 | - |
| 相识 | Ch 4-? | - |
| 暧昧 | Ch ?-? | 核心甜蜜期 |
| 确认 | Ch ?-? | - |
| 磨合 | Ch ?-? | - |
| 升华 | Ch ?-? | - |

### 感情线设计原则

1. 每个阶段必须有明确的标志性事件
2. 感情推进要有内在逻辑（不是因为"设定"而是"合理"）
3. 暧昧期是女频核心甜蜜期，务必重点设计
4. 虐点设计要有意义，不是为虐而虐

### 推拉设计（如选择推拉型感情线）

| 推（推开） | 拉（靠近） |
|-----------|-----------|
| 误会/误解 | 解释澄清 |
| 外界压力 | 共同克服 |
| 内心恐惧 | 逐渐信任 |
| 竞争对手 | 确认心意 |
