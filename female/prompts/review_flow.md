# 评测阶段提示词（女频版）

## 自我反思流程

```
章节创作完成后，进入自我反思评测阶段。

请依次调用以下评测 Agent，对章节进行全面检查：

## 评测顺序（女频版）

### 1. LogicChecker - 逻辑一致性检查
检查要点：
- 时间线是否矛盾
- 规则设定是否被打破
- 因果链是否完整
- 是否有常识错误

### 2. CharacterJudge - 人物塑造检查
检查要点：
- 人物行为是否符合性格
- 台词是否贴切
- 情感表达是否真实
- 关系动态是否合理
- 主角深度：性格缺陷、价值观底线、成长弧光
- 反派合理性：嘲讽质量、智商、城府

### 3. MaleRolesJudge - 男性角色专项检查（女频特有）
**这是Critical级别检查！**

检查要点：
- 男性角色是否有独立目标（不是只为女主）
- 男性角色是否有性格深度
- 男性角色是否有成长弧光
- 关系是否平等（根据用户选择）

### 4. FemaleCharacterJudge - 女性角色专项检查
检查要点：
- 女性角色是否有独立目标
- 女性角色是否有成长线
- 女性角色是否有自主选择权
- 女性角色是否有底线
- 是否沦为工具人

### 5. RomanceLineJudge - 感情线质量检查（女频特有）
检查要点：
- 感情线阶段推进是否合理
- 推拉节奏是否到位
- 甜虐平衡是否合理
- 角色在感情中是否有主动性
- 用户偏好的感情风格是否体现

### 6. PaceCritic - 节奏把控检查
检查要点：
- 开篇是否抓住读者
- 节奏是否有起伏
- 高潮是否有力
- 结尾是否有钩子

### 7. ForeshadowHunter - 伏笔呼应检查
检查要点：
- 之前埋下的伏笔是否回收
- 新埋伏笔是否自然
- 呼应是否合理

### 8. InfoAuditor - 信息交代检查
检查要点：
- 世界观交代是否清晰
- 是否信息冗余
- 是否信息缺失
- 展示方式是否自然

### 9. GoldfingerChecker - 金手指使用检查
检查要点：
- 金手指是否被正确使用
- 是否有代价或限制
- 主角是否过度依赖
- 金手指是否替代了主角的成长

### 10. ClimaxChecker - 爽点设计检查
检查要点（女频版）：
- 智慧碾压/情感突破/身份反转等女频爽点
- 铺垫是否到位
- 代价/来之不易
- 精神层面的爽点（成长/情感/认知）

### 11. SupportingChecker - 配角价值检查
检查要点：
- 配角是否有自己的目标
- 配角是否有自己的困境
- 配角是否沦为工具人
- 配角和主角是否平等

### BL/GL专项评测（仅BL/GL小说）

#### BLRelationshipJudge（仅耽美）
检查要点：
- 攻受关系是否平衡
- 是否娘受化
- 情感是否真实
- 身体描写vs情感描写比例

#### GLRelationshipJudge（仅百合）
检查要点：
- 两女主是否都有深度
- 女性真实性是否保持
- 情感共鸣是否真实
- 是否避免男性凝视

---

## 评测通过/失败标准（女频版）

### Critical（必须修复，否则不通过）

与男频一致：
- 反派出现无脑嘲讽
- 严重逻辑矛盾

**女频特有Critical：**
- 男性角色完全沦为工具人（无独立目标/无成长/无性格）
- 女性主角严重依赖男性拯救（违背用户偏好）

### Major（应该修复）

- 节奏过慢/过快
- 伏笔密度不合理
- 信息冗余/缺失
- 配角沦为工具人
- 反派嘲讽质量一般
- 感情线推拉失衡
- 甜虐比例失调（用户偏好偏差）
- 男性描写占比过高

### Minor（可以忽略）
- 个别错别字
- 微小的节奏问题
- 细节不够完善

## 评测后处理

### 如果需要修改
```
根据评测反馈，修改章节内容，修改后再次评测，直到通过。
```

```
收到人类反馈后：

   - 将反馈记录到数据库（可选）

2. 如果有修改意见：
   - 记录反馈到数据库
   - 根据意见修改章节
   - 再次提交审核
```
